The "Record Details" page lists all the details of the record that are visible from your current View. A Super User can change which fields are available in the View, and what headings group the fields together, from within the View setup. The field names and order can be changed from within the "Field Display" setup.

This help page outlines 5 major sections of Actions available from the Record Details on this page:

  • Moving through Search Results
  • Organization Name Links
  • Record Details Menu Items
  • Other Links on the Details Page
  • Changing Your View

Moving through Search Results

If this record is one of a number of records returned form your last search, you will see links at the top and bottom of the record:

First Previous | Next Last (10 of 100 Total)

You can use the First, Previous, Next and Last to move through your search results one at a time, or the Total link to return to your results page and see the full list of search results.

Note: The search results navigation links may not work properly or may not appear at all if you have executed a new search and then used your back button to go back to records from a previous search. For these links to work reliably, you need to have come to the Record Details pages directly from the "Search Results" or "Browse By Organization" page without performing another search in your browser.

Organization Name Links

CIOC uses a 5-level naming structure, using the fields named ORG_LEVEL_1 through ORG_LEVEL_5 (display names for these fields may vary based on your setup). ORG_LEVEL_1 represents the top-most parent organization, with lower levels representing various branches, programs or locations. If you use this structure properly and consistently, you will be able to use the name-linking feature to search for records belonging to the same parent organization structure at any level from 1 through 4 where at least one matching record is found. This feature is only available if it has been turned on in this View.

Changes made to the record here will not be seen immediately but will first be reviewed by a data manager prior to being added to the official record.

Update Record

You will have access to the Update Record option from the drop-down menu if you have permission to update the current record in the current View. Although the link always takes you to the form the current language first, you will have an opportunity to update other language versions after submitting your changes. You may also switch languages on this page if you want to start your editing from another language. Changes made to the record from the update form take effect immediately.

Copy Record

If you have permissions to create new records, you will have access to the Copy record feature. You will have an opportunity to select specific fields from this record to copy from those you have permission to update. To copy a record, you must have update permissions on the current record and permissions to add new records.

Delete / Restore

Delete will allow you to set a deletion date for the record; records with a past deletion date are no longer available in regular searches, while records with a deletion date in the future will remain active until the date has passed. Deleted records will show a Restore link instead; restoring the record will remove its deletion date. Delete and Restore functions are available only to those with record deletion privileges.

[ Top ]

Update Service Categories

This drop-down menu option allows you to update the Service Category (Human Services Taxonomy) indexing of the record. This option is available only to users with permissions to index with the Taxonomy, and is independent of the ability to update the rest of the record.

[ Top ]

Update Publications

Link to update the list of Publication codes attached to this record, as well as any related Publication information such as Publication-specific descriptions and General Headings. This link is visible if you have permissions to update publications, and you are not currently in a Limited Publication View. If you are working in such a "Limited" View, you will see an "Update Description/Headings" link instead.

[ Top ]

Update Description/Headings

Link to update the Publication-specific description and General Headings for the Publication on which your current View is based. This link is visible only when working in a Limited Publication View, if you have permissions to edit Publication information.

Email Update Request

This link will take you to a page where you can compose/customize an email to the organization to request an update to their information. This link is available if you have permissions to send update requests, the organization has an email address available for updates contained in the record, and the "Block Update Email" field has not been set in this record.

Mail Form

The Mail Form option from the drop-down menu is intended to be used when an you are not able to use the Email Update Request functionality to request an update to this record. If you have defined a Print Template for this View, the Mail/Fax form will use this template. As with the Feedback page, the Mail/Fax form gathers contact information based on what is specified for the owner Agency in the Agency setup.

You may want to go into your browser's "Page Setup" (found under the "File" menu) and expand your margins and/or remove header and footer information prior to printing. Removing header and footer information from the page setup will prevent your browser from automatically printing out page numbers, the url for the page, date and time information, etc. Increasing your page margins may also reduce the number of pages necessary to print out the update form.

The Mail Form opens in a new window or browser tab. When you are finished, close the window or tab to be returned to the previous page you were using in the database.

Reminders

The Reminders link allows you to view, edit, or add to the reminders tied to this record. If the record has a currently active reminder, an icon will appear next to the reminder link. An alarm icon signifies active reminders, while an alert icon signifies that at least one reminder is past due. For more help working with reminders, view the help file for the Main Reminders Page.

Other Links on the Details Page...

Within the record itself, there may be other links to new searches or external websites. Many CIOC fields allow you to enter HTML, giving you the ability to create your own links in the record. Some of the links created by the software include:

Website and Email Links

The various website email fields in the software will be linked automatically by the software unless you are viewing the print version of the record. Wherever more than one email address is listed in a single email field, clicking the link will open your email software to compose a message to all the addresses listed.

Social Media Links

Note that there may be two types of Social Media links available on the page:

  1. Social Media links about the Organization/Program itself; this could include links to the organizations Facebook page, X (Twitter) feed, etc.
  2. Social Media sharing links; these appear at the upper right of the record (if enabled in the View) and allow the user to share a link to the current record on their own Facebook page, X (Twitter) feed, etc. This feature uses the Add This service, which works hundreds of different social media sites.

Mapping Link

This link will appear in the "Address" field for an organization only if there is sufficient information in the record to allow for correct mapping. Clicking on this link will open up a map of the organization's location. The MapQuest map will open up in the same window as the database; you will need to use your browser's back button to return to the database after viewing the map. Note that the mailing address field is never mapped.

Which Mapping System is used will depend on the setting in the Map Record field; the choices available are configured in the Mapping Systems setup area.

Classification System Links (Subject, Taxonomy, Publication, Distribution, NAICS etc)

Depending on your view of the database, you will see various subject/term, publication/heading, distribution, NAICS, broader and related term links within the record fields. Clicking these links will perform a search for records containing the chosen term or code.

Other Search Links

Changing Your View

If your security level permits it, you will also see a drop down menu at the top of the page where you can change your View of the database; this may change the look of the record display page, or change which fields are available. Unlike changing your View from the Main Menu, this change is temporary and will last only while you look at this record. Important Note: you can only update records in your own View; if you switch to another View other than your default, you will no longer have access to the Update Record link.
