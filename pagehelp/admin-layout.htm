Template Layouts are used within Design Templates to control the layout of customizable parts of the page, including headers, footers, and main page search forms and content.

Layouts that are exclusively controlled by a particular Agency have that Agency Code at the end of the name in square brackets (i.e. [ABC]). You will be able to select Layouts that you don't own from the list for viewing, but will not be able to make changes to those Layouts.

You will have one or more built-in system Layouts in your database for each type (Header, Footer, Search). You can view or copy these built-in Layouts, and use them in your Templates if desired, but they cannot be edited in any way. System Layouts are denoted with a *.

Edit an Existing Template Layout

Select the name of the Layout that you wish to edit from the drop-down menu and click the "View / Edit Template Layout" button.

Add a New Template Layout

To create a new Layout, click the "Add Template Layout" button to be taken to a new Layout data entry form. You can copy an existing Layout by selecting it from the copy drop-down list. If you do not select a Layout to copy, your new Layout will be blank (empty).
