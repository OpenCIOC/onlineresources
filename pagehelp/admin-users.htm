The Manage Users page is available to users with security levels that allow them to manage users.

IMPORTANT NOTES

  • Only users designated as Super Users can edit account settings for another Super User.
  • You can only edit the User Type information for the module(s) in which you have user management privileges.
  • Users who have user management privileges but are not Super Users can edit accounts for those with equivalent and lower security levels. They can not edit the accounts of Super Users.
  • No User including a Super User can edit the permissions and security level of their own account. This reduces the possibility of a single Super User for a database inadvertently "locking themselves out" of database administration. If only a single Super User exists for a database, a new Super User account must first be created before the first Super User account can be edited.
  • Note that it is not possible to delete existing users, only inactivate them.

Using this Manage Users page, you can:

Create a New User

To create a new user, click the "Create New User" link.

Edit An Existing User Account

Locate existing user accounts using the provided search form. To retrieve a list of all users, select whether you would like active/inactive users or both, and submit the search form with no other criteria. If you have permission to update the user, you will see an "Update" link next to the account in the search results.

Manage Your Own Account

As noted in the "Important Notes" section above, you can not edit your own account security level and database permissions. Only another user with the equivalent or higher security level can edit your account. However, using the "My Account" link on this page, you can change your name, initials, email address and password.

View Account Change History

The Account Change History tool allows you to review past changes to user accounts.
