From this page you can export chosen records from the database in three formats, depending on your permissions:

  • Microsoft Saved Recordset (XML) - uses Microsoft's XML recordset format; a tool is available for CIOC members to convert this format into a Microsoft Access table (reference available on the CIOC Community site).
  • CIOC Sharing Format (XML) - works with the CIOC Import Tool. Note: to share records with other CIOC users, their CIOC software version must support a compatible version of the CIOC Sharing Format. If the partner you are working with uses a newer or older version of CIOC, or 3rd party software, your export may not be compatible.
  • Excel Spreadsheet - formats compatible with various versions of Microsoft Excel and/or other spreadsheet programs.

Choose the format you wish to export in. Depending on your permissions, you may be given the option to restrict your export based on a set of codes (e.g. Publication, Distribution, or Category). Choose one or more codes from any one list of codes to include in the export and then click the "Next" button. You choose multiple codes by holding your "Ctrl" button down while making selections from the list.

Export Profiles / Excel Profiles

Export and Excel Profiles allow you to specify specific fields to export. You must specify a Profile to use from those available to you. Note that Profiles are made available on a per-View basis; edit the list of Views for each Profile on the Profile setup page.

Generating the Export File

Once you have made your selections, click "Next" to begin the export process. The file will be available on the server for a limited time only, so please download the file promptly. Only the person who generates the file may download it.
