The Manage Export Fields page allows you to select which fields will be included in the records you export. Check or uncheck the boxes next to the field names you want to include or exclude from this Export Profile, and click the "Submit" button at page bottom to save your changes. Not all fields are available for export, and Distribution and Publication fields are managed separately from the rest of the fields in the Export Profile.

Submitted changes take effect immediately. It is not necessary to submit the main Edit Export Profile form to save your changes.
