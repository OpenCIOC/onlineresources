Creating Referrals

Note: New Referrals are created through a button on the Record Details page of the Opportunity. There are two ways a new Referral might be created:

  • Yes, I'd Like to Volunteer Button - Public and staff without permission to Manage Volunteer Referrals use this method to create a new Referral. Only the contact information for the Volunteer and the Volunteer's own notes for the Organization can be entered using this form.
  • Create Referral Button - Super Users always see this option. Staff with a User Type that has the Can Manage Referrals permission will be allowed to create new referrals for records owned by their own Agency. This Referral form is the full edit form for Referrals, and staff can decide whether or not to send out notification Emails to the Volunteer and Contact Person for the Opportunity.

Find a Referral

There are currently two types of predefined reports you can use to list Referrals in the database. In addition, you can perform a custom search referrals using the form provided on this page. Most users will only see Referrals that are created for records owned by their own Agency; Super Users see all Referrals, regardless of the Agency that owns the associated Opportunity record.

You can list Opportunities Requiring Follow-up; this list includes all flagged Opportunities, regardless of the date on which they were created or modified. All new referrals created by Volunteers using the Yes, I'd Like to Volunteer button are automatically flagged for follow-up.

You can list Opportunities Modified in the Past [X] Days; this list includes all Opportunities created or modified in the past [X] number of days.

Custom Referral Search

The following types of search parameters are available using the Custom Referral Search form:

  • Referral Date: On or after the date: Enter the date you wish to be the first day to include the search results. This date is included in the results. Before the date: Enter the date you wish to be one day after the last date you wish to include in the search results. This date is not included in the results. To generate a report for exactly the previous month, click the "First of Last Month" and "First of This Month" buttons.
  • Modified Date: On or after the date: Enter the date you wish to be the first day to include the search results. This date is included in the results. Before the date: Enter the date you wish to be one day after the last date you wish to include in the search results. This date is not included in the results. To generate a report for exactly the previous month, click the "First of Last Month" and "First of This Month" buttons.
  • Organization Keywords: You can include keywords or "quoted phrases" from the Organization name to locate Referrals for specific Organizations.
  • Position Title Keywords: You can include keywords or "quoted phrases" from the Position Title to locate Referrals for specific Positions.
  • Volunteer Name: You can include all or part of a volunteer's name to locate a specific volunteer's Referrals.
  • Follow-Up: You can search for records with a specific follow-up flag state. Any will show all records regardless of flag state, Required will select only referrals that are flagged as requiring follow-up and Not Required will select only referrals that are not flagged as requiring follow-up.
  • Records owned by: If there is more than one record owner in the database, you can specify here to return results based on this criteria.

Note: To get a list of Referrals for a specific Opportunity, or to create a new Referral, go to the details page of the Opportunity.
