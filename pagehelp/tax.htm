Search by Keyword

In general, the Taxonomy Term Keyword search can be expected to behave like the CIOC keyword search from the basic search form. All words or phrases entered must be matched, and some reserved or very common words - like "and" and "or", will be automatically ignored. You can search for a phrase by enclosing the desired words in quotation marks. Character accents are automatically ignored by the software and are not required for your search in order to get a match, even when searching in French.

The software automatically searches many different word forms for the current language. For instance, typing in library or libraries produces the same result, so either can be used. You can explicitly prevent the searching of word forms by using quotation marks around the word of phrase. You can also add the wildcard character * to the beginning or end of a word in your search to allow a partial word match (e.g. dog* would match dog, dogs, dogma, dogged, etc; *bus* would match bus, abuse, bust, business, etc. In many cases, allowing the software to use its built-in ability to search word forms is more effective than using the asterisk for simple variations of the same word (child and children, help and helping). This is because it reduces the number of unwanted extra matches, such as scan* matching scant and scandal when what was intended was just matching scan, scans, and scanning.

Keyword search allows you to look for keywords within the Term's Name (or the Alternate Name) and - if you choose the Anywhere option - it will also search in the Term's Definition or Alternate Definition. All the Keywords you specify must appear in any one of the fields you are searching in. That means if you searched for the keywords "abuse" and "neglect" using the Anywhere search option, it would have to find both of those words either in the Name/Alternate Name or the Definition/Alternate Definition. You would not get a match if "abuse" was in the Name and "neglect" was in the Definition.

Search by Code

Code search allows you to list all Terms that belong to the branch of a given Taxonomy Code. You must enter a valid code to perform this search. That means if you searched for TA-04 you would not get any matches; however, TA-0400 would list TA-0400 and all its child Terms, because it is a valid Code.

Often the Drill-Down search is a better option when you want to navigate branches of the Taxonomy hierarchy. However, unlike the Drill-Down search, a Code search will allow you to sort by name and display the entire branch at once without having to drill into it first. The Code search results is also where you would end up if you click on a Narrower, Broader or Related Term when Displaying More Term Information.

Drill-Down Search

The Drill-Down search allows you to navigate one or more branches of the Taxonomy hierarchy. It has similar features to the other searches, but you cannot sort your search. Unlike other search types, changing your settings or leaving the page will always reset the search and all branches will close (See Effects of Using the Back Button).

To expand a branch of the hierarchy, click on the plus sign () which will be available next to any branch that can be expanded. The next level of terms will be automatically retrieved for you and the branch will expand, and the plus sign will turn into a minus sign (). Note that you should wait for the branch to open, which could take a little while if there are many Terms to display. If a minus sign has appeared after clicking, it means that your browser is waiting for the Terms to be sent and will display them when they are ready. If you click on the minus sign, you are making a request to close the branch again and your terms may not appear at all.

Displaying More Term Information

When viewing a list of Taxonomy Terms returned from one of the above types of searches, you can View more detailed information about the Term by clicking on the name of the Term. The Term's Definition (or Alternate Definition) will be displayed as well as any associated Related Concepts and the Broader Term, Sub-Topics, or Related Topics for the Term. Concepts will be linked to a Related Concept Search, while any associated Terms will display a count of the number of records using the Term and be linked to a Code Search for that Term. Super Users will see additional Data Management fields.

You can display additional Term information for any number of Terms in your search results simultaneously. To hide the extra information, click again on the name of the Term.

Changing the Sort Order

The sort order of your search results can be changed for all types of searches except the Drill-Down search. The initial sort order varies depending on the type of search you perform; for example the Keyword search sorts by Name by default while the Code search will initially sort by Code. Note that the sort order is not remembered between searches.

Your current sort order is denoted by an arrow next to the column name ( or ). The arrow indicates which column is being sorted and whether the sort is Ascending or Descending. To change the sort to a different column, or change the direction of the sort on the existing sorted column, click on the column's name in the table header and your results will be resorted. Changing the sort order may close some or all of the More Information boxes you have open.

Search Settings

Search settings from the search menu last only until you leave the Basic Search page (this includes going to Manage Taxonomy). If you turn on one of these menu-based search settings, it will be highlighted to remind you that it is active.

Note that changing your settings will reset or redisplay your current search results. In the case of a Drill-down search, you will have to re-open the branch(es) you were displaying when you changed your settings.

Terms with Records / Show All Terms

You can choose to display all Terms (the default) or only those that are attached to records in your current View. If you have elected to show only terms with records, the menu option will change to say Show All Terms and will be highlighted. If you are having trouble finding an appropriate Term, check that you are showing all Terms; even if there are no records associated with the Term, you may be able to use the More Term Information feature to find related or narrower Terms that do have records.

Show / Hide Inactive Terms

You can choose to include Inactive Terms and Use References in your search by turning on the Show Inactive Terms option. This option is not available to Super Users in the Basic Search; they have the option to search in Admin Mode instead.

Admin Mode / Normal Mode

Admin Mode is available in lieu of Show Inactive Terms for Super Users from the Basic Search. Admin Mode offers the following features:

  • Inactive Terms and Use References are available in all searches; however unlike the regular Show Inactive Terms, Admin Mode also includes Inactive Terms that are Rolled-Up into Use References for higher-level Active Terms. When Viewing More Term Information, Use References that represent Rolled-Up Terms are linked so that they can be viewed and edited.
  • Keyword Search results include all possible matches, including Active and Inactive Use References and Rolled-Up Terms, even if the Preferred Term also matches the search parameters.
  • Edit Icons appear next to Terms.

Effects of Using the Back Button

Some features of this search use JavaScript to change your display without having to reload the page. Examples of this will be the display of More Term Information box, opening branches of a Drill-Down search, or changing your sort order.

This means that use of the browser's back button after leaving the page (for example after leaving the page to perform a record search, edit a term, or going to a different search option) may mean that you are unable to return to the same state of your search results if you leave the page and then use the back button to return. In addition, the back button will not undo the types of actions mentioned above that are controlled by JavaScript. Use the methods explained in this help file if you want to change back your sort order, close the More Term Information box, or close a branch of the hierarchy in a Drill-Down search - the back button will not work to do this!

Do not rely on the browser's back button - get in the habit of always using the available menu items to do your searching and navigate the site.

Select a Term to Search

If there are records associated with a Term or any of its Sub-Topics in this View, a magnifying glass icon will appear next to the Term (). Clicking on this icon will perform a record search for the Term and all its Sub-Topics.

If the Term is being directly used by records in this View, there will be a record count next to the Term name in square brackets indicating the number of records using the Term in this View: [#]. Clicking on the record count will perform a search for all records that directly use this Term - it does not include any Sub-Topics.
