The "Record Details" page lists all the details of an organization that are visible from your current View. A Super User can change which fields are available in the View from within the View setup. The field names and order can be changed from within the "Field Display" setup. Note that the fields under "Other Agency Information" are included automatically by the software and cannot be changed.

This help page outlines 6 major sections of Actions available from the Record Details on this page:

  • Moving through Search Results
  • Yes, I'd Like to Volunteer / Create Referral
  • Record Details Menu Items
  • Agency Details Menu Items
  • Other Links on the Details Page
  • Changing Your View

Moving through Search Results

If this record is one of a number of records returned form your last search, you will see links at the top and bottom of the record:

First Previous | Next Last (10 of 100 Total)

You can use the First, Previous, Next and Last to move through your search results one at a time, or the Total link to return to your results page and see the full list of search results.

Note: The search results navigation links may not work properly or may not appear at all if you have executed a new search and then used your back button to go back to records from a previous search. For these links to work reliably, you need to have come to the Record Details pages directly from the "Search Results" page without performing another search in your browser.

Yes, I'd Like to Volunteer / Create Referral

Someone looking to fill a volunteer position can click "Yes, I'd Like to Volunteer!" button to open a request form. When the potential volunteer fills in their contact information and any requests for information, an Email is sent to the Contact in this record as well as the Email for the Agency that owns this record in the database to follow up with the potential volunteer about this position. More information is available from the "Yes, I'd Like to Volunteer" page.

If you have permissions to create a Referral for this record, you will instead see a link to "Create Referral", which includes everything from the "Yes, I'd Like to Volunteer" form as well as additional fields for data management and tracking your contacts for this Referral. Even if you have permission to create referrals, you will only be able to use the "Create Referral" option for records owned by your own Agency unless you are a Super User.

Add Record (to My List / Client Tracker)

If you use the "My List" feature in your View, or are working in a Client Tracker call, you will see an  Add Record menu option (or for Client Tracker), allowing you to add this record to either your list or call. If you have already added this record to your list, it will instead say  Record Added (or for Client Tracker)

Suggest an Update

Visible to all users and the public, this link will take you to a feedback page where you can make a suggestion for a change to the details of the current record. Changes made to the record here will not be seen immediately but will first be reviewed by a data manager prior to being added to the official record.

Update Record

You will have access to the Update Record option from the drop-down menu if you have permission to update the current record in the current View. Although the link always takes you to the form the current language first, you will have an opportunity to update other language versions after submitting your changes. You may also switch languages on this page if you want to start your editing from another language. Changes made to the record from the update form take effect immediately.

Copy Record

If you have permissions to create new records, you will have access to the Copy record feature. You will have an opportunity to select specific fields from this record to copy from those you have permission to update. To copy a record, you must have update permissions on the current record and permissions to add new records.

Delete / Restore

Delete will allow you to set a deletion date for the record; records with a past deletion date are no longer available in regular searches, while records with a deletion date in the future will remain active until the date has passed. Deleted records will show a Restore link instead; restoring the record will remove its deletion date. Delete and Restore functions are available only to those with record deletion privileges.

Visible to logged-in users with permission to manager referrals. View a list of referrals for this opportunity. Referrals may come from staff creating a referral, or members of the public clicking "Yes, I'd Like to Volunteer!".

Email Update Request

This link will take you to a page where you can compose/customize an email to the organization to request an update to their information. This link is available if you have permissions to send update requests, the organization has an email address available for updates contained in the record, and the "Block Update Email" field has not been set in this record.

Print Version

If a Print Template has been specified for the current View, you will have access to a link that will open the record in a new window. Print versions are formatted for printing using the template provided, and all system-generated links and menu items will be hidden.

Reminders

The Reminders link allows you to view, edit, or add to the reminders tied to this record. If the record has a currently active reminder, an icon will appear next to the reminder link. An alarm icon signifies active reminders, while an alert icon signifies that at least one reminder is past due. For more help working with reminders, view the help file for the Main Reminders Page.

Other Links on the Details Page...

Within the record itself, there may be other links to new searches or external websites. Many CIOC fields allow you to enter HTML, giving you the ability to create your own links in the record. Some of the links created by the software include:

Website and Email Links

The various website email fields in the software will be linked automatically by the software unless you are viewing the print version of the record. Wherever more than one email address is listed in a single email field, clicking the link will open your email software to compose a message to all the addresses listed.

Social Media Links

Note that there may be two types of Social Media links available on the page:

  1. Social Media links about the Organization/Program itself; this could include links to the organizations Facebook page, Twitter feed, etc.
  2. Social Media sharing links; these appear at the upper right of the record (if enabled in the View) and allow the user to share a link to the current record on their own Facebook page, Twitter feed, etc. This feature uses the Add This service, which works hundreds of different social media sites.

Mapping Link

This link will appear in the "Address" field for an organization only if there is sufficient information in the record to allow for correct mapping. Clicking on this link will open up a map of the organization's location. The MapQuest map will open up in the same window as the database; you will need to use your browser's back button to return to the database after viewing the map. Note that the mailing address field is never mapped.

Which Mapping System is used will depend on the setting in the Map Record field; the choices available are configured in the Mapping Systems setup area.

Volunteer Areas of Interest and Skills/Experience

Each Specific Area of Interest or Skills/Experience type associated with the record will be linked. Clicking these links will perform a search for records containing the chosen Areas of Interest.

Changing Your View

If your security level permits it, you will also see a drop down menu at the top of the page where you can change your View of the database; this may change the look of the record display page, or change which fields are available. Unlike changing your View from the Main Menu, this change is temporary and will last only while you look at this record. Important Note: you can only update records in your own View; if you switch to another View other than your default, you will no longer have access to the Update Record link.
