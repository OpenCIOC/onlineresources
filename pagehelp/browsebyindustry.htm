The North American Industry Classification System (NAICS) is used to classify businesses based on the type of products and/or services they provide. The Canadian version of this system is used in the software by permission from Statistics Canada. The classification structure is a hierarchy, starting with a set of two-digit codes representing major industry Sectors. The system then further divides each Sector into finer sub-categories (Subsector, Industry Group, Industry, and National Industry) with codes ranging from 3-6 digits.

Clicking the Search link as you browse through the classifications will return a list of records classified under the selected classification category or any of its Sub-Categories. By clicking on the Sub-Categories link next to any of the higher-level classifications, you can narrow your search to a more specific type of industry before returning a list of records. You can reverse your path through the hierarchy by going Up 1 Level, or Return to Top Level to return to the full list of Sectors.

At any time you can use the NAICS Code Finder to open a pop-up window which will allow you to browse or search through the entire NAICS system. If you are not sure which sector is appropriate for the industry you are seeking, a keyword search using the Code Finder may help you to locate the appropriate branch of the system to browse.
