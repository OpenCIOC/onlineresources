Design Templates control the style / look of the database, including colours, logos, style sheets, headers and footers, etc.

Templates that are exclusively controlled by a particular Agency have that Agency Code at the end of the name in square brackets (i.e. [ABC]). You will be able to select Templates you don't own from the list for viewing, but will not be able to make changes to those Templates.

You will have one or more built-in system Templates in your database. You can view or copy these built-in Templates, and use them in your Views if desired, but they cannot be edited in any way. System Templates are denoted with a *.

Edit an Existing Template

Select the name of the template you wish to edit from the drop-down menu and click the "View / Edit Template" button.

Add a New Template

To create a new Template, click the "Add Template" button to be taken to a new template data entry form. If you have existing Templates, you can copy a Template by selecting it from the copy Template list. If you do not select a Template to copy, your new Template will have default settings.
