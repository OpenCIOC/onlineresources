Each Taxonomy Term can be associated with a Facet which is meant to represent different types or categories of Terms. Examples of Facets include: "Service", "Target", and "Organization/Facility Type". A Facet does not indicate anything about the specific kind of service, population, or format of delivery, but rather the broad idea of whether the Term describes some kind of service, population, or format of delivery, and so on. For more information about Facets and other structures of the Taxonomy, you should use an official reference such as the library on 211taxonomy.org.

Note: Values on this page can only be added, updated and deleted one row at a time. Remember to use the "Action" buttons beside each row after each change is made before making a change to another row or adding another value.

Add a New Facet

Type the name you wish to use as the Facet for Taxonomy Terms into the bottom most text entry box on the page and click the "Add" Button.

Edit an Existing Facet

Make the required changes to the Facet you wish to edit on the page and click the "Update" button beside that value. The Usage column indicates the number of Taxonomy Terms that currently use each Facet.

Delete an Existing Facet

Click the "Delete" Button beside the value you wish to delete. You cannot delete values that are in use by Taxonomy Terms.
