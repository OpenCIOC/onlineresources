The management pages for Standard Email Update Text values allow you to customize sections of the email update request messages. You may create several different pre-defined texts for each type of message (Community Information Organization/Program update, Volunteer Opportunity update, Volunteer Organization update for "All Opportunities"). One message for each type must be designated as the default; the default message will be the one initially selected when you create a new Email Update Request. If you have multiple Email Update Text values, you will have an opportunity to make a choice between them at the time of sending.

Add a New Email Text

To create a new Email Text for the current message type (Community Information Organization/Program update, Volunteer Opportunity update, Volunteer Organization update for "All Opportunities"), click on the "Add Standard Email Text" button. Optionally, you may copy an existing text.

Edit an Existing Email Text

To edit an existing Email Text, select it from the drop-down and click "View/Edit Standard Email Text". Note that each type of message (Community Information Organization/Program update, Volunteer Opportunity update, Volunteer Organization update for "All Opportunities") is managed separately, and available from a different link on the main setup page of the software.

Changing the Default Value

To change which text is the default for the database, you must go into the text that you want to make the default and click the appropriate checkbox.
