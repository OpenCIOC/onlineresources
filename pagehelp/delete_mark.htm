The "Mark Record(s) Deleted" page allows you to set the date on which a record should be deleted. In some cases, you may land on this page while restoring a record that was previously marked for deletion.

Marking a Record Deleted

If the date you enter on this page is in the past, the record will immediately become unavailable in regular searches (it may still be available in Advanced Search if deleted records are visible in the current View). If you set a date in the future, the record will be "pending deletion" and will be removed from searches at 12:00:00 AM EST on the date specified.

You may optionally set the record to non-public status at the same time; this is useful for instance if you want it to disappear from staff searches in 6 months, but want it removed from public view immediately. Note that this feature only works in one direction - it can only set records to non-public. Unchecking the box will not make records public if they are currently non-public.

Note that you cannot restore the record by leaving the date blank after following a delete record link; attempts to leave the date blank will insert today's date. If you wish to restore the record, you can do so by clicking the "Restore" link from within the "Record Details" page, or the "Manage Deleted Records" page.

Restoring a Record

After clicking the "Restore" link for a record, the record is automatically restored; no further action is required on this page. You may return to the Record Details or Deleted Records pages by clicking the appropriate links.
