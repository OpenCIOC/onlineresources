There are two types of statistical tools provided with your CIOC database:

Record View Statistics

These statistics are generated within your database when a record in your database is accessed. It records the date and time, IP Address, View, UserID, Record Number, and Language. Record View Statistics are not kept when you change to print mode or temporarily change your View, to avoid inflating statistics when a person is looking at multiple versions of the same record for the same purpose. These statistics are most useful for:

  • Understanding which records in your database are popular
  • Understanding how each of your different Views are being used
  • Understanding how different record languages are being used
  • Identifying how a particular User or group of Users are using the database

Record View statistics are available to Super Users as an offline Microsoft Access database through the download menu, if desired for further analysis.

Web Log Statistics

These statistics represent the total use of your database, and includes all types of file (including images, CSS , JavaScript, and other files). These statistics are very useful for understanding the overall use of your database and traffic growth over time. Web Log Statistics are kept in a location separate from your database; the official technical contact for your Membership can contact CIOC staff to obtain a login to this tool. For a small fee, separate web statistics tool areas with their own login can be configured for specific domain names used by your database. Contact CIOC technical staff for information.

This page contains the main Record View Statistics search and reporting form, and links to several other statistical tools: Total Record Use, Top 50 Records, Use By Agency, Quick Reports, and Delete Old Statistics.

User's IP address begins with

If you know a specific user IP (or the relevant start of a block of IP Addresses) and are trying to determine that IP's use of the database, enter it in this text box. For example, if you know that your local CCAC has IP Addresses that all begin with 199.235.235.XXX where XXX represents different identification numbers for individual machines in that CCAC, enter 199.235.235 in this text box and the results should accurately reflect usage of the database by the local CCAC.

Users with the User Type

You may confine the statistics only to those record views made by users whose account is associated with a particular User Type

View Search

You may confine the statistics only to those record views made from a particular View (this does not take into account whether the user was logged in, or what their User Type is)

Date Search

on or after the date: Enter the date you wish the statistical report to begin on. This date is included in the results.

before the date: Enter the date you wish to be one day after the last date you wish to include in the statistical report. This date is not included in the results. To generate a report for exactly the previous month, click the "First of Last Month" and "First of This Month" buttons.

Records owned by

If there is more than one record owner in the database, you can specify here to return results based on this criteria.

Record views by

Return results based on:

  • any uses of the database, or
  • uses by logged in users only, or
  • uses by public (non-logged in) users only.

Record views in

Confine statistics to records viewed in a specific language.

Search engine record views

Include or exclude known search engine traffic, or show traffic by a specific search engine. Please be aware that this tool can only approximate search engine traffic based on known addresses; it will not cover all crawlers to your database.

Print Version

If you want the report to be published in Print Mode, select Yes for this option. Reports run in Print Mode open in a new window.
