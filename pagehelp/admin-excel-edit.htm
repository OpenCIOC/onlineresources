This page allows you to edit the details of an Excel Profile, including which data elements will be included, and in which View(s) it can be accessed. For general information about Excel Profiles, see the help file for the Manage Excel Profiles page.

Profile Name

A unique name for the Profile. Used to select the profile during the export process.

Views

To make this Excel Profile available for use, you must enable it in one or more Views. For security reasons, it is recommended that you only activate each profile in the View(s) in which it will be needed.

Fields

Fields may be added to your Excel output, and/or used to sort the records within the file. It is not necessary for a field to have a value in both the "Column Order" and "Sort Order" columns. Placement within the column order or record order is determined by sorting first on the number provided, and then the name of the field. You may use the same number for multiple fields if you wish them to sort alphabetically (for instance, you can put 0 for every field you choose in the  Column Order and it will automatically put the columns of the spreadsheet in alphabetical order; you can add fields ORG_LEVEL_1 through ORG_LEVEL_5 to the sort order all with a value of 0 and they would sort first by level 1, then level 2, then level 3, and so on. The maximum value you can specify is 255, and numbers do not have to be sequential; in fact, adding items by increments of 10 leaves you room to add new fields in between at a later date.
