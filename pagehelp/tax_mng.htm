Manage Taxonomy

Create New Term

To create a new TaxonomyTerm, click the Create New Taxonomy Term link. You will be taken to a form to enter the term information.

Edit Existing Term

To edit an existing Subject Term, use one of the Search types in the menu (Keyword, Code, Drill-Down, Related Concept) to locate the Term(s) you want to edit. An edit icon () will appear next to the name of the Term; clicking on this icon will bring up a form to edit the Term. If you are having trouble locating the Term you want to edit, try changing your settings to include Inactive Terms, and check that you are not limiting your search to only Terms with Records.

Update Taxonomy Version

This tool allows you to update your Taxonomy version to the most recent one posted to a central location on CIOC's servers (to ensure all members can stay in synch to the same version of the Taxonomy).

Manage Source Values

Each Term can be associated with a Source (i.e. "INFO LINE"). To edit the available list of Source values, click the Manage Taxonomy Source Values link.

Manage Facet Values

Each Term can be associated with a Facet (i.e. "Major Programs"). To edit the available list of Facet values, click the Manage Taxonomy Facet Values link.

Activation Toolkit

There are six built-in tools to help you manage your Taxonomy Activations according to best practices:

Taxonomy Activation and Usage Report

A simple report of currently active Terms and the number of times each is used in the database (this may include multiple uses by the same record, e.g. within multiple linked Terms).

Drill-Down Activation Tool

This is the main tool for managing Activations. It allows you to use a drill-down type interface to review all your current activations, and change them with one click. Basic rules are enforced, such as not being allowed to inactivate a Term in use, and only allowing terms to be rolled-up as Use References if there is a higher-level Active Term.

Preferred Taxonomy Terms Tools

The Preferred Taxonomy Terms tools allow you to input a list of Terms that are preferred for activation (using Manage Preferred Taxonomy Term List) and then evaluate your current compliance to that list of Preferred Term activations (using Preferred Term Compliance Report).

Multi-Level Activation Report

Best practice is to activate your Taxonomy at only one level on each branch. This means for each Active Taxonomy Term, there will be no other Active Terms either above or below it in the hierarchy. Use this report to help you identify branches on which you have activated Terms at more than one level.

Activation Recommendation Report

This report uses some general best practices, along with your current usage of the Taxonomy, to make specific recommendations about Terms you should activate or deactivate. You may accept some or all of the recommendations, and the tool can automatically apply your choices.

Manage Related Concept Values

Each Term can be associated with any number of Related Concepts (i.e. "Domestic Violence "). To edit an existing Concept, select it from the drop-down list and click "View / Edit Concept". To create a new Concept, choose the ">> CREATE NEW <<" option from the drop-down list.
