Each Subject Term can be associated with a Source which is meant to represent where the term originated from (i.e. "InformOntario" or "Local"). This is a general explanation of where the term originated from, unlike the Modified By information that indicates a particular person who last made modifications to the Term. This page allows you to add new source values, as well as edit or delete existing values. Use the "Return to Manage Thesaurus" link at top of page to return to thesaurus management main page.

Add, Update and Delete

The entire Source form (all values) are added, updated, or deleted in a single submission. There is one "Update" button that must be used to accept all changes. Remember: you must always submit all your changes via the Update button, whether you are adding, updating, or deleting.

Add: To create a new entry, click the "Add New Item" button to insert a row for entering new data. You may enter multiple rows per submission.

Delete: To delete an existing entry, check the box in the "Delete" column. If a checkbox is unavailable, it is because the item is in use and cannot currently be deleted.
