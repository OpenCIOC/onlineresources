This page appears after you have used the"Find and Replace Tool ", "Prepend / Append Tool ", "Clear Field Tool ", "Checklist Tool", "Name Tool", "Contact Tool", or "Record Note Tool" on one or more records in a result set.

In Alert colour text on this page, you will be notified of how many records were altered. Pay careful attention to this. If the number indicated does not match the number of records you attempted to perform the function on, then one or more records of the records were not modified in the operation. This could occur, for example, if you your Find and Replace operation is looking for the word "information" in 10 records but it only appears in 8 of them.

From this page you can Return to Previous Search Results, which returns the results of the last search you performed before the Find and Replace operation; The List of Altered Records link (if available) displays the records impacted by the Find and Replace operation; The Original List of Records for Find/Replace link will display a list of the records you selected to perform the Find and Replace operation on.

Important Note: the Find and Replace tool cannot perform validation on any of the modifications you make to your data through this tool. Please use Find and Replace with extreme caution and save the Find and Replace report for future reference in case you discover a problem with your modifications. You may wish to print this report for reference.
