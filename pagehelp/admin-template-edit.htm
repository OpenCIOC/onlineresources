The Add / Edit Design Templates pages allow changes to any previously created Template or the creation of a new Design Template. Design templates bring together various design elements (layouts, colours, styles) to help you to change the look of any View in your database.

If you are editing or copying an existing an existing template, the content of the existing template will be in the form fields; otherwise you will have a blank form with which to create a new Template.

The primary elements to include in the design of a template from this page:

  • external links to bring in logos, icons, style sheets (CSS), and JavaScript files
  • copyright notice
  • selection of fonts, colours, backgrounds etc.
  • selection of a header and footer layout, and optional menu items for each
  • selection of a search (main page) layout, and optional menu items

The second half of the template setup - everything after the selection of layouts - refers to various jQuery UI elements. jQuery is a JavaScript library used extensively within CIOC Software, and jQuery UI is a complementary styling library. For more information on jQuery UI, visit http://jqueryui.com/.

More information on the creation/editing of templates can be found in the references (including training video) on the CIOC Community site, located at http://community.cioc.ca/. Any questions or difficulties encountered while creating a template should be posted to the CIOC Community Message Board where other users or CIOC Help Desk staff will assist in resolution of specific questions or difficulties. Help with style sheets, custom html headers and footers, JavaScript, etc. can not be addressed within this help document and should be posted to the discussion board referenced above.
