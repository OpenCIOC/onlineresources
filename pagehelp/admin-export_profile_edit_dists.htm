The Manage Export Distribution Codes page allows you to select which Distribution codes will be included in the Distribution field of the records you export. Check or uncheck the boxes next to the Distribution codes you want to include or exclude from this Export Profile, and click the "Submit" button at page bottom to save your changes. It is possible that there will be no Distribution codes available to select from this page if none exist within your application.

Important Note: Selecting particular Distribution codes for your profiles does not impact which records you will be exporting, only the contents of the Distribution field you are sending. Records to send are selected at the time of export.

Submitted changes take effect immediately. It is not necessary to submit the main Edit Export Profile form to save your changes.
