This page allows you to edit the details of an Export Profile, including which data elements will be included, and in which View(s) it can be accessed.

General Profile Setup

  • Language - specify the record language(s) to be exported (currently only English and/or French may be selected for export).
  • Profile Name - unique name for the Profile. Used to select the profile during the export process.
  • Source Database Info - specify a name for your database, a link to your database or website, and the publicly-accessible View to which you would like the recipient to submit change requests for the records (this will create a link to the feedback form in that View).
  • Privacy Profiles - if you use privacy profiles in your database, select whether the recipient is allowed to have access to private information. If in doubt, do not include private information!
  • In Views - To make this Export Profile available for use, you must enable it in one or more Views. For security reasons, it is recommended that you only activate each profile in the View(s) in which it will be needed.

"Include Data": Managing the Data Elements for the Profile

The data elements that will be included in the profile are edited via a series of pop-up windows linked from this form. Changes made in the pop-up windows are processed independently from the rest of the information in the form. This section allows you to pick fields, distribution, and publication data to include, but it does not specify which records are included. Records must be selected at the time of export, and will be determined in part by the View the user is in.

  • Manage Fields - select which database fields will be included.
  • Manage Distributions - select which Distribution codes will be included in the Distribution field.
  • Manage Publications - select which Publication codes will be included in the Publication field. You will also be able to choose whether to include related Publication Description and General Heading information for each code you select.

Important Note: Selecting particular Publication or Distribution codes for your Profiles does not impact which records you will be exporting, only the contents of the Publication and/or Distribution fields you are sending.
