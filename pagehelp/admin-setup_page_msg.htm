A Page Message allows you to insert some information on a page, such as special instructions or announcements, where the information needs to occur on some pages but not others. You may also select the View(s) in which you want to display the message.

Create a New Message

To create a new Page Message, select >>CREATE NEW<< from the drop-down box and click the "View / Edit Message " button to be taken to a "Add New Page Message " page.

Edit An Existing Message

To view or edit the settings of an existing Page Message, select the Message from the list and click "View / Edit Message "
