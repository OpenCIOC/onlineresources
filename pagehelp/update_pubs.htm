The Update Publications page allows you to add a new publication code to a record in the database or edit / remove an existing publication code attached to a record. If you want to add, modify or delete a publication code used in the database, this needs to be accomplished from the Publications menu item accessible from the menu menu / basic search page. Actions performed on this page change only the chosen record.

What appears on this page will depend upon whether any publication codes are already attached to the record, and whether the attached publications have associated descriptions or general headings.

Note that your current View settings limit the Publications you are permitted to work with - you can only see, add or remove Publications that are available in the current View.

The table on this page has six columns:

  • Code: Any publication codes already attached to this record will appear in this column.
  • Name: The "friendly" name for this Publication, if one exists.
  • Has Description: If a Publication has an associated Publication Description for this record, three asterisks (***) will appear in this column.
  • Has Headings: If a Publication has an associated General Heading for this record, three asterisks (***) will appear in this column.
  • Has Feedback: If this record contains Publication Feedback three asterisks (***) will appear in this column.
  • Action: If there are no publication codes yet attached to this record, there will only be an Add button available. Beside each Publication Code already associated with this record will be Update and Delete buttons. Click on the Update button to move to a new screen where you can edit / add a publication description or modify the General Headings associated with this record. The Delete button allows you to remove a Publication Code (and any associated publication description and general headings) from a record. Clicking delete will take you to a new screen where you need to confirm your decision to remove the Publication Code from the record. To add a new Publication Code to the record, select the desired value from the drop-down list and click the Add Button
