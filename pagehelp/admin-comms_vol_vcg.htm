The "Volunteer Community Groups" page allows you to edit the list that will appear in the geographic legend on the search results page, showing which balls/icons match which areas. Community Groups are the link between a Community Set and the Communities that are then associated with an Opportunity record. With a Community Set, each Community must belong to exactly one Community Group, although the same Volunteer Community may be part of other Groups in other Sets. For that reason, you must add Community Groups to your database before you will be able to add Communities to your Set.

If you do not use many Communities, you may not need to group them together. In a case like this, it may make sense to set up one Community Group for each Community you want people to search with (it can have the same name as the Community); you will still need to assign that one Community to the Community Group, even if they have the same name.

If you need a further overview about the organization of Volunteer Communities, Community Sets, Community Groups, and Community Group Members, see the page help file on the main Volunteer Community administration page.

Within a Community Set, each "Group Name" must be unique; however, you could use the same Group Name again in a different Community Set.

Community Groups are each associated with a coloured "Ball" or other icon of your choice in the legend on search results pages. The number of the coloured balls is limited, and you can only use each colour once per Set. For that reason, there is a maximum total number of groups you can create (currently 11) unless you supply your own icons by providing an "Image URL". Please note that having a large number of Groups is not a good idea, because the legend and list of icons will be overwhelming to users. Consider reorganizing your groups to have larger coverage areas instead.

Editing an Existing Group

There are no issues with re-organizing or renaming your Community Groups at any time, since no Community Group information is actually stored in your records. Only one Community Group can be updated at a time. Click the "Update" button to save your changes.

This form only allows you to edit the name of the group and the icon associated with that group; editing the specific Communities that are members of this group is done from the setup form for Volunteer Community Group Members.

Adding a New Group

The area to add a new Community Group is always the last entry on the page. Enter a unique name of the Community Group and the coloured ball icon you want to use in the search results legend from the drop-down and click "Add".

Removing a Group

You can only delete a Community Group that isn't being used directly by any specific Volunteer Community Group Members. The "Delete" button will not be available beside communities that are being used. You will first need to reassign the Communities to another Group before the group value can be deleted.
