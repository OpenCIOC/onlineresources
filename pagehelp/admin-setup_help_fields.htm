The Update Field Help selection page is used to choose fields for the purpose of editing Field Help. Field Help links appear next to each field on data entry forms, including adding / updating forms, and feedback forms including "Suggest New Record". If help is available, a question mark image will appear next the field on the form which will open a pop-up window displaying the help: .

To edit the help text for any field, select it from the list and click the "Edit Field Help" link. If you want to change the friendly name of a field, this is done in a different area (through Setup > Field Display).

Clicking on the "Print Version" links will open a print friendly report of all existing field help in new window (Help can be printed in any language used in the database). Where no field help exists for a given field, the help column in the printout will be blank.

Important Notes:

  • Sample field help may be provided with an installation of the software, but you should review this help to ensure in meets local needs. If you do modify your help, you should keep a backup of your changes and notify the system administrator that you have made changes if an update to the distributed Field Help is announced.
  • You can create Field Help for custom fields like Publication Descriptions and Headings. Once you make these fields available on forms through the Publication setup, they will appear in this list for you to add help text.
  • There are several fields that are automatically included/excluded at the top of the Update Form, including the Update Date field and Record Number field; the Field Help link for these automatically added fields is always printed, so it is recommended that you make sure there is help information available for all of these fields.
