Each Taxonomy Term can be associated with any number of Related Concepts. Related Concepts are meant to represent or relate to a general theme. The Concept is an umbrella under which a number of Terms, often from different branches of the Taxonomy, can be clustered together under a general topic of interest or need. Examples of Related Concepts include: "Blindness/Visual Impairments ", "Domestic Violence", and "Homelessness". For more information about Related Concepts and other structures of the Taxonomy, you should use an official reference such as the library on 211taxonomy.org.

Status

This information is read-only and indicates whether the Concept is being used by Organization/Program records. If a Concept is being used, you will not be allowed to delete it.

Created/Modified Information

Created and Modified dates / names are set automatically by the software as records are added and updated.

Code

All Related Concepts must have a unique Code identifying them. The Code is made up of 2 letters, or 2 letters and 3 numbers (e.g. "AA" or "AA-123").

Concept Name

You must specify a unique English and/or French name for all Concepts. If you do not provide a name for a Concept in a given language, it will not be available when searching in that language.

Source

Indicate the classification system or catalogue from which the term was sourced (i.e. the AIRS/INFO LINE Taxonomy). The list of sources may be customized (see link on main Taxonomy management page).

Authorization

Indicates whether a Concept is an "official" INFO LINE Taxonomy Related Concept, or specific to the local system. We recommend that you use Concept Codes beginning with ZZ for all local Related Concepts.
