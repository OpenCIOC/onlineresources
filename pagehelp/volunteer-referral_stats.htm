Note: The main Statistics page is your starting point for generating record view / usage statistics. The form on this page only deals with Referral Statistics. For more information on Referrals, view the help for the main Volunteer Referrals page.

On or after the date: Enter the date you wish the statistical report to begin on. This date is included in the results.

Before the date: Enter the date you wish to be one day after the last date you wish to include in the statistical report. This date is not included in the results. To generate a report for exactly the previous month, click the "First of Last Month" and "First of This Month" buttons.

Threshold: Include Opportunities that have had at least this many Referrals (0 or blank will give you all Opportunities).

Organization Keywords: You can include keywords or "quoted phrases" from the Organization name to locate Referrals for specific Organizations.

Records owned by: If there is more than one record owner in the database, you can specify here to return results based on this criteria.

Show Placement Count: If you want to see a count of the number of Referrals that have been marked as a Successful Placement, select Yes for this option.

Print Version: If you want the report to be published in Print Mode, select Yes for this option. Reports run in Print Mode open in a new window.
