The management pages for Standard Email Update Text values allow you to customize sections of the email update request messages. You may create several different pre-defined texts for each type of message Community Information Organization/Program update, Volunteer Opportunity update, Volunteer Organization update for "All Opportunities"). One message for each type must be designated as the default; the default message will be the one initially selected when you create a new Email Update Request. If you have multiple Email Update Text values, you will have an opportunity to make a choice between them at the time of sending.

Changing Values at the Time of Sending

The "Subject" and "Message Body" values can be altered at the time messages are sent; if you change these values here, it will change the default values. All other setup items can only be altered from this setup screen.

Links to Record pages

Depending on the type of message, you will either have a "Detail Link" and a "Feedback Link" or an "All Opportunities Link" and a "Suggest Opportunity Link". If you prefer, you can decide to include only one of either the "Detail Link" or "Feedback Link" in your message. To remove the link you do not want to use, leave the description of that link blank; you cannot leave both descriptions blank, however. Note: if a record is not available in the View you are sending out the request for (this could be either the Default View or some other publicly-accessible View you have chosen), the system automatically generates only a Feedback Link using a pre-determined description.

Changing the Default Message

To change which text is the default for the database, you must go into the text that you want to make the default and click the "" checkbox.
