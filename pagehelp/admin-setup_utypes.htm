User Types represent the security restrictions for the user, dictating which information the user will be able to access and what changes (if any) they will be able to make to that information. Selecting a User Type will determine which View the user sees when they log in and what permissions they have to change data, export records, make setup change, etc. User Types are set individually for each module of the database (Community Information and Volunteer).

The Manage User Types page allows you to create or edit different User Types, which can then be assigned to Users. For example, you may decide to create different User Types for your database administrators, I&R staff, volunteers, or a 3rd party partner who has a specialized View.

Edit an Existing User Type

To view or edit the settings of an existing User Type, select the it from the list and click "View / Edit User Type"

Create New User Type

To create a new User Type, select >>CREATE NEW<< from the drop-down box and click the "View / Edit User Type" button to be taken to a form to add the new User Type.
