The Add / Edit Agency page is used to either add a new Agency (AKA Record Owner) to the database, or to edit information for an existing Agency. For the most part, the information displayed will be identical whether you are adding a new Agency or editing existing information. Some additional information is available when editing an existing Agency. This will be outlined below.

Adding A New Agency

Member

This field only applies in a database with multiple CIOC Members (a "Multi-Member" database). This field will be available for Global Super Users (those who control setup items shared across all memberships in the database). Once set, the Membership of each Agency cannot be changed.

Agency Code

This code consists of three capitalized letters. This code will be added to the beginning of every record number that is owned by this Agency. Because the CIOC community attempts to keep record owner codes unique among its members, it is important to ensure that you do not create a new code that is already in use by another member.

Record Owner Information

Information from this section is used to display contact information on mail/fax forms, to people providing feedback on the record; the email address is used to generate various emails from the system including feedback emails and update requests. This information must be filled out for each area of the database the owner code is used (Community Information and/or Volunteer).

Update Email:
The email address that will be attached to all outgoing email update requests for this agencies records as well as other automatically generated emails (i.e. feedback submissions). If this email is not supplied, the default module email (as specified in the General Setup) will be used instead. It is strongly recommended that an email be supplied for every Agency.
Update Phone:
The phone number that will be attached to update requests, mail/fax forms, and feedback forms for the Agency's records.
Inquiry Phone:
The number that the public should call for general inquiries (Not currently used in the software).
Agency Record #:
ALWAYS REQUIRED! Each Agency should be associated with a record in the database that can provide name and address information. The Record # is the "friendly" record number (record owner + 4 digits). Use the "Org Finder" link to locate the Record # of the record for the Agency. Enter the organization name you are looking for in the "Org Finder" pop-up window and copy the appropriate Record number from the result set returned. The record chosen can be a non-public record.

Enforce Required Fields

If selected, a user will be forced to complete all fields marked as required on the Organization / Program update form before the update will be accepted.

User Account Updates

Select the default behaviour for self-updating permissions on new accounts for this Agency. If there are accounts in the database for this Agency where the user is not permitted to update their own personal details or password, ensure that the email field is filled in so that the user can request changes to their account.

Editing An Existing Agency

The form to edit an existing Agency code provides all the fields listed in "Add New" above, as well as a status section and a search link to review the records associated with the given Agency. Note that the search will only retrieve records in the current View you are working in.

Status

The Status section indicates how the Agency Code is being used throughout the database, by users and records.

If the Agency is not assigned as a record owner for any records, a delete button will be available. If you wish to delete this Agency and there is no delete button available, you will first need to remove the Agency code (Record Owner) from any records or users it is associated with and then return to this page.
