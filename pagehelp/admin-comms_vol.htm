Volunteer Community information is organized into several layers:

  1. Communities: where Opportunities are offered. Drawn from the master list of Communities that is shared with the Community Information module of the database.
  2. Community Sets: a Community Set allows you to take a selection of Communities and Volunteer Opportunities and make them available in a particular View.
  3. Community Groups: each Community Set has its own set of Community Groups which cluster one or more Communities together under the title of your choice.
  4. Community Group Members: each Community Group contains one or more values from the list of Communities, for use as defaults in search and data entry forms.

The Volunteer Opportunity Community Management tool allows you to review the assignment of Volunteer Opportunities to specific Community Sets and add or remove a number of Opportunities from a Set at once.
