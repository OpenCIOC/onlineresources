The Manage Communities page allows you to add a new Community to the database or edit an existing Community. You may also upload a new file from the Communities Repository to update your entire list.

There are two types of communities:

  • Standard Hierarchy Communities - each of these communities has exactly one parent.
  • Alternative Search Areas - these provide a way to have areas that do not fit into the hierarchy, for example because they cross boundaries or group a number of regions together for a special designation.

Add a New Community

New Communities or Alternative Search Areas can be created by clicking on the appropriate links at the top of the page.

CIOC software generally ships with an existing set of "Authorized" community values, based on the values in the CIOC Communities Repository. Any new communities created by users will be marked as "Local"; local terms in the drop-down are designated with a *.

Edit an Existing Community

Select the name of the Community you wish to edit from the drop-down menu and click the "View/Edit Community" button.
