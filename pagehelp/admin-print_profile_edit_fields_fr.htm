The Print Profiles Find and Replace Tool will give you the opportunity to manipulate your data before it is displayed. Find and Replace commands operate on individual fields, so you if you want a command run on multiple fields you will need to set up a command for each one. You can have as many commands as you want, and specify the order in which they are run.

What Can Regular Expressions Do?

Regular Expressions are an incredibly powerful tool. Two somewhat-complex (and hopefully useful) examples of what can be done are included in the sections that follow. Although you may not feel skilled in writing these commands yourself, you may be able to have someone with the technical knowledge produce the commands for you. Documentation on Microsoft's Regular Expression functionality can be found through Microsoft's online libraries.

Example 1: Creating a bulleted list

This example will transform the following formatting:

* a bullet
* another bullet
* a third bullet
  • a bullet
  • another bullet
  • a third bullet

Relative Order Find Replace With
1 ((\n|^)\* (.*))+ <ul>$&</ul>
2 (\n|^)\* (.*) $1<li>$2</li>

Example 2: Removing/Changing Hyperlinks

The following example will remove a hyperlink and replace it with bold formatting instead, for example:

Yahoo!   becomes   Yahoo!

Find Replace With
<a [^>]+>(.*)<\/a> <b>$1</b>

The next example will grab the web site address, strip the http:// if it exists, and place it next to the name in brackets, for example:

Yahoo!   becomes   Yahoo! (www.yahoo.com)

Find Replace With
<a href="([^"]+)"[^>]*>(.*)<\/a> <b>$2</b> ($1)
