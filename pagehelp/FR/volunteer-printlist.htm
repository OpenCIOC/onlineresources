Imprimer une liste de dossiers

Les profils d'impression sont associés à des vues spécifiques ; un super-utilisateur peut créer ou modifier des profils d'impression et les assigner à des vues en allant dans Configuration > Profils d'impression. S'il n'y a pas de profil d'impression dans votre vue, vous ne pourrez pas utiliser cette fonctionnalité.

Étape 1 : sélectionner vos dossiers et votre profil

Les outils d'impression du module Bénévolat du logiciel ne sont disponibles actuellement que par l'action « Afficher les cases à cocher » sur une ensemble de dossiers dans les résultats de recherche. Après avoir sélectionné les dossiers, la première étape sera de sélectionner le nom du profil d'impression que vous souhaitez utiliser et choisir un ordre de tri pour le rapport final.

Étape 2 : personnaliser votre message

Après avoir sélectionné les dossiers que vous voulez imprimer et un profil d'impression, vous aurez l'occasion de modifier, augmenter ou retirer le message de rapport par défaut pour le profil d'impression sélectionné. En fonction des paramètres du profil, le message peut apparaître une fois en haut du rapport et avant chaque dossier dans le rapport.

Étape 3 : consulter votre rapport

Après avoir personnalisé votre message, cliquez sur « Suivant » pour ouvrir votre dossier dans une nouvelle fenêtre. Après avoir imprimé ou sauvegardé votre rapport, vous pouvez fermer la fenêtre du rapport et revenir à la fenêtre précédente pour continuer votre travail dans la base de données.
