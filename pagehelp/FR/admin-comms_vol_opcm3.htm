Cette page contient les résultats de votre recherche sur l'écran précédent de l'outil de gestion des ensembles de communautés. Depuis cette page, vous pouvez cocher les cases à côté d'un dossier ou groupe de dossiers, et les ajouter ou les retirer de l'ensemble de communautés sur lequel vous travaillez en utilisant le bouton en bas du formulaire.

Vous pourrez voir la colonne d'alerte, le titre du poste, le nom d'organisme et la classification existante de l'ensemble de communautés des dossiers dans vos résultats. Si un dossier d'occasion est visible dans votre vue actuelle, il y aura un lien pour ouvrir les détails du dossier dans une nouvelle fenêtre. Vous ne pouvez pas voir les détails complets des dossiers qui ne sont pas visibles dans votre vue actuelle ; le cas échéant, il peut être utile de changer de vue pour optimiser votre capacité à consulter les détails complets des dossiers que vous essayez d'ajouter ou de retirer.
