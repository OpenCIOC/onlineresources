La zone Gérer les titres de page vous permet de remplacer le titre intégré d'une page par une valeur personnalisée. Cette option est disponible uniquement pour un nombre de pages limité pour le moment. Ce titre apparaît dans la barre de titre de votre navigateur, ainsi que dans la zone de titre de l'en-tête de la page (tel que précisé dans le modèle de conception).

Pour déterminer la page que vous souhaitez modifier, examinez la barre d'adresse de votre navigateur lorsque vous êtes sur la page. Notez toute la partie de l'adresse située après l'adresse de base (tel que précisé dans les options générales de configuration). Par exemple, votre page de connexion sera « login.asp », tandis que cette page sera « admin/pagetitle ». Si la page sur laquelle vous souhaitez travailler n'est pas dans la liste, il n'est pas possible de remplacer le titre pour le moment. Si besoin, vous pouvez demander aux développeurs CIOC d'ajouter la page à la liste des pages dont le titre peut être remplacé, par l'entremise du babillard CIOC (en anglais).
