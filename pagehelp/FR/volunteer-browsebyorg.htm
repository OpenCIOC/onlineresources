La page « Parcourir les organismes offrant des possibilités de bénévolat » affiche une liste des organismes ayant des occasions de bénévolat référencées dans la base de données. Cliquez sur une lettre du menu « Choisir une lettre » pour consulter une liste de tous les organismes dont le nom commence par cette lettre. Il est également possible de lister tous les organismes alphabétiquement sur une seule page en utilisant le lien « Afficher tout ».

Le nombre entre parenthèses à la suite du nom d'organisme indique combien d'occasions sont disponibles au sein de l'organisme (à noter qu'il s'agit seulement des occasions dans votre vue actuelle). Pour consulter la liste de toutes les occasion disponibles au sein d'un organisme particulier, cliquez sur le nom de l'organisme.
