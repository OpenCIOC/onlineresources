Remarque : la page d'aide sur le formulaire de recherche principal sur les statistiques peut s'avérer utile pour obtenir un aperçu général des différents types de statistiques conservées dans votre base de données CIOC et savoir comment les utiliser de manière adéquate.

Cette page fournit une liste des tous les dossiers de la base de données avec leur utilisation par le public (utilisateurs non connectés) et l'utilisation totale (utilisateurs connectés et public). Les dossiers sont triés par ordre alphabétique du nom d'organisme, puis du titre du poste. Si le dossier est accessible dans votre vue actuelle, vous pouvez le consultez en cliquant sur le lien du numéro de dossier (remarque : ce lien n'est pas disponible quand vous consultez le rapport en mode impression).

Si vous constatez une entrée sans information au bas du formulaire, cela représente les statistiques qui sont associées à des dossiers ayant été supprimés de manière permanente de la base de données.

À noter que ces valeurs sont basées sur les statistiques sur les dossiers consultés internes au logiciel ; les statistiques sur les dossiers consultés ne sont pas conservées lorsque vous passez en mode impression ou changez de vue temporairement, pour ne pas gonfler les statistiques si quelqu'un consulte plusieurs versions d'un même dossier pour les mêmes raisons.

En haut de la page figurent des liens vers plusieurs autres outils statistiques : Page d'accueil des statistiques, 50 premiers dossiers, Consultations par agence, Rapports automatiques et Supprimer les statistiques anciennes.
