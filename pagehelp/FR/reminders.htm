Une initiation aux rappels

Le système de rappels vise à aider les utilisateurs à créer des listes de tâches pour eux-mêmes ou d'autres utilisateurs, ainsi qu'à offrir un moyen de communication avec d'autres utilisateurs de CIOC. Les rappels peuvent porter sur des tâches générales ou ils peuvent être associés à un ou plusieurs dossiers. Vous pouvez travailler avec les rappels en cliquant sur le lien Rappels dans la barre supérieure ou (dans le cas de rappels associés à des dossiers précis) par la page de détails du dossier. Vous pouvez également créer un rappel pour un groupe de dossiers par vos résultats de recherche.

Les rappels ont plusieurs états possibles :

  • Un rappel en court est une tâche active qui n'est pas en retard. Cela signifie qu'il n'a pas de date de début spécifique ou que la date de début est déjà passée. Il n'a pas de date d'échéance ou sa date d'échéance est dans le futur. Si vous avez un rappel en court et pas de rappel en retard, vous verrez une icône de rappel à côté du lien de rappels dans votre barre supérieure, ainsi que dans les dossiers auquel le rappel s'applique.
  • Un rappel en retard a une date d'échéance qui est passée. Si vous avez un rappel en retard, vous verrez une icône de rappel à côté du lien de rappels dans votre barre supérieure, ainsi que dans les dossiers auquel le rappel s'applique.
  • Un rappel en attente n'a pas encore atteint la date de début.
  • Un rappel rejeté a été retiré de la liste, habituellement car la tâche a été accomplie ou la communication a été lue. Les rappels ayant été rejetés pour tous les utilisateurs désignés sont régulièrement supprimés de la base de données, au moins une semaine après le dernière rejet.

Travailler avec votre liste de rappels

La liste de rappels fonctionne de la même manière, que vous travailliez dans la page de détails des dossiers ou sur la page des rappels. Lorsque qu'ils travaillent sur la page des rappels, certains utilisateurs peuvent aussi accéder à la section Avis admin. Il s'agit de la section où les super-utilisateurs globaux reçoivent les demandes de changements de configuration dans une base de données multimembre. La nouvelle page de rappels remplace l'ancienne page Avis.

Lorsque vous consulterez votre liste de rappel pour la première fois, vous verrez tous les rappels en cours et en retard. Les rappels en retard incluront la date d'échéance du rappel en couleur d'alerte. Chaque liste est divisée en fonction des rappels créés pour moi (assignés à vous par vous-même ou un autre utilisateur) ou créés pour d'autres (créés par vous, mais assignés à un autre utilisateur). Si vous avez des rappels dont la date de début n'est pas encore passée, vous pouvez cliquer sur le lien Rappels en attente pour consulter la liste des futurs rappels ; cliquez de nouveau pour masquer la liste.

Par défaut, les rappels rejetés ne sont pas inclus dans votre liste. Si vous avez un rappel rejeté qui n'a pas encore été supprimé, vous pouvez cliquer sur le bouton Afficher les rejets pour les inclure dans votre liste. Le fait de cliquer sur le bouton de nouveau masquera les rappels rejetés.

Les actions ou informations suivantes sont accessibles depuis votre liste de rappels :

  •  Info rapide - Cliquez sur l'icône de page pour voir rapidement qui a créé le rappel (si ce n'est pas vous), à qui le rappel est assigné, quand il deviendra actif (pour les rappels en attente).
  •  Modifier - Si vous avez l'autorisation de modifier le rappel, cliquez sur l'icône de dossier pour ouvrir le rappel et consulter les détails ou pour modifier le rappel.
  •  Rechercher - Si le rappel est associé à des dossiers, passez la souris au-dessus de l'icône de loupe pour voir combien de dossiers d'information communautaire ou d'occasions de bénévolat sont associés au rappel. Si le rappel est associé à des dossiers des deux modules, vous verrez deux icônes. Cliquez sur l'icône pour lancer une recherche contenant ces dossiers ; à noter que vous ne pouvez retourner que les dossiers qui sont accessibles dans votre vue actuelle.
  •  Rejeter - Si vous avez accompli la tâche ou lu la communication, vous pouvez supprimer le rappel de votre liste en utilisant l'icône X. Vous pourriez avoir le choix de supprimer le rappel pour tout le monde ; si c'est le cas, faites-le seulement si vous avez entièrement accompli la tâche et si les autres n'ont plus besoin du rappel.
  •  Restaurer - Vous pouvez restaurer un rappel rejeté de votre liste normale en utilisant l'icône rafraîchir. À noter que les rappels rejetés sont supprimés régulièrement, au moins une semaine après le rejet du rappel par tous les utilisateurs associés.
  •  Priorité élevée - Si vous êtes assigné à un type de note hautement prioritaire d'un dossier, vous verrez l'icône de haute priorité à côté du rappel dans votre liste.

Configurer vos rappels

Que vous configuriez votre rappel depuis la page de détails d'un dossier, de la page de rappels ou par vos résultats de recherche, le formulaire de modification du rappel est exactement le même.

Si vous souhaitez associer votre rappel à un dossier ou plus, vous pouvez le faire dans la section Dossiers en saisissant les numéros de dossiers. Vous pouvez saisir une combinaison de dossiers d'information communautaire ou d'occasions de bénévolat. Vous pouvez passer votre souris sur des dossiers ajoutés auparavant pour voir le nom complet du dossier (cela ne s'applique aux dossiers nouvellement ajoutés avant d'avoir soumis le formulaire). Cliquez sur l'icône de recherche si disponible pour retourner un écran de résultats de recherche avec les dossiers associés au rappel.

De manière optionnelle, vous pouvez préciser que votre rappel s'applique uniquement à une langue spécifique du dossier. Par exemple, si vous choisissez uniquement l'anglais, le rappel s'affichera uniquement si vous consultez la version anglaise du dossier.

Si vous ne précisez pas explicitement à qui s'adresse le rappel, il est entendu que le rappel s'applique juste à vous. Vous pouvez préciser que le rappel est pour vous ou d'autres utilisateurs de la base de données. Les super-utilisateurs peuvent assigner des rappels aux membres de leur agence. Veuillez ne pas inonder les autres utilisateurs de votre base de données avec des rappels ; assignez les rappels uniquement aux utilisateurs qui doivent les consulter ou en accomplir la tâche.

Certains rappels exigent que chaque utilisateur les consulte ou en accomplisse la tâche, alors que d'autres n'ont besoin que de l'un des utilisateurs. Utilisez la section Rejet pour indiquer si un utilisateur peut rejeter le rappel juste pour lui-même ou pour tous les utilisateurs (p.ex., parce qu'il a accompli la tâche et que plus personne d'autre n'a besoin de l'accomplir).

Une date de début s'avère utile quand vous savez qu'une tâche n'a pas besoin d'être accomplie dans l'immédiat. En mettant une date future dans le champ Débit de votre rappel, vous pouvez choisir d'exclure ce rappel de vos tâches actives jusqu'au moment où vous devrez vraiment travailler à accomplir la tâche. La date de départ doit généralement être posée bien avant la date d'échéance afin de vous permettre d'accomplir la tâche avant son échéance. Ceci est un champ facultatif.

Si vous devez accomplir votre tâche avant une date spécifique, vous pouvez saisir une date d'échéance. Si la tâche n'est pas rejetée avant la date d'échéance, elle deviendra un rappel en retard et une alerte apparaîtra sur votre barre supérieure ainsi que le dossier auquel il s'applique.

Comme avec le champ Note interne, vous pouvez choisir un type de note pour votre rappel. Ces types de notes sont partagées avec le champ Note interne et vous pouvez avoir de nombreux types ou aucun dans votre base de données. Des types supplémentaires peuvent être configurés dans les paramètres de types de notes dans les listes de contrôle générales partagées. Si le type de note est marqué comme hautement prioritaire, une icône de point d'exclamation apparaîtra à côté du rappel, indiquant qu'il s'agit d'une priorité.

La seule information que vous devez remplir dans un rappel est la section des Détails. Il s'agit de la section où vous saisissez les détails spécifiques de votre tâche ou communication.
