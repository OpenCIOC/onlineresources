La page Gérer les champs d'exportation vous permet de sélectionner quels champs à inclure dans les dossiers à exporter. Cochez ou décochez les cases correspondantes aux noms des champs que vous souhaitez inclure ou exclure de ce profil d'exportation, puis cliquez sur le bouton « Soumettre » en bas de page pour sauvegarder vos modifications. Tous les champs ne sont pas disponibles à l'exportation et les champs Distribution et Publication sont gérés séparément du reste des champs dans le profil d'exportation.

Les changements soumis sont mis en œuvre immédiatement. Il n'est pas nécessaire de soumettre le formulaire principal de modification du profil d'exportation pour sauvegarder vos modifications.
