Cette page est utilisée soit pour ajouter une nouvelle communauté à la base de données, soit pour modifier les informations d'une communauté existante. La plupart du temps, les champs sont identiques que vous ajoutiez une communauté ou modifiez une communauté existante. De l'information supplémentaire est disponible pour modifier une communauté existante. Voir plus bas.

N'oubliez pas qu'il s'agit d'un système de classification partagé, utilisé à la fois dans les modules Information communautaire et Occasions de bénévolat. Soyez respectueux des agences partenaires lorsque vous modifiez ces valeurs.

Zones de recherche alternatives et Communautés appartenant à la hiérarchie normalisée

Il existe deux types de communautés, qui peuvent toutes être ajoutées aux formulaires de recherche et aux dossiers :

  • Communautés appartenant à la hiérarchie normalisée - celles-ci constituent la liste de base des communautés, regroupant de petites régions sous de plus grandes régions. Chacune de ces communautés possède exactement une communauté mère.
  • Zones de recherche alternatives - celles-ci offrent un moyen de décrire des régions qui ne correspondent pas à la hiérarchie, par exemple parce qu'une zone s'étend au-delà de frontières ou regroupe plusieurs autres régions sous une appellation spéciale, comme une zone de services de santé.

Le formulaire de saisie des données sur cette page dépend du type de communauté que vous modifiez.

Ajouter une nouvelle communauté

Remarque : les communautés que vous ajoutez à la base de données, autrement que par l'importation du registre officiel, sont considérées locales ou non-autorisées. Toute communauté créée localement sera disponible dans votre système, mais ne sera pas disponible dans les autres bases de données CIOC. Afin d'« autoriser » vos communautés, veuillez soumettre vos demandes de changement par le Registre des communautés CIOC.

Nom

Saisissez le nom de la nouvelle communauté ici. À noter que si vous essayez de saisir un nom de communauté qui est déjà utilisé dans la base de données, vous recevrez un message d'erreur lors de la soumission du formulaire. On vous demandera alors de retourner au formulaire et de réessayer. Vous pourriez devoir faire preuve d'imagination pour vos noms de communautés si vous trouvez des doublons dans la base de données. Si vous utilisez une version multilingue du logiciel, vous pouvez également saisir les noms dans d'autres langues, si ceux-ci sont différents.

Affichage

Vous pouvez saisir une version différente du nom de la communauté, pour l'utilisation dans les formulaires de recherche et sous la valeur « Situé à » des résultats de recherche. Cela s'avère particulièrement utile si vous devez modifier ou allonger le nom afin de créer un nom de communauté unique. Si vous utilisez une version multilingue du logiciel, vous pouvez également saisir une version en français du nom d'affichage, si celui-ci est différent du nom anglais.

Communauté mère

Chaque communauté saisie appartient à une communauté mère, sauf la communauté la plus élevée dans la hiérarchie (Inconnu). Quand vous saisissez un nouveau nom de communauté, vous devez également assigner une communauté mère. Cette information est nécessaire pour les recherches basées sur les champs « Lieu » et « Secteurs desservis » dans la base de données. Si vous ne savez pas quelle communauté mère choisir, vous pouvez utiliser l'outil Localisateur de communautés qui est attaché aux formulaires de création / mise à jour (dans les champs « Lieu » et « Secteurs desservis »), afin de déterminer cette information. Sélectionnez une valeur dans le menu déroulant « Communauté mère » pour assigner un nom de communauté nouvellement ajouté.

Province, État ou Pays

De manière optionnelle, vous pouvez préciser la province, l'état ou le pays en sélectionnant dans la liste déroulante proposée. Cela est nécessaire uniquement pour distinguer les cas où plusieurs communautés ont le même nom.

Noms alternatifs

Vous pouvez donner un nom alternatif ou plus à chaque communauté ; ceux-ci doivent être différents des autres noms de communauté dans la base de données. Les noms alternatifs n'apparaissent nulle part dans le logiciel, mais sont utilisés lors des recherches par communauté, comme sur la boîte « Autre communauté » sur la page de recherche, ou lors de la saisie des données.

Rechercher des communautés

En mode de gestion d'une zone de recherche alternative, une section supplémentaire est ajoutée au formulaire pour vous permettre de sélectionner quelles communautés dans la hiérarchie seront recherchées, chaque fois que cette zone de recherche alternative est utilisée. Le fait d'ajouter une zone de recherche alternative à un dossier ou un formulaire de recherche revient à ajouter toutes les communautés sélectionnées à cette section.

Modifier une communauté existante

Le formulaire pour modifier une communauté existante affiche tous les champs listés plus haut dans « Ajouter une communauté », ainsi qu'une section Statut et un lien de recherche pour consulter les dossiers associés à la communauté en question.

Statut

La section Statut indique la manière dont la communauté est utilisée par les dossiers d'organismes / programmes ou les dossiers d'occasions de bénévolat, d'autres communautés ou des éléments connexes comme les lignes de bus ou les quartiers.

Si la communauté n'est pas utilisée, un bouton de suppression sera disponible. Si vous voulez supprimer cette communauté et qu'il n'y a pas de bouton de suppression disponible, vous devrez d'abord retirer la communauté de tous les dossiers ou utilisateurs auxquels elle est associée, puis revenir sur cette page.

Remarque : il n'est pas recommandé de supprimer une communauté autorisée. Si vous souhaitez suggérer des modifications sur les communautés autorisées, veuillez soumettre vos demandes de changement par le Registre des communautés CIOC.

Travailler avec les langues

Dans les bases de données multilingues, le logiciel utilisera par défaut les langues de travail normales de votre base de données (en général, anglais et français). Si des langues supplémentaires peuvent être modifiées, vous aurez accès à des cases à cocher en haut du formulaire ; sélectionnez les langues supplémentaires que vous voulez modifier en cochant la case à côté de la langue pour l'ajouter au formulaire.

Remarque importante : si vous ne précisez pas de valeur dans chaque langue, le nom de la première langue disponible sera utilisé. Il n'est pas nécessaire de copier le nom dans chaque langue, mais ce n'est pas grave de le faire non plus.
