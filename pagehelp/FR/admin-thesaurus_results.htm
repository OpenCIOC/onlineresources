Modifier l'affichage des résultats

Si vous souhaitez modifier l'affichage de vos résultats de recherche, vous devez revenir au formulaire de recherche. En bas du formulaire de recherche, vous verrez des options pour modifier l'affichage de vos résultats de recherche.

Vous pouvez trier la liste des sujets par le terme du sujet (anglais ou français, en fonction de la langue d'affichage actuelle) ou par la date de dernière modification du sujet, avec les sujets modifiés le plus récemment en premier.

Par défaut, l'affiche des résultats comprendra :

  • Le terme du sujet (dans la langue actuelle)
  • Le terme préféré (si le terme n'est pas employé, quel terme utiliser à la place)
  • Le nombre de dossiers utilisant le sujet. Le décompte varie en fonction de vos autorisations et si vous travaillez dans une base de données avec d'autres membres CIOC.
    • Dans une base de données à membre unique, le décompte inclut tous les dossiers dans toutes les vues.
    • Dans une base de données multimembre, si vous êtes super-utilisateur local, vous verrez le décompte des dossiers appartenant à votre membriété CIOC uniquement. Cela n'inclura pas les dossiers qui sont partagés avec vous.
    • Dans une base de données multimembre, si vous êtes super-utilisateur global, vous verrez un décompte de vos propres dossiers entre parenthèses (#), ainsi qu'un décompte entre crochets de tous les autres dossiers de la base de données appartenant à d'autres membres CIOC [+#].

En sélectionnant Afficher toute l'information sur le sujet, vous verrez toutes les informations disponibles sur le sujet, y compris :

  • Les termes génériques, spécifiques et apparentés
  • Les informations Employé pour - où le terme donné est un terme préféré à un autre terme non utilisé.
  • Informations sur la gestion de données.
