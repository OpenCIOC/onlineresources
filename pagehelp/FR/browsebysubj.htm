Cette page liste tous les sujets et sujets associés qui commencent par la lettre sélectionnée dans le menu « Parcourir par sujet ».

Si le message qui apparaît sur cette page est : « Il n'y a pas de dossier correspondant à vos critères », alors il n'existe pas de sujet commençant par la lettre sélectionnée (à noter qu'en fonction des disponibilités de la vue actuelle, vous pourriez voir ou ne pas voir les sujets qui ne sont pas associés à des dossiers d'organismes / programmes). Dans ce cas, utilisez le menu « Choisir une lettre » pour sélectionner une autre lettre. Sinon, retournez à la page de recherche principale ou utilisez le bouton retour de votre navigateur pour débuter une nouvelle recherche.

Le nombre entre parenthèses à la suite d'un sujet indique combien de dossiers dans la base de données correspondent exactement au sujet. Le nombre représente uniquement le nombre de dossiers dans la vue actuelle de votre base de données ; il peut exister d'autres dossiers de la base de données associés à ce sujet, mais vous n'avez pas accès à ces dossiers. Même si un sujet n'a pas de dossier associé (0), vous pouvez cliquer dessus et voir les sujets génériques, spécifiques ou associés ayant des dossiers associés.
