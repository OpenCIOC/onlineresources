L'outil d'activation de la Taxonomie vous permet de travailler sur vos activations à travers la base de données entière en une seule fois, plutôt que d'avoir à modifier l'activation de chaque terme individuellement. L'interface est très similaire à une recherche hiérarchique. Il y a trois actions possibles disponibles pour les termes : Activer, Désactiver ou Remonter. Le logiciel affiche uniquement les icônes d'action des états d'activation permis et, dans de nombreux cas, vous ne pourrez pas modifier le statut (comme pour les termes utilisés dans des dossiers).

Le statut existant d'un terme est visible en fonction de l'apparence. Dans les paramètres ordinaires du logiciel, cela signifie que les termes inactifs sont en rouge, avec une icône de flèche montante affichés devant les termes remontés. Votre base de données pourrait utiliser un style différent ou supplémentaire pour chacun de ces statuts ; veuillez donc utiliser votre configuration locale comme guide.

Les icônes pour modifier le statut d'un terme apparaissent à la fin du terme et comprennent une coche verte pour activer, un X rouge pour désactiver et une flèche orange pour remonter.

Les modifications d'activations sont mises en œuvre immédiatement ; il n'est pas nécessaire de soumettre les changements.

Si vous travaillez dans une base de données multimembre, en fonction de vos autorisations, vous pouvez avoir la capacité de modifier les activations à la fois au niveau global (s'appliqueront à tous les membres de la base de données) et au niveau local (s'appliqueront uniquement à votre membriété). Lorsque vous travaillez sur les activations locales, vous pouvez activer et désactiver (pas remonter) et vous pouvez uniquement activer les termes qui ont été activés globalement.
