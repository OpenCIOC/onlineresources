Profils d'impression : les fondamentaux

Créer un profil d'impression vous permettra de générer une liste de dossiers bien formatés pour l'impression, sans information de menu superflue, et intégrant tous les champs que vous aurez sélectionnés. Contrairement aux résultats de recherche, où la mise en page est un tableau de champs à l'horizontal, l'outil d'impression affiche les dossiers dans un format vertical personnalisable ; chaque champ est affiché sur sa propre ligne (bien qu'il soit possible de combiner des données de plusieurs champs en une seule). Les profils d'impression sont associés à des vues particulières.

L'outil d'impression fonctionne en HTML. Cela signifie que :

  • Il est très utile de connaître le HTML et les feuilles de style en cascade (CSS) pour utiliser les fonctionnalités plus avancées de l'outil. Si vous ne connaissez pas le HTML, vous pourrez quand même utiliser l'outil. D'autres membres peuvent vous aider en fournissant des exemples de feuilles de style (CSS) et d'options de configuration.
  • Dans la plupart des cas, les données des dossiers s'afficheront de la même manière que sur les pages de détails des dossiers. Tous les champs qui supportent la saisie de HTML pour l'affichage sur la page de détails de dossier supportent également le HTML dans l'outil d'impression, et les mêmes règles de formatage s'appliquent. Tout code HTML supplémentaire inséré automatiquement dans les pages de détails de dossiers par le logiciel (comme un lien vers un site Internet) sera supprimé lors de l'utilisation de l'outil d'impression.
  • Le concept de « page » est limité en HTML, car l'affichage à l'écran peut varier d'un utilisateur à l'autre. Ainsi, la capacité à ajouter des éléments de page spécifiques, comme des en-têtes et des pieds de pages, la numérotation de page, etc., est limitée.De nombreux navigateurs web offrent des options de configuration d'impression personnalisables qui peuvent vous permettre d'ajouter des en-têtes, pieds de page, numérotation, etc.
  • Votre rapport généré par l'outil d'impression devrait pouvoir être utilisé par la plupart des programmes de traitement de texte principaux, y compris Microsoft Word. Des logiciels de traitement de texte plus récents interprèteront plus facilement le HTML et les styles que des versions plus anciennes. Vous pouvez utiliser un logiciel de traitement de texte pour faire des actions comme personnaliser les emplacements des sauts de page, ajouter des en-têtes et des pieds de page, créer des colonnes et ajouter la numérotation des pages.

Modifier votre profil d'impression

Votre profil d'impression se gère en deux parties :

  1. Vos configurations générales du profil (y compris les feuilles de style CSS, les en-têtes et pieds de page, etc.).
  2. La configuration des champs du profil, où vous sélectionnez vos champs et précisez la manière dont ils sont affichés.

Cette page traite particulièrement de vos configurations générales du profil ; vous pouvez trouver de l'aide pour établir vos champs dans l'aide sur la page de configuration des champs. On peut trouver le lien vers les données de gestion des champs au bas de la page de configuration générale ; vous devriez toujours soumettre les changements apportés à la configuration générale du profil avant de cliquer sur le lien pour gérer les champs.

Configuration générale du profil

Nom

Chaque profil doit avoir un nom unique. Le nom doit être assez descriptif afin qu'une personne souhaitant exécuter un rapport sache quel profil choisir. Les noms peuvent comporter jusqu'à 100 caractères maximum, mais il est fortement recommandé de limiter la longueur des noms à moins de 50 caractères.

Renseignement sur les feuilles de style / CSS

Pour profiter pleinement des capacités des outils d'impression, vous devriez préciser une feuille de style en cascade (CSS) à utiliser avec le profil. Deux échantillons de feuilles de style sont fournies avec le logiciel, que vous pouvez utiliser telles quelles, ou copier et modifier en fonction de vos besoins. L'emplacement des échantillons est :

mabasededonnees.cioc.ca/styles/printSampleA.css
mabasededonnees.cioc.ca/styles/printSampleB.css

... où mabasededonnees.cioc.ca est l'URL de votre base de données CIOC.

Dans votre feuille de style, vous pouvez créer des éléments de CLASSE pour préciser le format des différents éléments dans le rapport. Depuis la configuration générale, vous pouvez préciser le nom de la CLASSE à utiliser pour la TABLE HTML qui contiendra vos données pour tous les champs secondaires (consultez l'aide sur la page de configuration des champs pour plus d'information).

Séparateur de dossier

Vous pouvez ajouter du HTML à insérer entre les dossiers pour les séparer. Par exemple, cela peut être une ligne horizontale <HR>, ou un saut de ligne <BR>. Vous pouvez également préciser si chaque dossier doit apparaître sur sa propre page si vous imprimez depuis un navigateur... mais sachez que cette fonctionnalité peut ne pas être disponible sur tous les navigateurs et, pour le moment, n'est pas transférable lorsque vous ouvrez votre rapport dans un logiciel de traitement de texte comme Microsoft Word.

En-têtes et pieds de page

Vous pouvez ajouter des en-têtes et pieds de page de rapport personnalisés au profil. Les en-têtes et pieds de page doivent être formatés en HTML pour s'afficher correctement. À noter que les en-têtes et pieds de page s'appliquent au rapport entier : les en-têtes et pieds de page sont affichés une fois chacun.

Message par défaut

Vous pouvez ajouter un message personnalisable au rapport. On peut modifier le message au moment d'exécuter le rapport.

Emplacement du message

Vous pouvez préciser si le message est imprimé juste une fois (en haut du rapport) ou chaque fois devant chaque dossier.

Renseignements sur les vues

Sélectionnez les vues qui pourront accéder à ce profil d'impression. Le profil sera accessible dans les vues que vous précisez uniquement et par tout utilisateur connecté ayant accès à cette vue. Vous pouvez avoir autant de profils d'impression par vue que vous le souhaitez.
