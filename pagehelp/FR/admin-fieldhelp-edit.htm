La page Mettre à jour l'aide sur les champs vous permet de modifier l'aide sur les champs expliquant le contenu et les normes de données pour chaque champ. Les liens d'aide sur les champs apparaissent à côté de chaque champ sur les formulaires de saisie de données, y compris les formulaires d'ajout et de mise à jour et sur les formulaires de rétroaction, y compris les formulaires « Proposer un nouveau dossier ». Si de l'aide est disponible, une icône représentant un point d'interrogation apparaîtra à côté du champ sur le formulaire, et s'ouvrira dans une fenêtre contextuelle pour y afficher l'aide : .

Un maximum de 8 000 caractères est autorisé pour chaque entrée d'aide sur les champs. Le HTML doit être utilisé pour afficher la mise en forme, comme les titres, les paragraphes, etc. Si de l'aide existe déjà pour le champ, l'aide existante s'affichera ; sinon, la zone d'entrée du texte sera vide. Pour visualiser l'aide actuelle pour ce champ, cliquez sur le lien « Visualiser l'aide actuelle ». À noter que les changements que vous effectuez ne seront pas affichés quand vous cliquez sur « Visualiser l'aide actuelle », tant que vous n'aurez pas cliqué sur le bouton « Soumettre » en bas du formulaire.

Remarque importante : lorsque vous rédigerez les fichiers d'aide sur les champs, rappelez-vous qu'ils seront visibles aux membres du public qui envoient des rétroactions, ainsi qu'aux membres du personnel connectés. On devrait rédiger l'aide sur les champs de manière à ce qu'elle soit comprise par un membre du public sans expérience de la base de données, tout en reflétant les normes de saisie des données qui sont importantes pour le personnel administratif de la base de données.
