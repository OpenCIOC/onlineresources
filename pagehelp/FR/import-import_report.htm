La page Importer les données des dossiers affiche une liste des dossiers qui ont été ajoutés ou mis à jour avec succès à partir du fichier d'exportation donné.

Le champ Propriétaire du dossier dans ce rapport représente le propriétaire désigné dans le fichier d'exportation, et non pas le propriétaire du dossier actuel dans la base de données.

Si des problèmes sont survenus lors de l'importation des dossiers, ils seront rapportés dans le champ Rapport d'importation. Les types de problèmes rencontrés peuvent comprendre des éléments de listes ou de classifications qui sont utilisés dans la base de données source, mais pas la base de données destinataire ; si une action a été prise pour résoudre des incompatibilités (comme transférer un élément de liste dans le champ de notes générales) ou si les données n'ont pas été importées, cela sera consigné dans le rapport.
