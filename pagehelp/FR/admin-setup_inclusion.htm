La page de Politique d'inclusion des dossiers est l'emplacement où est saisie l'information à afficher sur la page « Proposer un nouveau dossier » de la base de données (feedback.asp). Le texte saisi ici s'ouvre dans une fenêtre contextuelle lorsqu'on clique sur le lien « Politique d'inclusion des dossiers » en haut de feedback.asp. Une fois que vous avez créé des politiques d'inclusion pour votre base de données, vous pourrez sélectionner une politique pour chaque langue utilisée dans une vue donnée depuis la page de configuration des vues. Vous pouvez créer plusieurs politiques ou utiliser une seule politique pour tous les modules (Information communautaire et Bénévolat) et toutes les vues. Les politiques d'inclusion sont spécifiques à chaque langue.

S'il n'y a pas de texte de politique d'inclusion, aucun lien ne sera fourni sur le formulaire. Il est fortement recommandé d'avoir au moins une politique d'inclusion de base disponible si vous prévoyez de donner un lien aux personnes pour suggérer de nouveaux dossiers.
