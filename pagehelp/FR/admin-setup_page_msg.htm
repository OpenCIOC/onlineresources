Un Messages de page vous permet d'insérer certaines informations dans une page, comme des instructions ou des annonces spéciales, lorsque cette information doit être présentée sur certaines pages mais pas d'autres. Vous pouvez également sélectionner les vues dans lesquelles vous souhaitez afficher le message.

Créer un nouveau message

Pour créer un nouveau message de page, sélectionnez >>CRÉER<< dans le menu déroulant, puis cliquez sur le bouton « Voir / Modifier un message » pour être dirigé vers la page « Ajouter un nouveau message ».

Modifier un message existant

Pour consulter ou modifier les paramètres d'un message de page existant, sélectionnez le message dans la liste, puis cliquez sur « Voir / Modifier le message ».
