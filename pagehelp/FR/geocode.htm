La page Mettre à jour le géocodage des dossiers sélectionnés utilise vos choix sur le type de cartographie (Configuration du dossier actuel, Adresse du site, Intersection ou Aucune valeur (ne pas cartographier)), ainsi que les paramètres de la catégorie de cartographie (marqueurs de cartes) que vous avez sélectionnés sur la page précédente pour mettre à jour l'emplacement sur la carte et les paramètres des dossiers sélectionnés.

Certaines choses importantes à savoir :

  • Vous avez l'autorisation de géocoder un maximum de 15 000 emplacements par jour, par adresse IP. Cela comprend n'importe quel emplacement saisi pour la recherche à proximité.
  • Le géocodage se fait un dossier à la fois. Si vous quittez la page avant que le statut du dernier dossier de la page ne soit établi, votre géocodage ne sera pas terminé.
  • Si la colonne de statut indique Mis à jour, alors les informations de géocodage du dossier ont été établies (ou réinitialisées) avec la latitude et la longitude reçues du fournisseur de carte. Si vous avez sélectionné une catégorie ou un type de cartographie, ces nouvelles valeurs seront également mises à jour dans le dossier.
  • Si la colonne de statut indique Inchangé, alors les informations de géocodage restent complètement inchangées dans le dossier (y compris tous les changements du type ou de la catégorie de cartographie demandés). Cela peut survenir pour l'une des raisons suivantes :
    • Vous n'avez pas l'autorisation de mettre à jour les dossiers sélectionnés ; ou
    • Vous avez sélectionné « Configuration du dossier actuel » et le type actuel de cartographie est Aucune valeur (ne pas cartographier) ou Placement manuel ; ou
    • Vous avez sélectionné un type de cartographie Intersection ou Adresse du site, mais ces champs ne contiennent aucune valeur valide.
  • Si la colonne de statut indique que L'adresse est introuvable, alors le fournisseur de carte n'a pas pu localiser l'emplacement spécifié. Cela peut survenir si certaines parties de l'adresse sont erronées (le plus souvent, le type de rue, l'orientation ou le code postal) ; il faut donc réviser les dossiers pour trouver les erreurs possibles. Certains emplacements nouveaux ou ruraux ne peuvent être cartographiés automatiquement ; dans ce cas, utilisez le placement manuel.
  • Si la colonne de statut indique Contactez votre administrateur système, cela peut survenir souvent lorsque vous tentez de cartographier un grand nombre de dossiers les uns après les autres. Essayez de cartographier de plus petits ensembles, moins de 500 dossiers à la fois, et laissez passer quelques minutes entre chaque ensemble.
