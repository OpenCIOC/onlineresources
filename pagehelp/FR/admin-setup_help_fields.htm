La page de sélection Mettre à jour l'aide sur les champs est utilisée pour choisir les champs aux fins de modification de l'aide sur les champs. Les liens d'aide sur les champs apparaissent à côté de chaque champ sur les formulaires de saisie de données, y compris les formulaires d'ajout et de mise à jour et sur les formulaires de rétroaction, y compris les formulaires « Proposer un nouveau dossier ». Si de l'aide est disponible, une icône représentant un point d'interrogation apparaîtra à côté du champ sur le formulaire, et s'ouvrira dans une fenêtre contextuelle pour y afficher l'aide : .

Pour modifier le texte d'aide de n'importe quel champ, sélectionnez le dans la liste, puis cliquez sur le lien « Modifier l'aide sur les champs ». Si vous souhaitez modifier le nom convivial d'un champ, cela s'effectue dans une autre section (par Configuration de la base de données > Affichage des champs).

Le fait de cliquer sur les liens « Version imprimable » ouvrira un rapport en version imprimable de toutes les aides sur les champs dans une nouvelle fenêtre (l'aide peut être imprimée dans n'importe quelle langue utilisée dans la base de données). Lorsqu'il n'y a pas d'aide existante pour un champ donné, la colonne d'aide dans le document imprimé sera vide.

Remarques importantes :

  • Des échantillons d'aide sur les champs peuvent être fournis lors de l'installation du logiciel, mais vous devriez réviser cette aide pour s'assurer qu'elle répond à vos besoins locaux. Si vous modifiez vos fichiers d'aide, vous devriez conserver une sauvegarde de vos changements et avertir l'administrateur du système que vous avez effectué des changements, au cas où une mise à jour de l'aide sur les champs distribuée est annoncée.
  • Vous pouvez créer des aides sur les champs pour des champs personnalisés comme les descriptions de Publications et les en-têtes. Une fois que vous rendez ces champs disponibles sur des formulaires par la configuration des Publications, elles apparaîtront dans cette liste pour que vous puissiez y ajouter des textes d'aide.
  • Il s'agit de plusieurs champs qui sont automatiquement inclus / exclus en haut du formulaire de mise à jour, y compris le champ Date de mise à jour et le champ Numéro de dossier ; comme le lien vers l'aide sur les champs pour ces champs ajoutés automatiquement est toujours imprimé, il est donc recommandé de s'assurer que de l'aide est disponible pour tous ces champs.
