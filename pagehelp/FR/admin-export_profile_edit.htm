Cette page vous permet de modifier les détails d'un profil d'exportation, y compris les éléments de données à inclure et dans quelles vues on peut y accéder.

Configuration générale du profil

  • Langue - préciser les langues des dossiers à exporter (présentement, seuls anglais et français peuvent être sélectionnés pour l'exportation).
  • Nom - le nom unique du profil. Utilisé pour sélectionner le profil au cours de la procédure d'exportation.
  • Renseignements sur la base de données source - préciser un nom pour votre base de données, un lien vers votre base de données ou votre site web, et la vue accessible publiquement dans laquelle vous souhaitez qu'un destinataire envoie les demandes de mises à jour aux dossiers (cela créera un lien vers le formulaire de rétroaction dans cette vue).
  • Profils de confidentialité - si vous utilisez des profils de confidentialité dans votre base de données, précisez si le destinataire est autorisé à accéder aux renseignements privés. En cas de doute, n'incluez pas de renseignements privés !
  • Dans les vues - pour rendre ce profil accessible aux fins d'utilisation, vous devez l'activer dans une ou plusieurs vues. Pour des raisons de sécurité, il est recommandé de n'activer chaque profil que dans les vues où il est nécessaire.

« Inclure les données » : gérer les éléments de données pour le profil

Les éléments de données pouvant être inclus dans le profil sont modifiés par une succession de fenêtres contextuelles reliées au formulaire. Les changements apportés dans les fenêtres contextuelles sont traités indépendamment des autres renseignements du formulaire. Cette section vous permet de sélectionner les champs, codes de distribution et codes de publication à inclure, mais ne précise pas quels dossiers sont inclus. Les dossiers doivent être sélectionnés au moment de l'exportation et seront établis en partie en fonction de la vue dans laquelle travaille l'utilisateur.

  • Gérer les champs - sélectionnez les champs de la base de données à inclure.
  • Gérer les distributions - sélectionnez les codes de distribution à inclure dans le champ Distribution.
  • Gérer les publications - sélectionnez les codes de publication à inclure dans le champ Publication. Vous pourrez également choisir d'inclure les renseignements relatifs aux Descriptions des publications et En-tête généraux pour chaque code sélectionné.

Remarque importante : le fait de sélectionner des codes de publication ou de distribution particuliers pour vos profils n'affecte pas le choix des dossiers à exporter, seulement les contenus des champs Publication ou Distribution que vous envoyez.
