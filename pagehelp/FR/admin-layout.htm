Les mises en page de modèles sont utilisées au sein de modèles de conception pour contrôler la mise en page des sections personnalisables d'une page, y compris les en-têtes, les pieds de page et les formulaires de recherche et contenus de la page principale.

Les mises en page qui sont contrôlées exclusivement par une agence particulière contiennent le code d'agence entre crochets à la fin du nom (c'.-à-d. [ABC]). Vous pourrez sélectionner dans la liste des mises en page que vous ne possédez pas, afin de les consulter, mais ne serez pas capable de les modifier.

Vous disposerez d'une mise en page intégrée au système ou plus dans votre base de données, pour chaque type (en-tête, pied de page, recherche). Vous pouvez consulter ou copier ces mises en page intégrées et les utiliser dans vos vues si désiré, mais elles ne peuvent être modifiées en aucune manière. Les mises en page du système sont indiquées par un *.

Modifier une mise en page de modèle existante

Sélectionnez dans le menu déroulant le nom de la mise en page que vous souhaitez modifier, puis cliquez sur le bouton « Voir / Modifier la mise en page de modèle ».

Ajouter une nouvelle mise en page de modèle

Pour créer une nouvelle mise en page, cliquez sur le bouton « Ajouter une mise en page de modèle » pour être dirigé vers le formulaire d'ajout d'une nouvelle mise en page. Vous pouvez copier une mise en page existante en la sélectionnant dans la liste déroulante. Si vous ne sélectionnez pas de mise en page à copier, votre nouvelle mise en page sera blanche (vide).
