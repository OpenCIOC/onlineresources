Nouveau Rappel

Le système de rappels vise à aider les utilisateurs à créer des listes de tâches pour eux-mêmes ou d'autres utilisateurs, ainsi qu'à offrir un moyen de communication avec d'autres utilisateurs de CIOC. Les rappels peuvent porter sur des tâches générales ou ils peuvent être associés à un ou plusieurs dossiers. Vous pouvez travailler avec les rappels en cliquant sur le lien Rappels dans la barre supérieure ou (dans le cas de rappels associés à des dossiers précis) par la page de détails du dossier. Vous pouvez également créer un rappel pour un groupe de dossiers par vos résultats de recherche. Cliquez sur le bouton « Prochain » pour débuter la création d'un nouveau rappel sur les dossiers sélectionnés.

Imprimer la liste

Si au moins un profil d'impression est accessible dans votre vue actuelle, vous pourrez choisir un profil spécifique dans la liste déroulante de cette page et commencer la préparation d'un rapport à imprimer personnalisé. On peut ajouter ou modifier des profils d'impression par Configuration > Profils d'impression.

Imprimer la carte

Si vous avez géocodé (cartographié) les dossiers de votre base de données, vous pouvez générer une carte numérotée à imprimer montrant la carte et la liste de dossier. Vous pouvez cartographier jusqu'à 99 dossiers maximum à la fois (s'il y a moins de dossiers, les résultats sont plus lisibles et exploitables).

Supprimer / Restaurer les dossiers

Cette fonction vous permet d'établir ou de supprimer la date de suppression d'un groupe de dossiers. Les options disponibles sont « Établir la date de suppression » (à choisir pour supprimer un groupe de dossiers ou modifier la Date de suppression de dossiers déjà supprimés) ou « Restaurer les dossiers » (à choisir pour retirer le statut de suppression d'un groupe de dossiers).

Changer le propriétaire du dossier

Cette fonction vous permet de modifier l'agence désignée comme propriétaire du dossier des dossiers sélectionnés. Sélectionnez l'agence désirée dans la liste déroulante, puis cliquez sur « Prochain ». La liste des agences disponibles ici est établie en fonction des agences marquées comme détenant des dossiers dans le module actuel (Information communautaire ou Bénévolat), dans la section Agences de la configuration.

Ajouter / Supprimer un code de distribution

Cela vous permet soit d'ajouter, soit de supprimer un code distribution d'un ensemble de dossiers sélectionnés. Vous devez sélectionner soit « Ajouter », soit « Supprimer » dans le premier menu déroulant, puis sélectionner parmi les codes de distribution disponibles dans le second menu déroulant. Un super-utilisateur peut ajouter de nouveaux codes à la liste en allant sur le lien Listes de contrôle > Distribution dans le menu principal de la configuration. Si vous choisissez l'option « Supprimer toutes les valeurs », vous devrez cocher une case pour confirmer que vous souhaitez bien retirer chaque code de distribution de tous les dossiers sélectionnés.

[ Haut ]

Ajouter / Supprimer un en-tête

Cette fonction vous permet de retirer un en-tête général attaché/associé à un code de publication assigné aux dossiers. La procédure est en deux étapes. Vous devez d'abord sélectionner le code de publication associé à l'en-tête général que vous souhaitez ajouter ou supprimer, puis cliquer sur « Soumettre » (remarque : seules les publications associées à des en-têtes seront listées). Puis sélectionnez soit « Ajouter », soit « Supprimer » dans le premier menu déroulant, suivi de l'en-tête désiré dans le deuxième menu déroulant, et enfin cliquez sur « Soumettre ». Si vous choisissez l'option « Supprimer toutes les valeurs », vous devrez cocher une case pour confirmer que vous souhaitez bien retirer tous les en-têtes de la publication actuelle de tous les dossiers sélectionnés.

En plus de l'outil général permettant d'ajouter ou de retirer des en-têtes, vous pouvez également synchroniser votre indexation de la publication actuelle à l'indexation effectuée à une autre publication. Cela s'applique uniquement dans les cas où le même en-tête existe pour les deux publications. Il est important de choisir une publication « principale », dans laquelle vous maintenez l'indexation des dossiers et la copiez sur les autres publications. Vous pouvez sélectionner la publication principale dans le menu déroulant « Synchroniser les en-têtes des classifications trouvées dans ».

Ajouter / Supprimer un code SCIAN

Cette fonction vous permet d'ajouter ou de supprimer un code du Système de Classification des Industries de l'Amérique du Nord (SCIAN) valide d'un groupe de dossiers. Sélectionnez soit « Ajouter », soit « Supprimer » dans le menu déroulant, puis saisissez un code SCIAN de 2 à 6 chiffres valide à ajouter/supprimer. Si vous ne connaissez pas le code recherché, utilisez le lien « Localisateur de codes SCIAN » pour ouvrir une fenêtre contextuelle offrant un formulaire pour rechercher et parcourir les codes SCIAN. Si vous choisissez l'option « Supprimer toutes les valeurs », vous devrez cocher une case pour confirmer que vous souhaitez bien retirer chaque code SCIAN de tous les dossiers sélectionnés.

Ajouter / Supprimer un code de publication

Cela vous permet soit d'ajouter, soit de supprimer un code de publication d'un ensemble de dossiers sélectionnés. Vous devez sélectionner soit « Ajouter », soit « Supprimer » dans le premier menu déroulant, puis sélectionner parmi les codes de publication disponibles dans le second menu déroulant. Faites attention à l'avertissement sur la page car le fait de supprimer un code de publication de dossiers a d'autres conséquences potentielles en plus de simplement retirer le codes des dossiers. Un super-utilisateur avec des privilèges des gestion peut ajouter de nouveaux codes à la liste en allant sur Publication depuis dans le menu principal. Si vous choisissez l'option « Supprimer toutes les valeurs », vous devrez cocher une case pour confirmer que vous souhaitez bien retirer chaque code de publication de tous les dossiers sélectionnés.

Ajouter / Supprimer un terme de la Taxonomie

Cela vous permet soit d'ajouter, soit de supprimer un seul terme de la Taxonomie (catégorie de service) d'un ensemble de dossiers sélectionnés. Vous devez sélectionner soit « Ajouter », soit « Supprimer » dans le premier menu déroulant, puis saisir le le code taxonomique valide d'un terme actif dans le cadre fourni. Si vous ne connaissez pas le code du terme à ajouter/supprimer, utilisez le lien de l'outil Chercher une catégorie de service fourni.

Si vous supprimez un terme existant des dossiers, certains choix supplémentaires peuvent s'appliquer. Comme certains termes taxonomiques peuvent apparaître dans une liste de termes liés, vous devez choisir ce qu'il faut faire lorsque le terme fait partie d'un ensemble de termes liés. Par défaut, le terme sera seulement supprimé s'il n'est pas lié à d'autres termes.

Si vous choisissez l'option « Supprimer toutes les valeurs », vous devrez cocher une case pour confirmer que vous souhaitez bien retirer tous les termes taxonomiques de tous les dossiers sélectionnés.

Mettre à jour le géocodage

Il n'est pas nécessaire d'utiliser cet outil régulièrement pour les dossiers qui ont déjà été géocodés. Si vous avez inclus le champ GEOCODE sur le formulaire de mise à jour, la cartographie sera automatiquement mise à jour lorsque vous modifierez le dossier.

Cette fonction vous permet de mettre à jour l'emplacement sur la carte sur une liste de dossiers. Par défaut, cet outil utilisera les paramètres de cartographie existants et confirmera que l'emplacement sur la carte est correct en fonction des renseignements d'adresse ou d'intersection actuels (le cas échéant). De manière optionnelle, vous pouvez modifier ou établir le type de cartographie (à notes sur l'adresse, intersection ou ne pas cartographie) et/ou établir la catégorie de cartographie des dossiers mis à jour. Pour voir le nom de la catégorie de cartographie (si disponible), passez la souris sur le bouton de carte (passez la souris sur le bouton pour afficher « Catégorie de cartographie » ).
