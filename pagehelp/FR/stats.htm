Deux types d'outils statistiques sont fournis dans votre base de données CIOC :

Statistiques sur les dossiers consultés

Ces statistiques sont générées dans votre base de données lorsque quelqu'un accède à un dossier de la base de données. Sont enregistrés : la date et l'heure, l'adresse IP, la vue, l'identifiant d'utilisateur, le numéro de dossier et la langue. Les statistiques sur les dossiers consultés ne sont pas conservées lorsque vous passez en mode impression ou changez de vue temporairement, pour ne pas gonfler les statistiques si quelqu'un consulte plusieurs versions d'un même dossier pour les mêmes raisons. Ces statistiques s'avèrent le plus utile pour :

  • comprendre quels dossiers de votre base de données sont les plus populaires
  • comprendre de quelle manière chacune des vue est utilisée
  • comprendre les différences de consultation pour chaque langue des dossiers
  • identifier la manière dont un utilisateur ou groupe d'utilisateurs particulier utilise la base de données

Les super-utilisateurs peuvent accéder aux statistiques sur les dossiers consultés sur une base de données Microsoft Access hors ligne accessible par le menu de téléchargement, aux fins d'analyses plus poussées si besoin.

Statistiques sur les connexions web

Ces statistiques représentent l'utilisation totale de votre base de données et incluent tous les types de fichiers (y compris les images, CSS, JavaScript et autres fichiers). Ces statistiques s'avèrent très utiles pour comprendre l'utilisation globale de votre base de données et la croissance du trafic au fil du temps. Les statistiques sur les connexions web sont conservées dans un emplacement distinct de votre base de données ; le contact technique officiel de votre membriété peut communiquer avec le personnel de CIOC afin d'obtenir un compte pour cet outil. Contre des frais minimes, des zones d'outils statistiques web distinctes associées à votre compte peuvent être configurées pour les noms de domaines spécifiques utilisés par votre base de données. Communiquez avec le personnel technique de CIOC pour des renseignements.

Cette page contient les statistiques principales sur les dossiers consultés et un formulaire de rapport, ainsi que des liens vers d'autres outils statistiques : Utilisation totale des dossiers, 50 premiers dossiers, Rapports automatiques et Supprimer les statistiques anciennes.

L'adresse IP de l'utilisateur commence par

Si vous connaissez l'IP d'un utilisateur spécifique (ou le début d'un bloc d'adresses IP) et tentez de déterminer l'utilisation de la base de données par cette IP, saisissez la dans ce champ. Par exemple, si vous savez que les adresses IP de votre CASC local commencent toutes par 199.235.235.XXX, où XXX représente plusieurs numéros d'identification différents pour des ordinateurs de ce CASC, saisissez 199.235.235 dans le champ ; les résultats doivent refléter précisément l'utilisation de la base de données par le CASC local.

Utilisateurs avec type d'utilisateur

Vous pouvez limiter les statistiques uniquement aux dossiers consultés par des utilisateurs dont le compte est associé à un type d'utilisateur particulier.

Recherche par vue

Vous pouvez limiter les statistiques uniquement aux dossiers consultés dans une vue particulière (ce qui ne prend pas en compte si l'utilisateur est connecté ou le type d'utilisateur).

Recherche par date

le ou après le - saisissez la date à laquelle vous souhaitez faire débuter le rapport statistique. Cette date sera comprise dans les résultats.

avant le - saisissez la date suivant le jour auquel vous souhaitez faire cesser le rapport statistique. Cette date ne sera pas comprise dans les résultats. Pour générer un rapport correspondant exactement au mois précédent, cliquez sur les boutons « Premier du dernier mois » et « Premier de ce mois ».

Dossiers appartenant à

S'il existe plus d'un propriétaire de dossiers dans la base de données, vous pouvez demander ici de retourner des résultats en fonction de ce critère.

Dossiers consultés par

Retourne des résultats en fonction :

  • de n'importe quel utilisateur de la base de données, ou
  • des utilisateurs connectés uniquement, ou
  • des utilisateurs du public (non connectés) uniquement.

Dossiers consultés en

Limite les statistiques aux dossiers consultés dans une langue particulière.

Vues des dossiers sur les moteurs de recherche

Inclue ou exclue le trafic des moteurs de recherche connus ou affiche le trafic par moteur de recherche spécifique. Soyez conscient que cet outil ne peut qu'estimer le trafic des moteurs de recherche en fonction d'adresses connues ; il ne prend pas en compte tous les robots de votre base de données.

Version imprimable

Si vous souhaitez publier le rapport en mode impression, sélectionnez Oui pour cette option. Les rapports exécutés en version imprimable s'ouvrent dans une nouvelle fenêtre.
