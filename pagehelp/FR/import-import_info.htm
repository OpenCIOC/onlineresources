La page Renseignements supplémentaires fournit un rapport sur le fichier d'exportation sélectionné. Il est recommandé de réviser attentivement les rapports de Renseignements supplémentaires avant de lancer une importation.

Le rapport indique le nom de fichier, le nom convivial, le nombre de dossiers à ajouter ou mettre à jour, les langues dans lesquelles les dossiers sont disponibles, les codes de publication et de distribution inclus et les champs à mettre à jour. À noter que seuls les codes de publication et de distribution qui existent dans votre base de données au moment du chargement du fichier exporté sont inclus ; vous ne pouvez importer aucun de ces types de codes sans les avoir ajouter auparavant à votre base de données par l'entremise des paramètres de publication ou de distribution.

Il est important de s'assurer que le fichier comprend les données attendues, de manière à ne pas écraser accidentellement les champs ou codes lorsque vous souhaitez conserver votre propres données. Vous aurez l'occasion de voir les données des dossiers individuels lors de la procédure Ajouter / Modifier.
