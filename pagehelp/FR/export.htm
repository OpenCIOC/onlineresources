Depuis cette page, vous pouvez exporter des dossiers sélectionnés dans la base de données en trois formats, en fonction de vos permissions :

  • Recordset Microsoft sauvegardé (XML) - utilise des données Microsoft Recordset au format XML ; un outil est offert aux membres CIOC pour convertir ce format en tableau Microsoft Access (référence disponible sur le site de la communauté CIOC).
  • Format de partage CIOC (XML) - fonctionne avec l'Outil d'importation CIOC. Remarque : pour partager des dossiers avec d'autres utilisateurs de CIOC, leur version du logiciel CIOC doit supporter une version compatible du Format de partage CIOC. Si le partenaire avec qui vous travaillez utilise une version de CIOC plus récente ou plus ancienne, ou un autre logiciel, l'exportation peut être incompatible.
  • Tableau Excel - formats compatibles avec les différentes versions de Microsoft Excel et autres programmes de tableur.

Sélectionnez le format dans lequel vous voulez exporter. En fonction de vos permissions, vous pourriez avoir le choix de restreindre votre exportation en fonction d'un ensemble de codes (p. ex., Publication, Distribution ou Catégorie). Dans l'une des listes de codes, sélectionnez un code ou plus à inclure dans l'exportation, puis cliquez sur le bouton « Suivant ». Vous pouvez choisir plusieurs codes en gardant le bouton « CTRL » enfoncé pendant que vous sélectionnez dans la liste.

Profils d'exportation / Profils Excel

Les profils d'exportation et Excel vous permettent de préciser les champs spécifiques à exporter. Vous devez préciser un profil à utiliser parmi ceux qui vous sont accessibles. À noter que les profils sont rendus accessibles en fonction de la vue ; la liste des vues pour chaque profil peut être modifiée depuis la page de configuration d'un profil.

Générer le fichier d'exportation

Une fois vos sélections effectuées, cliquez sur le bouton « Suivant » pour démarrer le processus d'exportation. Le fichier sera disponible sur le serveur pour une durée limitée uniquement, donc vous devez télécharger le fichier rapidement. Seule la personne qui génère le fichier peut le télécharger.
