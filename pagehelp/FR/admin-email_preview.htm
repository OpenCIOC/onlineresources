La page Prévisualisation du courriel vous permet de prévisualiser le message du courriel que vous êtes sur le point d'envoyer à un ou plusieurs organismes ou personnes contacts.

Si vous envoyez une demande de mise à jour pour plus d'un dossier, votre prévisualisation sera basée sur le premier dossier dans la liste. Si le dossier prévisualisé contient des caractéristiques différentes des autres messages à envoyer (par exemple, un dossier bilingue et un dossier unilingue français, ou un dossier interne et un dossier public), il est possible que le contenu d'un message varie légèrement avec les autres dossiers dans la liste.

Si vous êtes satisfait avec l'objet du message et la section modifiable du corps du message, cliquez sur le bouton « Envoyer le courriel » en bas de la page pour envoyer les courriels.

Si vous souhaitez modifier le courriel après l'avoir prévisualisé, utilisez le bouton retour de votre navigateur pour revenir à la page précédente où vous pourrez modifier les détails du courriel.
