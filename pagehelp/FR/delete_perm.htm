Cette page demande votre confirmation avant de supprimer définitivement les dossiers de la base de données. Les dossiers marqués d'une coche verte sont prêts à être supprimés.

Les dossiers supprimés définitivement sont retirés de la base de données de manière irrévocable et ne seront plus visibles par personne. Le fait de retirer des dossiers peut affecter les rapports statistiques et d'autres renseignements ; il donc est généralement déconseillé de supprimer les dossiers de manière permanente. Si les dossiers doivent être supprimés, le meilleur moment est après avoir purgé les statistiques, comme il est coutume de le faire à la fin de l'année. Si le dossier a été supprimé depuis suffisamment longtemps, il aura peu, voir pas de statistiques actuelles, et pourra être retiré en toute sécurité, sans causer d'anomalies dans les rapports. Une fois qu'un dossier a été supprimé définitivement, son numéro de dossier d'agence peut être réutilisé.

Si vous supprimez un seul dossier, vous obtiendrez un lien vers la page « Détails sur le dossier d'organisme » pour réviser le contenu du dossier avant de le supprimer. Si vous prévoyez de supprimer un ensemble de dossiers en une seule fois, il est fortement conseillé d'exporter d'abord ces dossiers en utilisant un profil de Format de partage CIOC complet, qui vous donnera la chance de restaurer ces dossiers en cas d'erreur.
