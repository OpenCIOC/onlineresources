On peut considérer les Vues comme des portails individuels dans la base de données. Les vue contrôlent quels dossiers sont accessibles, quels champs peuvent être consultés ou modifiés, quels types de recherches sont disponibles sur les différents formulaires de recherche, quels systèmes de classification utiliser, les langues dans lesquelles l'utilisateur peut travailler, le contenu du menu personnalisé et des pieds de page, les modèles de conception à utiliser, etc. Les vues peuvent être configurées de manière être accessibles au public ou uniquement à des utilisateurs connectés spécifiques.

La page Gestion des vues vous permet de créer ou de modifier plusieurs vues, qui peuvent alors être rendues publiques ou assignées à des types d'utilisateurs. Par exemple, vous pourriez décider de dédier une vue de la base de données spécifiquement à un centre d'accès aux soins communautaires local. Le fait d'utiliser des vues combinées à des types d'utilisateurs et des modèles de conception vous permettrait de créer une vue très personnalisée de la base de données, spécifiquement aux fins d'utilisation par le personnel du CASC.

Modifier une vue existante

Pour consulter ou modifier les paramètres d'une vue existante, sélectionnez le profil dans la liste, puis cliquez sur « Voir / Modifier la vue ». La liste des vues fait la distinction entre les vues accessibles aux utilisateurs sans compte (public) ou seulement aux utilisateurs connectés. Il est recommandé de réviser cette liste périodiquement pour s'assurer qu'aucune des vues que l'on souhaite privée ne soit publique.

Les vues qui sont contrôlées exclusivement par une agence particulière ont le code de cette agence à la fin du nom entre crochets (c'.-à-d. [ABC]). Vous pourrez sélectionner dans la liste des vues que vous ne possédez pas, afin de les consulter, mais ne serez pas capable de les modifier.

Créer une nouvelle vue

Pour créer une nouvelle vue, vous devez donner un nom unique à la vue. Vous pouvez copier une vue existante en la sélectionnant dans la liste. Si vous ne sélectionnez pas de vue à copier, votre nouvelle vue contiendra les paramètres de la vue publique par défaut. Cliquez sur le bouton « Ajouter une vue » pour créer la nouvelle vue ; vous serez dirigé vers le formulaire pour la consulter / modifier.
