Ces instructions sont partagées par plusieurs types de recherches, y compris la recherche (simple) de la page principale, la recherche avancée, la recherche par étapes, les nouveaux ensembles de résultats (en utilisant les cases à cocher pour créer une nouvelle liste de dossiers), les résultats de recherche précédente et les différentes recherches « Parcourir ». Toutes les fonctionnalités n'existent pas pour toutes les recherches ; consultez les notes dans chaque section le cas échéant.

La page de résultats de recherche affiche les dossiers qui correspondent aux critères que vous saisissez dans la recherche simple ou la recherche avancée. Si le message qui apparaît sur cette page est : « Il n'y a pas de dossier correspondant à vos critères. », alors il n'y a pas de dossier dans votre base de données qui correspondent aux critères que vous avez sélectionnés. Dans ce cas, utilisez le bouton retour ou retournez au menu principal / page de démarrage ou à la page de recherche avancée pour débuter une nouvelle recherche moins restrictive.

En haut de la page, une légende indique les noms de groupes communautaire spécifiques correspondant aux boutons de couleur. Les boutons de couleur sont imprimés à côté des occasions, indiquant dans quels domaines généraux le poste est offert. Un super-utilisateur peut modifier les paramètres de regroupement des communautés, les noms des regroupements et le bouton de couleur associé à un groupe depuis la configuration des Communautés de bénévolat.

Chaque dossier dans vos résultats de recherche comporte un lien pour consulter tous les détails du dossier. Ce lien est soit le Titre du poste, soit l'identifiant d'occasion. Les options d'affichage déterminent si le nom ou l'identifiant est hyperlié, comme mentionné plus bas.

À propos de votre recherche

La plupart des recherches comprennent une description de votre recherche en haut de la page. Depuis la recherche avancée, les détails de la recherche ne sont pas inclus par défaut, mais vous pouvez cocher une case en bas du formulaire de recherche pour afficher les détails de votre recherche sur la page de résultats. Lorsque les détails de votre recherche sont disponibles, cette information peut être utilisée pour décrire votre recherche si vous souhaitez préciser votre recherche, lorsque que vous consultez la liste des vos recherches précédentes et lorsque vous sauvegardez votre recherche.

Préciser votre recherche

Le fait de cliquer sur le lien  Préciser la recherche au-dessus de vos résultats de recherche redirigera vos critères de recherche actuels vers le formulaire de recherche avancée, afin que vous puissiez ajouter des critères de recherche supplémentaires aux présents résultats.

Options d'affichage de la liste de dossiers

Tous les utilisateurs ayant un compte dans la base de données ont le choix de modifier la manière dont les résultats sont affichés. Les options exactes qui vous sont accessibles pour l'affichage sont établies en fonction de vos autorisations dans la base de données. Vous pouvez modifier vos options d'affichage temporairement pour votre session ou sauvegarder vos changements comme le choix par défaut à utiliser chaque fois que vous vous connectez à la base de données. Pour modifier vos paramètres, cliquez sur le lien Changer les options d'affichage ; une fenêtre contextuelle apparaîtra, vous permettant de modifier vos paramètres actuels. À noter que vos changements ne seront mis en œuvre que si vous lancez une nouvelle recherche ou rafraîchissez votre page de résultats actuelle.

Remarque très importante : l'option Utiliser le format tableau (traitée dans la section Afficher les options plus bas) est requise pour accéder à de nombreuses options. La vue ordinaire (où le format tableau est désactivé) est disponible uniquement pour les besoins de recherche simple.

Afficher les champs

Afficher les champs vous permet de décider quelles informations de vos dossiers seront affichées dans vos résultats. Vous pouvez sélectionner depuis un ensemble de champs prédéterminés ou choisir un nombre illimité de champs « personnalisés ». La liste des champs personnalisés qui vous est proposée est établie en fonction des renseignements que vous êtes autorisé à consulter sur la page de détails d'un dossier.

La colonne d'alerte est une boîte spéciale qui apparaît à l'extrême gauche de vos résultats de recherche. Vos autorisations établissent si vous pouvez voir la colonne d'alerte et quelles alertes vous êtes autorisé à voir. Si vous n'utilisez pas le format tableau pour l'affichage, la colonne d'alerte n'est pas disponible ; à la place, certaines alertes seront imprimées à côté du lien sur les détails de l'occasion. La liste suivante est une liste des alertes disponibles :

  • Boîte couleur d'alerte / N - Lorsque la colonne d'alerte est colorée, le dossier est interne (la lettre N -- Non-public en anglais -- est utilisée s'il n'y pas de colonne d'alerte).
  • C - Le dossier contient un Commentaire ou une note interne.
  • E - Le dossier est Expiré (affiché jusqu'à expiration de la date).
  • F - Le dossier contient une rétroaction (F pour Feedback en anglais).
  • P - Le dossier est en attente de suppression (P pour Pending en anglais).
  • S - Le dossier est partagé avec vous par un autre membre CIOC dans votre base de données (S pour Shared en anglais).
  • X - Le dossier a été supprimé.

Afficher les options

Afficher les options vous fournit des liens directs pour effectuer plusieurs types d'opérations sur votre liste de dossiers. Les options qui vous sont accessibles dépendent de vos autorisations.

L'affichage par défaut (pas au format tableau) imprime le titre du poste, les boutons de couleur indiquant les communautés où les postes sont disponibles, le champ des fonctions et le nom de l'organisme offrant le poste. L'option Utiliser le format tableau permet à l'utilisateur de modifier l'affichage en forme de tableau avec en-têtes de colonne, où chaque dossier est affiché sur une rangée du tableau. Cela est similaire à l'affichage des résultats dans le module Information communautaire du logiciel et de nombreuses autres options d'affichage sont possibles, y compris les options Afficher les champs décrites dans la section suivante.

L'option Afficher les cases à cocher est utilisée pour effectuer des opérations sur plusieurs dossiers à la fois. Une case à cocher apparaîtra à côté de chaque dossier dans vos résultats, vous permettant de sélectionner les dossiers sur lesquels vous souhaitez effectuer des opérations. Certaines options (en fonction de vos autorisations) comprennent :

  • Créer un nouvel ensemble de résultats de recherche à partir des dossiers sélectionnés
  • Créer un nouveau rappel à partir des dossiers sélectionnés
  • Imprimer une liste de dossiers
  • Créer un rapport statistique des utilisations
  • Demander des mises à jour par courriel en nombre
  • Ajouter / Modifier des « domaines d'intérêt »
  • Établir le statut public des dossiers sélectionnés
  • Supprimer / Restaurer des dossiers
  • Modifier le propriétaire des dossiers sélectionnés
  • Ajouter les dossiers sélectionnés à un profil de partage pour les rendre accessibles à un autre membre CIOC dans votre base de données

Ordre de tri

Cette section vous permet de modifier l'ordre dans lequel les dossiers sont listés. Vous pouvez sélectionner l'un des choix de tri préétablis ou un choix personnalisé dans la liste déroulante.

Version imprimable

Si vous souhaitez imprimer un rapport de vos résultats de recherche, cliquez sur le lien Version imprimable (à côté de Changer les options d'affichage) pour consulter un rapport de vos résultats formatés pour l'impression, sans le menu ni les autres liens. Le super-utilisateur de la base de données peut personnaliser la version imprimable et l'optimiser pour l'impression en créant un modèle de conception spécial pour l'impression et en l'ajoutant à la vue actuelle.

Sauvegarder cette recherche

Pour sauvegarder vos critères de recherche dans votre liste de recherches sauvegardées, cliquez sur le lien « Sauvegarder cette recherche ». Conseil utile : si les détails de votre recherche sont précisés en haut de la page, ils seront transférés dans le champ de notes de votre recherche.

Cette fonctionnalité n'est pas disponible pour les recherches « Parcourir » ou lorsqu'on travaille sur un « nouvel ensemble de résultats » ou consulte des « résultats de la recherche précédente ».
