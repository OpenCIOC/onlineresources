Les pages de gestion des versions du texte normalisé de mise à jour par courriel vous permettent de personnaliser des sections du courriel de demande de mise à jour. Vous pouvez créer plusieurs textes prédéfinis différents pour chaque type de message (mise à jour d'organisme / programme dans Information communautaire, mise à jour d'occasion de bénévolat, mise à jour d'organisme de bénévolat dans « Toutes les occasions »). Un message doit être désigné comme message par défaut pour chaque type ; le message par défaut sera sélectionné en premier quand vous créerez une nouvelle demande de mise à jour par courriel. S'il existe plusieurs versions du texte de mise à jour par courriel, vous pourrez choisir entre elles au moment de l'envoi.

Modifier des valeurs au moment de l'envoi

On peut modifier les valeurs de l'« Objet » et du « Corps du message » au moment d'envoyer les messages ; si vous modifier ces valeurs à ce moment, cela changera les valeurs par défaut. Tous les autres éléments de configuration ne peuvent être modifiés que depuis cet écran de configuration.

Liens vers des pages de dossier

En fonction du type de message, vous aurez soit un « Lien vers les renseignements » et un « Lien vers la rétroaction » ou un « Lien vers toutes les occasions » et un « Lien pour proposer une occasion de bénévolat ». Si vous préférez, vous pouvez choisir d'inclure uniquement le « Lien vers les détails » ou le « Lien vers la rétroaction » dans votre message. Pour retirer le lien que vous ne souhaitez pas utiliser, ne remplissez pas la description du lien ; cependant, vous devez remplir au moins l'une des deux descriptions. Remarque : si un dossier n'est pas visible dans la vue depuis laquelle vous envoyez la demande (cela peut être soit la vue par défaut, soit une autre vue accessible au public que vous avez sélectionnée), le système ne génère automatiquement que le Lien vers la rétroaction utilisant une description préétablie.

Modifier le message par défaut

Pour changer la sélection du texte par défaut dans la base de donnée, vous devez accéder au texte que vous souhaitez par défaut et cliquer sur la case «».
