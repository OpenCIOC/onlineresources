Les pages de gestion des versions du texte normalisé de mise à jour par courriel vous permettent de personnaliser des sections du courriel de demande de mise à jour. Vous pouvez créer plusieurs textes prédéfinis différents pour chaque type de message (mise à jour d'organisme / programme dans Information communautaire, mise à jour d'occasion de bénévolat, mise à jour d'organisme de bénévolat dans « Toutes les occasions »). Un message doit être désigné comme message par défaut pour chaque type ; le message par défaut sera sélectionné en premier quand vous créerez une nouvelle demande de mise à jour par courriel. Si il existe plusieurs valeurs de texte du courriel de mise à jour, vous pourrez choisir entre eux au moment de l'envoi.

Ajouter un nouveau texte de courriel

Pour créer un nouveau texte de courriel pour le type de message actuel (mise à jour d'organisme / programme dans Information communautaire, mise à jour d'occasion de bénévolat, mise à jour d'organisme de bénévolat pour « Toutes les occasions »), cliquez sur le bouton « Ajouter un texte de courriel normalisé ». De manière optionnelle, vous pouvez copier un texte existant.

Modifier un texte de courriel existant

Pour modifier un texte de courriel existant, sélectionnez-le dans le menu déroulant, puis cliquez sur « Voir / Modifier le texte du courriel normalisé ». À noter que chaque type de message (mise à jour d'organisme / programme dans Information communautaire, mise à jour d'occasion de bénévolat, mise à jour d'organisme de bénévolat dans « Toutes les occasions ») est géré séparément et est disponible depuis des liens différents sur la page de configuration générale du logiciel.

Modifier la valeur par défaut

Pour changer le texte par défaut dans la base de données, vous devez aller sur la page du texte que vous souhaitez par défaut et cocher la case appropriée.
