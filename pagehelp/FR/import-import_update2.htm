La page Lancer l'importation permet d'effectuer les fonctions Ajouter / Mettre à jour sur un dossier. Ce processus peut être long, particulièrement pour les dossiers bilingues. L'heure de début s'affiche dès que l'importation commence et un « . » apparaît pour chaque dossier importé avec succès ; une fois l'importation terminée, l'heure de fin s'affiche. Une fois ce processus terminé, assurez-vous de lire le rapport d'importation disponible sur le menu de l'importation principal. À noter qu'avec certains navigateurs, la barre de progrès ... n'apparaît pas avant la fin de l'importation. Cela ne signifie pas que l'importation n'est pas réussie.

Remarque importante : si vous importez un grand nombre de dossiers et que l'importation échoue car plus longue que prévue par le logiciel, vous pouvez relancer l'importation autant de fois que nécessaire pour terminer le processus. Les dossiers qui ont été importés avec succès seront transférés dans la section Terminé après chaque lancement de l'outil, et les importations non terminées resteront dans les sections Ajouter ou Mettre à jour.
