La page Gérer les codes de distribution pour l'exportation vous permet de sélectionner quels codes de distribution à inclure dans le champs Distribution des dossiers à exporter. Cochez ou décochez les cases correspondantes aux codes de distribution que vous souhaitez inclure ou exclure de ce profil d'exportation, puis cliquez sur le bouton « Soumettre » en bas de page pour sauvegarder vos modifications. Il est possible qu'aucun code de distribution ne soit disponible sur cette page, si aucun n'existe dans votre application.

Remarque importante : le fait de sélectionner des codes de distribution particuliers pour vos profils n'affecte pas le choix des dossiers à exporter, seulement les contenus du champ Distribution que vous envoyez. Les dossiers à envoyer sont sélectionnés au moment de l'exportation.

Les changements soumis sont mis en œuvre immédiatement. Il n'est pas nécessaire de soumettre le formulaire principal de modification du profil d'exportation pour sauvegarder vos modifications.
