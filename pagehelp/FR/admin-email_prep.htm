La page Préparation de la demande mise à jour par courriel vous permet de préparer (et de personnaliser si vous le souhaitez) une demande de mise à jour par courriel à envoyer à un ou plusieurs organismes ou personnes contacts.

Si vous envoyez une seule demande de mise à jour, le texte en haut de la page indiquera le numéro du dossier pour lequel vous envoyez la demande (et qui servira également de lien vers les détails du dossier), ainsi que des renseignements sur le dernier courriel de mise à jour envoyé à l'organisme ou la personne contact. Si vous envoyez une demande de mise à jour en nombre à plusieurs organismes ou personnes contacts, les renseignements en haut de la page indiqueront le nombre de dossiers pour lesquels vous préparez le courriel de mise à jour.

Modifier le texte du message par défaut

S'il existe plusieurs versions du texte normalisé de mise à jour par courriel pour ce type de message, vous pouvez sélectionner un autre message par défaut dans le menu déroulant en haut de la page. Cliquez sur le bouton « Changer le message » pour mettre à jour la page avec votre sélection.

Voir et personnaliser le texte du message

La section « Message actuel » de la page vous montre le courriel qui sera envoyé si vous choisissez de ne pas personnaliser le message. Le texte du message contenu entre crochets, comme [NOM D'ORGANISME DU DOSSIER], sera automatiquement rempli par le logiciel lors de l'envoi du courriel. La section en bas de la page, « Sections modifiables », vous permet de personnaliser le sujet et la partie supérieure du corps du courriel qui sera envoyé.

Modifier la vue utilisée pour les liens vers un dossier

Si vous maintenez plusieurs vues publiques, vous pouvez choisir d'envoyer la demande de mise à jour avec des liens vers une vue spécifique. Si vous avez un nom de domaine associé à une vue, l'URL apparaîtra à côté du nom de la vue.

Prévisualiser le message

Une fois que vous êtes satisfait avec le message du courriel à envoyer, cliquez sur le bouton « Prévisualiser le courriel » en bas de la page afin de passer à la page suivante, où vous pouvez voir le message qui sera envoyé.
