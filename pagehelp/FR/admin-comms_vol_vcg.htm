La page « Groupes de communautés de bénévolat » vous permet de modifier la liste qui apparaîtra dans la légende géographique de la page de résultats de recherche, affichant quels boutons/icônes correspondent à quelles régions. Les groupes de communautés sont le lien entre un ensemble de communautés et les communautés qui sont alors associées à un dossier d'occasion. Dans un ensemble de communautés, chaque communauté doit appartenir exactement à un groupe de communautés, bien que la même communauté de bénévolat puisse faire partie d'autres groupes dans d'autres ensembles. Pour cette raison, vous devez ajouter des groupes de communautés à votre base de données avant de pouvoir ajouter des communautés à votre ensemble.

Si vous n'utilisez pas beaucoup de communautés, vous n'aurez peut-être pas besoin de les regrouper. Dans ce cas, il est logique de configurer un groupe de communautés pour chaque communauté pouvant être recherchée par les utilisateurs (il peut avoir le même nom que la communauté) ; vous devrez quand même assigner cette communauté au groupe de communautés, même s'ils ont le même nom.

Si vous avez besoin d'information supplémentaire sur l'organisation des communautés de bénévolat, ensembles de communautés, groupes de communautés et membres de groupe de communautés, consultez la page d'aide sur la page d'administration des communautés de bénévolat.

Dans un ensemble de communautés, chaque « nom de groupe » doit être unique ; cependant, vous pouvez réutiliser le même nom de groupe dans un autre ensemble de communautés.

Chaque groupe de communautés est associé à un « bouton » de couleur ou une autre icône de votre choix, dans la légende des pages de résultats de recherche. Le nombre de boutons de couleur est limité et vous ne pouvez utiliser chaque couleur qu'une fois par ensemble. Pour cette raison, il y a un nombre total maximal de groupes que vous pouvez créer (11 pour le moment), sauf si vous utilisez vos propres icônes en fournissant une « URL d'image ». À noter que ce n'est pas une bonne idée d'avoir un trop grand nombre de groupes, car les utilisateurs se sentiront submergé par la légende et la liste d'icônes. Vous pourriez plutôt réorganiser vos groupes en de plus grandes zones de couverture.

Modifier un groupe existant

Il est tout à fait possible de réorganiser ou renommer vos groupes de communautés en tout temps, puisque vos dossiers ne contiennent aucune information sur les groupes de communautés. On ne peut mettre à jour qu'un seul groupe de communautés à la fois. Cliquez sur le bouton « Mettre à jour » pour sauvegarder vos modifications.

Ce formulaire vous permet seulement de modifier le nom du groupe et l'icône associée au groupe ; la modification de communautés spécifiques qui sont membres de ce groupe peut s'effectuer dans le formulaire de configuration pour Membres du groupe de communautés Bénévolat.

Ajouter un nouveau groupe

La zone pour ajouter un nouveau groupe de communautés est toujours la dernière entrée sur la page. Saisissez un nom unique pour le groupe de communautés et l'icône du bouton de couleur que vous souhaitez utiliser dans la légende des résultats de recherche dans le menu déroulant, puis cliquez sur « Ajouter ».

Retirer un groupe

Vous ne pouvez supprimer un groupe de communautés que s'il n'est pas utilisé directement par aucun membre du groupe de communautés Bénévolat. Le bouton « Supprimer » ne sera pas visible à côté des communautés qui sont utilisées. Vous devrez d'abord réassigner les communautés à un autre groupe avant de pouvoir supprimer la valeur du groupe.
