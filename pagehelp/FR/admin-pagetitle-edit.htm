La zone Gérer les titres de page vous permet de remplacer le titre intégré d'une page par une valeur personnalisée. Cette option est disponible uniquement pour un nombre de pages limité pour le moment. Ce titre apparaît dans la barre de titre de votre navigateur, ainsi que dans la zone de titre de l'en-tête de la page (tel que précisé dans le modèle de conception).

Si vous laissez le titre de remplacement vide, la valeur intégrée sera utilisée. La valeur par défaut est affichée au dessus de la boîte de texte où vous saisissez votre nouveau titre personnalisé.
