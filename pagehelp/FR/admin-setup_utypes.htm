Les Types d'utilisateur représentent les restrictions de sécurité de l'utilisateur, dictant les renseignements auxquels l'utilisateur aura accès et quels changements (le cas échéant) il pourra apporter à ces renseignements. La sélection d'un type d'utilisateur établira quelle vue l'utilisateur utilisera lors de la connexion et quelles autorisations il aura pour modifier des données, exporter des dossiers, faire des changements de configuration, etc. Les types d'utilisateurs sont établis individuellement pour chaque module de la base de données (Information communautaire et Bénévolat).

La page Gérer les types d'utilisateur vous permet de créer ou de modifier plusieurs types d'utilisateurs, qui peuvent alors être attribués à des utilisateurs. Par exemple, vous pouvez décider de créer plusieurs types d'utilisateurs pour vos administrateurs de base de données, le personnel d'I&R, les bénévoles ou un tiers partenaire ayant une vue spéciale.

Modifier un type d'utilisateur existant

Pour consulter ou modifier les paramètres d'un type d'utilisateur existant, sélectionnez le dans la liste, puis cliquez sur « Voir / Modifier un type d'utilisateur ».

Créer un nouveau type d'utilisateur

Pour créer un nouveau type d'utilisateur, sélectionnez >>CRÉER<< dans le menu déroulant, puis cliquez sur le bouton « Voir / Modifier un type d'utilisateur » pour être dirigé vers le formulaire d'ajout d'un nouveau type d'utilisateur.
