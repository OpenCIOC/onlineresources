Remarque : la page principale des statistiques constitue votre point de départ pour générer des statistiques sur les vues / l'utilisation des dossiers. Le formulaire sur cette page traite uniquement des statistiques sur les mises en relation. Pour des renseignements supplémentaires sur les mises en relation, consultez l'aide sur la page Mises en relation de bénévolat.

Le ou après le - saisissez la date à laquelle vous souhaitez faire débuter le rapport statistique. Cette date sera comprise dans les résultats.

Avant le - saisissez la date suivant le jour auquel vous souhaitez faire cesser le rapport statistique. Cette date ne sera pas comprise dans les résultats. Pour générer un rapport correspondant exactement au mois précédent, cliquez sur les boutons « Premier du dernier mois » et « Premier de ce mois ».

Seuil : comprend les occasions qui ont fait l'objet d'au moins X mises en relation (0 ou vide pour obtenir toutes les occasions).

Mots-clés de l'organisme : vous pouvez inclure des mots-clés ou des expressions issues du nom d'organisme pour trouver les mises en relation d'organismes spécifiques.

Dossiers appartenant à : s'il existe plus d'un propriétaire de dossiers dans la base de données, vous pouvez demander ici de retourner des résultats en fonction de ce critère.

Afficher le décompte des placements : si vous souhaitez voir un décompte du nombre de mises en relation avec placement réussi, sélectionnez Oui pour cette option.

Version imprimable : si vous souhaitez publier le rapport en mode impression, sélectionnez Oui pour cette option. Les rapports exécutés en version imprimable s'ouvrent dans une nouvelle fenêtre.
