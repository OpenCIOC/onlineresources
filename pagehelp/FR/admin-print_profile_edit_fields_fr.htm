L'outil Rechercher / Remplacer des profils d'impression vous donne l'occasion de manipuler vos données avant de les afficher. Les commandes Rechercher / Remplacer fonctionnent sur les champs de manière individuelle, donc si vous souhaitez lancer une commande sur plusieurs champs, vous devez configurer une commande pour chacun d'entre eux. Vous pouvez lancer autant de commandes que vous le souhaitez, et préciser l'ordre dans lesquelles elles s'effectuent.

À quoi servent les expressions régulières ?

Les expressions régulières sont un outil incroyablement puissant. Deux exemples quelque peu complexes (et espérons-le, utiles) de ce qu'il est possible de faire sont inclus dans les sections ci-dessous. Même si vous ne vous sentez pas compétent à écrire ces commandes vous-mêmes, vous pourriez connaître quelqu'un ayant les connaissances techniques pour produire la commande. Écrire sur le Babillard de la communauté CIOC (en anglais) est une bonne manière de trouver l'aide pour créer une expression effectuant une tâche spécifique. On peut trouver de la documentation sur la fonctionnalité des expressions régulières de Microsoft sur les bibliothèques en ligne de Microsoft.

Exemple 1 : créer une liste à puces

Dans cet exemple, le formatage suivant est transformé :

* une puce
* une autre puce
* une troisième puce
  • une puce
  • une autre puce
  • une troisième puce

Ordre relatif Rechercher Remplacer par
1 ((\n|^)\* (.*))+ <ul>$&</ul>
2 (\n|^)\* (.*) $1<li>$2</li>

Exemple 2 : supprimer / modifier des hyperliens

Dans l'exemple suivant, un hyperlien est supprimé et remplacé par une formatage en gras à la place, par exemple :

Yahoo!   devient   Yahoo!

Rechercher Remplacer par
<a [^>]+>(.*)<\/a> <b>$1</b>

Le prochain exemple utilisera l'adresse du site web, retirera http:// si précisé, et la placera à la suite du nom entre parenthèses, par exemple :

Yahoo!   devient   Yahoo! (www.yahoo.com)

Rechercher Remplacer par
<a href="([^"]+)"[^>]*>(.*)<\/a> <b>$2</b> ($1)
