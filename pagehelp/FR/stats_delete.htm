Remarque : la page d'aide sur le formulaire de recherche principal sur les statistiques peut s'avérer utile pour obtenir un aperçu général des différents types de statistiques conservées dans votre base de données CIOC et savoir comment les utiliser de manière adéquate.

Il est recommandé de nettoyer régulièrement votre base de données des anciennes statistiques sur les dossiers consultés dont vous n'avez plus besoin, surtout si le trafic de votre base de données est élevé (p. ex., plus de 250 000 dossiers consultés par mois). Vous devriez choisir une date butoir appropriée pour supprimer les anciennes statistiques, comme la date de fin d'année de votre rapport précédent aux bailleurs de fonds. À noter que, en vertu de l'engagement de votre membriété envers CIOC, vous devez conserver au minimum 6 mois de statistiques en tout temps ; ces renseignements sont utilisés pour le calcul des frais d'adhésion.

Avant de supprimer

  • Assurez-vous d'avoir une sauvegarde/téléchargement complet de vos statistiques et de vos données à la fois (elles sont conservées dans des fichiers distincts). Le fait de conserver une sauvegarde complète vous permet de reproduire les rapports statistiques hors ligne, le cas échéant.
  • Consultez tous vos partenaires de données et confirmez votre intention de supprimer les anciennes statistiques. La suppression des statistiques n'est pas limitée à une vue ou une agence particulière ; toutes les statistiques du module en question seront supprimées pour la période courant jusqu'à la date sélectionnée.
  • Sauvegardez ou imprimez tout rapport statistique dont vous pourriez avoir besoin pour la période que vous supprimez.

Supprimer les statistiques

Cette page comporte un tableau avec deux colonnes : la colonne de gauche correspond à « Supprimer jusqu'à et y compris », avec les dates et des boutons radio pour sélectionner la date. La colonne de droite liste le nombre de statistiques qui seront supprimées si vous sélectionnez une date en particulier.

Pour supprimer les statistiques, cliquez sur le bouton radio du mois que vous souhaitez sélectionner. Cela supprimera toutes les statistiques du mois sélectionné et toutes les statistiques précédentes à ce mois. Si vous sélectionnez mars 2009, alors toutes les statistiques enregistrées durant et avant mars 2009 seront supprimées de manière permanente.

Remarque importante : une fois que les statistiques sont supprimées, elles le sont de manière permanente et ne peuvent pas être restaurées sans sauvegarde existante et l'aide d'un administrateur système (à vos frais). Il est recommandé de conserver une sauvegarde de votre téléchargement quotidien, donc si vous souhaitez consulter d'anciennes statistiques, elles seront disponibles dans votre fichier de téléchargement.
