La page Parcourir par domaine d'intérêt vous permet de consulter les occasions de bénévolat qui sont catégorisées dans un domaine d'intérêt spécifique. Cette page liste tous les domaines d'intérêt qui commencent par la lettre sélectionnée dans le menu « Choisir une lettre ». Il est également possible de lister tous les domaines alphabétiquement sur une seule page en utilisant le lien « Afficher tout ».

Le nombre de dossiers dans la vue actuelle qui contiennent le terme est identifié entre parenthèses à la fin de chaque domaine d'intérêt. Le fait de cliquer sur le nom d'un domaine d'intérêt spécifique retourne une liste des occasions actuelles dans votre vue qui contiennent le domaine d'intérêt.

Si le message qui apparaît sur le page est : « Trouvé 0 résultats », alors il n'y a pas de domaine d'intérêt dans la base de données qui commence par la lettre sélectionnée. Dans ce cas, utilisez le menu « Choisir une lettre » pour sélectionner une autre lettre. Sinon, retournez au menu principal en utilisant le bouton « Nouvelle recherche » ou le bouton retour de votre navigateur pour débuter une nouvelle recherche.

Un super-utilisateur peut ajouter à la liste des domaines d'intérêt en allant dans le menu Listes de contrôle > Domaines d'intérêt spécifique. Il est important de s'assurer que tous les dossiers d'occasions de bénévolat sont assignés aux domaines d'intérêt appropriés, afin de pouvoir les trouver lors de la recherche.
