Les ensembles de communautés vous permettent d'utiliser une sélection de communautés et d'occasions de bénévolat et de les rendre visibles dans une vue particulière. Vous pouvez utiliser un ensemble de communautés dans plusieurs vues, mais chaque vue ne peut avoir qu'un seul ensemble de communautés.

Chaque ensemble de communautés doit avoir un nom d'ensemble unique. Le nom d'ensemble est visible sur les formulaires de saisie de données lorsque vous avez plus d'un ensemble ; il est également visible dans la liste des ensembles à laquelle le dossier appartient si vous avez ajouté le champ Ensemble de communautés à la page de détails des dossiers dans votre vue. Il est utilisé dans le formulaire de configuration de la vue pour sélectionner l'ensemble de communautés utilisé pour cette vue, ainsi que dans le formulaire de mise à jour et l'outil de gestion des ensembles de communautés pour les occasions de bénévolat pour assigner un dossier à un ensemble ou plus.

La région desservie est visible sur la page du menu principal (« Cherchez-vous des possibilités de bénévolat dans ...») et sur la page Recherche : Étape 1. La même valeur Région desservie peut être utilisée dans plusieurs ensembles de communautés différents.

Certaines bases de données n'auront besoin que d'un seul ensemble de communautés. Vous aurez besoin de plus d'un ensemble de communautés si :

  • Vous souhaitez lister plusieurs zones géographiques dans plusieurs vues (comme sur les formulaires de recherche) ou
  • Vous souhaitez utiliser plusieurs ensembles de dossiers dans plusieurs vues (sans inclure les options d'affichage des dossiers internes et supprimés, etc.)

Pour modifier un ensemble existant, effectuez les changements, puis cliquez sur Mettre à jour à côté de l'ensemble de communautés que vous avez modifié. On ne peut modifier qu'un seul ensemble de communautés à la fois. Pour ajouter un nouvel ensemble, saisissez vos valeurs au bas du formulaire, puis cliquez sur Ajouter. Vous ne verrez l'option pour Supprimer un ensemble de communautés que s'il n'est pas utilisé dans une vue ou une occasion.
