Rapport de recommandation d'activation

Contrairement au Rapport d'activation multiniveaux, cet outil peut vous aider à modifier les activations directement. Au moment où cet outil est chargé, un formulaire affichant les options disponibles et les résultats obtenus avec les options actuelles (par défaut) vous est présenté.

De plus, contrairement au Rapport d'activation multiniveaux, ce rapport ne donne des recommandations que sur les modifications évidentes que l'on peut effectuer en l'absence du soutien éditorial d'un gestionnaire de données qualifié. Cela signifie que des éléments ne sont pas inclus, comme une branche où des dossiers sont indexés à plusieurs niveaux, puisqu'un rédacteur approprié doit d'abord prendre une décision sur le niveau préféré pour l'indexation future.

Quels types de recommandations l'outil offre-t-il ?

  • L'outil recommandera la désactivation lorsqu'il existe un terme actif sans dossier indexé avec ce terme, alors qu'un autre terme actif dans une même branche est indexé à des dossiers.
  • L'outil recommandera l'activation lorsqu'il existe un terme inactif indexé à des dossiers (ce changement sera obligatoire si vous soumettez le formulaire).
  • L'outil recommandera l'activation lorsqu'il existe un terme inactif dont le terme supérieur et un ou plusieurs termes inférieurs sont actifs (ce changement sera également obligatoire si vous soumettez le formulaire). Même s'il n'est pas conseillé d'activer à plusieurs niveaux, si vous choisissez de conserver des activations multiniveaux, vous devez le faire de manière complète.
  • Si vous avez sélectionné l'option « Recommander des activations », l'outil recommandera également l'activation des termes dans les branches où aucun autre terme n'est actif. Le cas échéant, l'outil recommandera d'utiliser le niveau auquel d'autres termes de même niveau sont déjà activés.

Paramétrer les options du rapport

Avant de soumettre les changements désirés, vous devez ajuster les options de recommandations et régénérer le rapport.

  • Afficher les activations globales - cette option s'adresse aux bases de données multimembres pour faire la distinction entre les activations globales qui s'appliquent à tous les membres d'une base de données et les activations locales qui s'appliquent uniquement à votre membriété.
  • Désactiver les termes inutilisés ayant des termes utilisés dans la même branche - si cette option n'est pas sélectionnée, aucun désactivation n'est recommandée.
  • Inclure les dossiers partagés - dans une base de données multimembre, utilisez cette option pour faire des recommandations qui prennent en compte les dossiers partagés avec vous par d'autres membres.
  • Faire remonter les termes de niveau inférieur - lorsqu'il est recommandé de désactiver un terme dont le terme supérieur est actif et utilisé, cette option remonte le terme en synonyme du terme supérieur au lieu de le désactiver.
  • Exclure la branche Y - comme les activations multiniveaux sont autorisés dans la branche Y (populations cibles), cette option ignore les recommandations de désactivation de cette branche.
  • Recommander des activations - cette option recommande de nouvelles activations sur les branches qui n'ont pas de termes actifs. Cela s'avère particulièrement utile suite à une mise à jour de la Taxonomie, pour mettre en valeur les nouveaux termes.

Consulter le rapport

Le rapport est divisé en deux sections :

  • la première section liste les recommandations de désactivation et les activations obligatoires (le cas échéant)
  • la seconde section liste les recommandations d'activation

À côté du code et du nom du terme, l'action indique si le statut d'activation du terme sera changé à Actif, Inactif ou Remonter.

Soumettre des modifications

Vous pouvez choisir d'accepter certaines ou toutes les recommandations proposées. Par défaut, toutes les recommandations sont cochées - mais vous pouvez cocher ou décocher les éléments pour sélectionner uniquement ceux que vous souhaitez. Vous pouvez utiliser le bouton « Désélectionner tout » pour commencer à partir d'une sélection vide. Les recommandations sans case à cocher contredisent les règles de base du logiciel (comme utiliser un terme inactif pour activer des dossiers) et sont donc obligatoires. Ces changements obligatoires seront mis en œuvre automatiquement dès la soumission du formulaire. Une fois que vous avez soumis le formulaire en cliquant sur « Appliquer tous les changements sélectionnés et obligatoires », vos modifications d'activation seront mises en œuvre immédiatement.
