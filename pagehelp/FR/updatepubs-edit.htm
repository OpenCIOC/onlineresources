Cette page est utilisée pour modifier la description de publication et gérer la liste des en-têtes généraux de la publication actuelle du dossier sélectionné. Si vous travaillez dans une vue limitée (par publication), vous ne serez autorisé à mettre à jour que la description et les en-têtes de la publication associée à votre vue. Autrement, vos modifications sur cette page seront appliquées à n'importe quelle publication que vous aurez choisi.

Gestion des données

Comme les mises à jour de publication peuvent s'effectuer indépendamment des mises à jour d'autres dossiers, les données de publication possèdent leurs propres renseignements de gestion des données (Date de dernière modification et Modifié par) qui sont enregistrées automatiquement par le système.

Descriptions

Utilisez ce champ pour ajouter une description spéciale à la publication actuelle. La description s'applique à la publication actuelle uniquement et sera supprimée si la publication est supprimée de ce dossier. Si vous utilisez un logiciel bilingue, un champ de description en français est également disponible. Depuis l'écran de configuration de la publication actuelle, on peut configurer la description de publication comme un champ d'affichage régulier, avec un nom personnalisé, et l'ajouter aux détails du dossier ou aux formulaires de rétroaction de chaque vue.

En-têtes généraux

Que sont les en-têtes généraux ?

Les en-têtes généraux vous permettent de classifier des dossiers dans une publication. Dans les vues basées sur des publications, on peut utiliser les en-têtes pour créer des recherches par « listes rapides » sur le formulaire de recherche simple. Les en-têtes sont également interrogeables depuis le formulaire de recherche avancée dans n'importe quelle vue ayant accès à cette publication. On doit d'abord assigner les dossiers à une publication avant de pouvoir les indexer aux en-têtes généraux de cette publication. Pour créer de nouveaux en-têtes généraux pour cette publication, utilisez le lien « Publications » sur le menu latéral de la page de démarrage de l'application ou depuis le menu de configuration (en admettant que vous ayez l'autorisation). Puis revenez sur cette page et ces en-têtes pourront alors être ajoutés à ce dossier.

Certains en-têtes peuvent être non utilisés et destinés aux index imprimés uniquement ou basés sur la Taxonomie (assignés en fonction des catégories de services indexée au dossier) et ne peuvent être ajoutés à des dossiers. Si des en-têtes peuvent être ajoutés à un dossier dans cette publication, vous aurez le choix de :

  • Retirer les en-têtes existants en décochant la case à côté des en-têtes sous « En-têtes existants ».
  • Ajouter de nouveaux en-têtes en cochant la case dans la section « Ajout de nouveaux en-têtes ».
