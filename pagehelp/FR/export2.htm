Cette page sert à générer le fichier d'exportation une fois que vous avez sélectionné votre format d'exportation et quels dossiers à exporter. En fonction du nombre de dossiers et du format d'exportation que vous avez sélectionnés, l'exportation peut prendre un temps important pour s'effectuer, sans qu'aucune indication de progrès n'apparaisse.

Pendant que l'exportation s'effectue, le message « Démarrage de l'exportation. L'exportation est en cours. Veuillez patienter... » apparaîtra sur la page. Soyez patient durant le traitement de l'exportation, car dans certains cas, la page ne comporte aucune indication du progrès de l'exportation (à noter que le format de partage affiche une barre de progression dans la plupart des navigateurs). Si vous quittez la page durant le traitement, le fichier d'exportation sera quand même créé la plupart du temps, mais ne sera pas accessible car vous n'aurez reçu aucun lien. Pour cette raison, veuillez patienter - n'essayez pas de recharger la page et ne quittez pas la page pour y retourner. Cela aura pour conséquence de créer inutilement de nombreux fichiers volumineux sur le serveur.

Une fois l'exportation effectuée, le message « L'exportation est terminé ! Faites un clic droit sur le lien ci-dessous et sélectionnez « Sauvegarder le fichier sous » pour le télécharger. » apparaîtra sur la page. C'est à moment là que l'exportation est terminée. Pour télécharger le fichier d'exportation traité sur votre ordinateur, faites un clic droit sur le lien « Fichier d'export » et sélectionnez « Sauvegarder le fichier sous... » ou « Enregistrer la cible sous... ». L'intitulé exact de ce choix dans le menu de votre navigateur dépendra du navigateur spécifique que vous utilisez. Dans la plupart des installations du logiciel, votre dossier sera converti au format Zip (compressé) par défaut ; vous aurez besoin d'un utilitaire de compression comme 7-zip pour l'ouvrir. Vous devriez utiliser la version compressée autant que possible, car elle est plus petite et plus rapide à télécharger. Vous pourriez également recevoir un lien vers une autre version non compressée du fichier. Dans certaines bases de données, les liens peuvent être inversés (lien non compressé en premier).

Vous pourriez recevoir un lien vers la version non compressée du fichier. Veuillez ne pas utiliser ce lien, à moins d'avoir un problème pour faire fonctionner les fichiers zippés ; le fichier non compressé prend plus de temps à télécharger et utilise plus de bande passante.

Si vous faites un clic gauche sur le fichier par accident, votre navigateur tentera d'ouvrir le fichier. Si vous ouvrez le fichier par accident, utilisez le bouton retour de votre navigateur pour revenir au lien « Fichier d'export » et faites un clic droit sur le lien du fichier d'exportation.

Il est recommandé de sauvegarder le fichier sur votre ordinateur local. À partir de là, vous pouvez travailler avec le fichier exporté comme vous voulez.
