La page « Communautés de bénévolat » vous permet de modifier la liste des communautés de Bénévolat qui apparaîtront comme critère pour le champ « Centres d'intérêt / Nombre de postes » (NUM_NEEDED) sur vos pages de recherche et dans les formulaires de rétroaction et de saisie. À noter que cela sert à préciser les valeurs du formulaire de recherche et les valeurs par défaut des formulaires de saisie ; mais vous pouvez pas ajouter n'importe quelle communauté à vos dossiers, mais si elle ne fait pas partie de votre ensemble de communautés.

Les groupes de communautés sont le lien entre un ensemble de communautés et les communautés qui sont alors associées à un dossier d'occasion. Dans un ensemble de communautés, chaque communauté doit appartenir exactement à un groupe de communautés, bien que la même communauté de bénévolat puisse faire partie d'autres groupes dans d'autres ensembles. Pour cette raison, vous devez ajouter des groupes de communautés à votre base de données avant de pouvoir ajouter des communautés à votre ensemble.

Si vous n'utilisez pas beaucoup de communautés, vous n'aurez peut-être pas besoin de les regrouper. Dans ce cas, il est logique de configurer un groupe de communautés pour chaque communauté pouvant être recherchée par les utilisateurs (il peut avoir le même nom que la communauté) ; vous devrez quand même assigner cette communauté au groupe de communautés, même s'ils ont le même nom.

Si vous avez besoin d'information supplémentaire sur l'organisation des communautés de bénévolat, ensembles de communautés, groupes de communautés et membres de groupe de communautés, consultez la page d'aide sur la page d'administration des communautés de bénévolat.

Remarque : la liste globale des communautés est une ressource partagée qui est également utilisée dans le module Information Communautaire ; elle peut être modifiée dans la liste partagée des systèmes de classification dans la page de configuration générale.

Modifier une communauté existante

Le seul changement que vous pouvez effectuer sur une communauté existante est le groupe de communautés auquel elle appartient. Si vous devez modifier une communauté utilisée, vous devrez ajouter une nouvelle communauté et retirer la référence à la communauté initiale.

On ne peut mettre à jour qu'une seule communauté à la fois. Cliquez sur le bouton « Mettre à jour » pour sauvegarder vos modifications.

Ajouter une nouvelle communauté

La zone pour ajouter une nouvelle communauté est toujours la dernière entrée sur la page. Sélectionnez un nom pour la communauté, ainsi que le groupe de communautés auquel vous souhaitez l'associer dans le menu déroulant, puis cliquez sur « Ajouter ».

Retirer une communauté

Quand vous « supprimez » une communauté de cette liste de membres du groupe, vous ne pouvez pas modifier les données trouvées dans les dossiers d'occasions. C'est pourquoi l'option pour supprimer de cette liste est toujours disponible. Une fois qu'une communauté a été supprimée de la liste des membres du groupe, elle ne sera plus disponible sur aucun formulaire pour cet ensemble de communauté, sauf s'il existe déjà des données dans le dossier d'occasion pour la communauté dans le champ « Centres d'intérêt / Nombre de postes » (NUM_NEEDED).
