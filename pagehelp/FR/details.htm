La page « Détails sur le dossier d'organisme » liste tous les détails du dossier qui sont visibles dans votre vue actuelle. Un super-utilisateur peut modifier les champs disponibles dans cette vue et sous quels en-têtes les champs sont regroupés, depuis la configuration de la vue. On peut modifier les noms des champs et leur ordre dans les paramètres « Affichage des champs ».

Cette page d'aide indique les 5 types d'actions principales disponibles sur les détails du dossier depuis cette page :

  • Parcourir les résultats de la recherche
  • Liens des noms d'organisme
  • Éléments du menu des détails du dossier
  • Autres liens sur la page de détails

Parcourir les résultats de la recherche

Si ce dossier est l'un des nombreux dossiers retournés lors de votre dernière recherche, vous verrez des liens en haut et en bas du dossier :

Premier Précédent | Suivant Dernier (10 de 100 Total)

Vous pouvez utiliser Premier, Précédent, Suivant et Dernier pour parcourir vos résultats de recherche un par un, ou le lien Total pour retourner à la page de résultats et consulter la liste complète des résultats.

Remarque : les liens de navigation dans les résultats de recherche peuvent ne pas fonctionner correctement ou peuvent ne pas apparaître du tout si vous avez lancé une nouvelle recherche, puis avez utilisé votre bouton retour pour retourner aux résultats d'une recherche précédente. Pour que ces liens fonctionnent de manière fiable, vous devez vous rendre sur les pages de détails des dossiers directement depuis la page « Résultats de recherche » ou « Explorer par organisme », sans lancer d'autres recherches dans votre navigateur.

Liens des noms d'organisme

CIOC utilise une structure de nom à 5 niveaux, avec des champs nommés de ORG_LEVEL_1 à ORG_LEVEL_5 (l'affichage du nom des ces champs peut varier en fonction de vos paramètres). ORG_LEVEL_1 représente l'organisme parent le plus élevé, avec des niveaux inférieurs représentant plusieurs succursales, programmes ou emplacements. Si vous utilisez cette structure correctement et de manière constante, vous pourrez utiliser la fonctionnalité des liens sur les noms afin de rechercher des dossiers appartenant à la même structure d'organisme parent, des niveaux 1 à 4, où au moins un dossier correspondant peut être trouvé. Cette fonctionnalité est disponible uniquement si elle est activée dans cette vue.

Proposer une mise à jour

Visible de tous les utilisateurs et du public, ce lien vous dirige vers une page de rétroaction où vous pouvez faire des suggestions de modification aux détails du dossier actuel. Les changements apportés au dossier ne seront pas visibles immédiatement, mais seront d'abord révisés par un gestionnaire de données avant d'être ajoutés au dossier officiel.

Mettre à jour le dossier

Vous aurez accès à l'option de mise à jour du dossier dans le menu déroulant si vous avez l'autorisation de mettre à jour le dossier en question dans la vue actuelle. Même si le lien vous dirige toujours d'abord vers le formulaire dans la langue actuelle, vous aurez l'occasion de mettre à jours les versions dans d'autres langues après avoir soumis vos modifications. Vous pouvez également basculer vers une autre langue si vous souhaitez commencer les modifications dans une autre langue. Les changements apportés au dossier par l'entremise du formulaire de mise à jour seront mis en œuvre immédiatement.

Supprimer / Restaurer

Supprimer vous permet d'établir la date de suppression d'un dossier ; les dossiers dont la date de suppression est passée ne seront plus accessibles par les recherches ordinaires, tandis que les dossiers dont la date de suppression est dans le futur resteront actifs jusqu'à échéance de la date. Les dossiers supprimés affichent le lien Restaurer ; le fait de restaurer un dossier supprimera sa date de suppression. Les fonctionnalités de suppression et de restauration sont disponibles uniquement aux utilisateurs ayant l'autorisation de supprimer des dossiers.

Mettre à jour les catégories de service

Cette option du menu déroulant vous permet de mettre à jour l'indexation des catégories de service (Taxonomie des services humains) du dossier. Cette option est disponible uniquement aux utilisateurs ayant l'autorisation d'indexer avec la Taxonomie et est indépendante de la capacité à mettre à jour le reste du dossier.

Mettre à jour les publications

Lien pour mettre à jour la liste des codes de publication attachés à ce dossier, ainsi que tout renseignement sur les publications, comme les descriptions de publication et les en-têtes généraux. Le lien est accessible si vous avez l'autorisation de mettre à jour les publications et si vous n'êtes pas actuellement dans une vue limitée par publication. Si vous travaillez dans une vue « limitée , vous verrez le lien « Mettre à jour les descriptions / en-têtes » à la place.

Mettre à jour les descriptions / en-têtes

Lien pour mettre à jour les descriptions de publication et les en-têtes généraux de la publication sur laquelle est basée votre vue. Le lien est accessible uniquement lorsque vous travaillez dans une vue limitée par publication, si vous avez l'autorisation de modifier les renseignements de publication.

Demander la mise à jour par courriel

Ce lien vous dirige vers une page où vous pouvez rédiger / personnaliser un courriel à l'attention d'un organisme afin de demander une mise à jour de leurs renseignements. Ce lien est disponible si vous avez l'autorisation d'envoyer des demandes de mise à jour par courriel, si l'organisme a un courriel disponible pour l'envoi de mise à jour contenu dans le dossier et si le champ Bloquer demandes par courriel n'a pas été activé dans ce dossier.

[ Haut ]

Expédier le formulaire par la poste

L'option d'envoi du formulaire par la poste dans le menu déroulant vise à être utilisé lorsque vous ne pouvez pas utiliser la fonctionnalité de demande de mise à jour par courriel pour demander la mise à jour de ce dossier. Si vous avez établi un modèle d'impression dans cette vue, il sera utilisé dans le formulaire de courriel ou de télécopie. Comme pour la page de rétroaction, le formulaire de courriel ou de télécopie réunit les coordonnées en fonction des spécifications sur l'agence propriétaire contenues dans la configuration de l'agence.

Vous pouvez aller dans la section « Mise en page » de votre navigateur (dans le menu « Fichier ») et élargir vos marges ou supprimer les en-têtes et pieds de page avant d'imprimer. Le fait de supprimer les en-têtes et pieds de page dans la mise en page empêchera votre navigateur d'imprimer automatiquement les numéros de page, l'URL de la page, les renseignements de date et d'heure, etc. Le fait d'augmenter les marges de la page peut également réduire le nombre de pages nécessaires pour imprimer le formulaire de mise à jour.

Le formulaire imprimable s'ouvre dans une nouvelle fenêtre ou un nouvel onglet du navigateur. Lorsque vous avez terminé, fermez la fenêtre ou l'onglet pour retourner à la page précédente.

Disponible uniquement si le module Bénévolat est utilisé. Les versions imprimables sont formatées pour l'impression en utilisant le modèle fourni ; tous les liens générés par le système et les éléments du menu seront masqués.

Rappels

Le lien Rappels vous permet de consulter, de modifier ou de compléter les rappels attachés à ce dossier. Si le dossier contient un rappel actuellement actif, une icône apparaîtra à côté du lien du rappel. Une icône d'alarme indique les rappels actifs, alors qu'une icône d'alerte indique qu'au moins un rappel est en retard. Pour de l'aide supplémentaire sur les rappels, consultez le fichier d'aide sur la page principale des rappels.

Autres liens sur la page de détails...

Dans le dossier lui-même peuvent figurer d'autres liens vers des nouvelles recherches ou des sites web externes. De nombreux champs CIOC vous permettent de saisir du HTML, vous permettant de créer vos propres liens dans le dossier. Certains des liens créés par le logiciel comprennent :

Liens vers le site web et le courriel

Les différents champs de site web ou de courriel du logiciel seront hyperliés automatiquement par le logiciel, sauf dans la version imprimable du dossier. Si un champ de courriel unique contient plusieurs adresses courriel, le fait de cliquer sur le lien ouvrira votre logiciel de courriel pour rédiger un message destiné à toutes les adresses listées.

Liens vers des médias sociaux

À noter que deux types de liens vers des médias sociaux peuvent être disponibles sur cette page :

  1. Liens vers des médias sociaux sur l'organisme / programme : cela peut comprendre les liens vers la page Facebook, le fil Twitter, etc., de l'organisme.
  2. Liens de partage sur les médiaux sociaux : ces liens apparaissent dans le coin supérieur droit du dossier (si activé dans la vue) et permettent à l'utilisateur de partager un lien vers le dossier en question sur sa propre page Facebook, fil Twitter, etc. Cette fonctionnalité utilise le service « AddThis » qui fonctionne avec des centaines de médiaux sociaux différents.

Lien cartographique

Ce lien apparaît dans le champ « Adresse » d'un organisme uniquement si le dossier contient suffisamment d'information pour cartographier correctement. Le fait de cliquer sur ce lien ouvrira une carte de l'emplacement de l'organisme. La carte MapQuest s'ouvrira dans la même fenêtre que la base de données ; vous devrez utiliser le bouton retour de votre navigateur pour revenir à la base de données après avoir consulté la carte. À noter que l'adresse postale n'est jamais cartographiée.

Le système de cartographie utilisé dépend du paramétrage du champ Cartographier le dossier ; les choix disponibles sont configurés dans la section Systèmes cartographiques de la configuration.

Liens vers les systèmes de classification (sujets, Taxonomie, publication, distribution, SCIAN, etc.)

En fonction de la vue de votre base de données, vous verrez plusieurs liens relatifs aux sujets / termes, publications / en-têtes, distribution, termes génériques et spécifiques, dans les champs du dossier. Le fait de cliquer sur ces liens lancera une recherche sur les dossiers qui contiennent le terme ou code sélectionné.

Autres liens de recherche

Un certain nombre d'autres champs contiennent également des liens vers de nouvelles recherches, y compris :

  • Accréditation
  • Lignes de bus
  • Certification
  • Champs Liste de contrôle supplémentaire (les noms peuvent varier)
  • Champs Liste déroulante supplémentaire (les noms peuvent varier)
  • Membriété
  • Écoles avoisinantes / Escorte aux écoles (garderie)
  • Type de programme (garderie)
  • Type de garde (garderie)

Modifier votre vue

Si votre niveau de sécurité l'autorise, vous pourrez voir également un menu déroulant en haut de la page où vous pouvez changer la vue de la base de données ; cela peut modifier l'apparence de la page d'affichage du dossier ou modifier les champs disponibles. Contrairement au fait de changer votre vue depuis le menu principal, ce changement est temporaire et cessera après la consultation du dossier. Remarque importante : vous ne pouvez mettre à jour que des dossiers dans votre propre vue ; si vous basculez vers une autre vue qui n'est pas votre vue par défaut, vous n'aurez plus accès au lien Mettre à jour le dossier.
