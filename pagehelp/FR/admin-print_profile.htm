Les profils d'impression vous permettent de créer des modèles personnalisés pour l'impression de listes de dossiers.

Modifier un profil d'impression existant

Pour consulter ou modifier les paramètres d'un profil d'impression existant, sélectionnez le profil dans la liste et cliquez sur « Voir / Modifier le profil ».

Ajouter un nouveau profil d'impression

Pour créer un nouveau profil d'impression, vous devez donner un nom unique au profil. S'il existe déjà des profils, vous pouvez copier un profil en le sélectionnant dans la liste « Copier le profil ». Si vous ne sélectionnez pas de profil à copier, votre nouveau profil contiendra les paramètres par défaut, sans ajouter de champ. Cliquez sur le bouton « Ajouter un profil » pour créer le nouveau profil ; vous serez dirigé vers le formulaire de modification de ce nouveau profil.
