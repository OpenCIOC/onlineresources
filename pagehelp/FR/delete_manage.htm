La page « Gérer les dossiers supprimés » affiche une tableau comprenant tous les dossiers qui ont été marqués pour suppression dans la base de données. Les dossiers avec une date de suppression dans le passé n'apparaîtront pas dans les recherches ordinaires, alors que les dossiers avec une date de suppression dans le futur (« en attente de suppression ») sont toujours accessibles. Les dossiers supprimés peuvent être accessibles par la Recherche avancée, si les dossiers supprimés sont visibles dans la vue actuelle.

Un dossier qui est marqué comme supprimé (avec une date de suppression) contient toujours toutes ses informations, classifications et relations, à moins qu'on les retire volontairement. Dans les cas des dossiers d'information communautaire, les dossiers supprimés continuent d'être associés au numéro de dossier de leur agence, ce qui empêche de réutiliser leur numéro dans de nouveaux dossiers.

Depuis cette page, les utilisateurs ayant des privilèges de suppression peuvent modifier la date de suppression ou restaurer un dossier (retirer son statut supprimé ou en attente de suppression). Les super-utilisateurs peuvent également supprimer définitivement un dossier ou un groupe de dossiers.

Les dossiers supprimés définitivement sont retirés de la base de données de manière irrévocable et ne seront plus visibles par personne. Parce que retirer des dossiers peut affecter les rapports statistiques et d'autres renseignements, il est généralement déconseillé de supprimer les dossiers de manière permanente. Si les dossiers doivent être supprimés, le meilleur moment pour le faire est suite à une purge des statistiques, comme il est coutume de le faire à la fin de l'année. Si le dossier a été supprimé depuis suffisamment longtemps, il aura peu, voir pas de statistiques actuelles, et pourra être retiré en toute sécurité, sans causer d'anomalies dans les rapports. Une fois qu'un dossier a été supprimé définitivement, son numéro de dossier d'agence pourra être réutilisé. Les suppressions permanentes ne peuvent être effectuées que par des comptes de super-utilisateurs.

Suppression définitive en nombre

Pour supprimer définitivement un grand groupe de dossiers, vous pouvez cocher les cases à côté des dossiers ou utiliser le bouton « Sélectionner tout » pour sélectionner tous les dossiers disponibles. Cliquez sur le bouton « Supprimer les dossiers sélectionnés définitivement » pour retirer tous les dossiers sélectionnés de manière permanente. S'il n'y a pas de case à côté d'un dossier, il ne peut pas être supprimé, en général parce qu'il est encore utilisé dans la base de données (comme un dossier d'information communautaire encore utilisé dans une occasion de bénévolat). Vous devez d'abord retirer les références au dossier avant de pouvoir le supprimer.

Veuillez utiliser cet fonctionnalité avec grande prudence, car les dossiers ne peuvent pas être récupérés une fois retirés. Si vous prévoyez de supprimer un ensemble de dossiers en une seule fois, il est fortement conseillé d'exporter d'abord ces dossiers en utilisant un profil de Format de partage CIOC complet, qui vous donnera la chance de restituer ces dossiers en cas d'erreur. Comme le module Bénévolat ne supporte pas d'outil d'importation / exportation pour le moment, il peut être approprié de sauvegarder les pages de détails des dossiers ou de conserver une archive de la base de données sur Microsoft Access.
