La page principale Importer les données du dossier constitue le menu principal de l'outil d'importation. L'outil d'importation vous permet de charger des fichiers XML de Format de partage CIOC qui ont été exportés par d'autres bases de données. Le menu sur cette page vous donne la liste des fichiers d'exportation existants avec lesquels travailler et offre un lien pour Télécharger un nouvel ensemble de données, afin de charger un nouveau fichier d'exportation.

Remarque importante : les fonctionnalités Mettre à jour / Ajouter des dossiers de l'outil d'importation utilisent beaucoup de ressources serveur, ce qui peut compromettre l'opération normale de la base de données. Il est fortement recommandé de ne pas utiliser l'outil dans les heures de pointe (9 h HNE-17 h HNE). L'utilitaire file d'attente des importations peut vous aider à planifier vos importations durant heures creuses. Le processus de chargement du fichier ne cause pas de problème de performance sur le serveur et peut s'effectuer en tout temps ; cependant, il est recommandé de ne pas laisser un grand nombre de fichiers d'exportation non traités chargés dans la base de données, pour des raisons d'espace.

Charger un nouvel ensemble de données

Pour travailler avec un nouveau fichier d'exportation, cliquez sur le lien Télécharger un nouvel ensemble de données ; une fenêtre contextuelle s'ouvrira avec l'outil de chargement des données. Vous devez utiliser un fichier XML actuel, au format de partage CIOC valide ; l'outil n'acceptera aucune autre forme de données et peut ne pas accepter un fichier d'une version antérieure ou plus récente du logiciel.

Dans la fenêtre contextuelle, cliquez sur le bouton « Parcourir » pour trouver sur votre ordinateur le fichier d'exportation que vous souhaitez charger. De manière optionnelle, si vous utilisez le champ « Importer comme », vous pouvez donner un nom convivial au fichier que vous chargez, pour vous aider à vous rappeler d'où et quand il provient, p. ex., « Données bilingues CIC, janvier 2013 ». Si vous ne précisez pas de nom, le nom de fichier original sera utilisé lors de l'affichage des renseignements sur le fichier. Il est possible de charger le même fichier deux fois et de donner le même nom convivial à deux fichiers différents. Prenez cela en compte lorsque vous chargez et renommez des fichiers.

Après avoir sélectionné le fichier sur votre ordinateur, cliquez sur le bouton « Charger le fichier ». Le fichier sera authentifié pour s'assurer qu'il s'agit d'un fichier XML au format de partage CIOC valide. S'il des problèmes surviennent lors de l'authentification du fichier, vous recevrez un rapport d'erreur et le fichier ne sera pas chargé en totalité. Signalez toute erreur à la personne qui vous a envoyé le fichier d'exportation et demandez-lui de créer un nouveau fichier d'exportation. Si vous recevez des erreurs, mais que certains dossiers semblent avoir été chargés avec succès, vous pouvez importer ces dossiers puisqu'ils ont passé l'étape d'authentification.

Travailler avec un ensemble de données existant

Une fois qu'un ensemble de données a été chargé, celui-ci apparaîtra dans la liste des ensembles de données, dans le menu principal de l'outil d'importation. Les dossiers à importer seront regroupés sous Ajouter ou Mettre à jour et les dossiers importés sous Terminé ; le décompte des dossiers sera indiqué à côté de chaque lien. Pour ajouter ou mettre à jour des dossiers, cliquez sur le lien approprié pour être dirigé vers la page Ajouter / Modifier ou cliquez sur « Mettre tous les dossiers dans la file d'attente » pour ajouter le fichier d'importation à la file d'attente et l'importer plus tard. On peut traiter les fichiers dans la file d'attente en cliquant sur le lien « Commencer le traitement » ; la file d'attente traitera un nombre de fichiers illimité, les ajouts comme les mises à jour, jusqu'à 12 heures à la fois. Pour voir une rapport sur les dossiers ayant déjà été importés, cliquez sur le lien Terminé .

Un rapport sur le fichier chargé est accessible par le lien « Renseignements supplémentaires ». Il est recommandé de réviser attentivement les rapports de Renseignements supplémentaires avant de lancer une importation. Le rapport indique le nom de fichier, le nom convivial, le nombre de dossiers à ajouter ou mettre à jour, les langues dans lesquelles les dossiers sont disponibles, les codes de publication et de distribution inclus et les champs à mettre à jour. Il est important de s'assurer que le fichier comprend les données attendues, de manière à ne pas écraser accidentellement les champs ou codes lorsque vous souhaitez conserver votre propres données. Vous aurez l'occasion de voir les données des dossiers individuels lors de la procédure Ajouter / Modifier.

Vous ne pouvez ajouter ou mettre à jour un dossier depuis un fichier chargé qu'une seule fois, car les données chargées sont remplacées par un rapport d'importation après l'importation du dossier ; si vous devez relancer l'importation, vous devez recharger le fichier d'exportation.

Une fois l'importation des dossiers effectuée, ou si vous n'avez plus besoin de l'ensemble de données, vous pouvez supprimer l'ensemble de données en utilisant le lien « Supprimer l'ensemble de données ». À noter que le rapport d'importation n'est plus accessible une fois que l'ensemble de données est supprimé ; s'il contient des renseignements à conserver aux fins d'archivage, vous devez sauvegarder ou imprimer le rapport d'importation disponible par le lien Terminé.

Vous pourriez voir certains dossiers marqués comme « Impossible d'importer » ; cela arrive si vous tentez d'importer un dossier ayant le même numéro qu'un dossier appartenant à un autre membre de votre base de données CIOC. Plutôt que d'importer le dossier, le membre doit le partager avec vous en utilisant un profil de partage. Si vous souhaitez avoir votre propre copie du dossier qui vous appartient, vous devez manuellement modifier le numéro du dossier en ouvrant le fichier XML dans un éditeur de texte et fournir un nouveau numéro unique de dossier avant de charger le fichier.

De nombreuses ressources sont disponibles sur le site de la communauté CIOC (en anglais) pour vous aider à partager vos dossiers. Consultez la section sur les Ressources ou envoyez un message sur le babillard pour plus de renseignements.
