La page « Téléchargement » fournit automatiquement des liens générés vers le téléchargement de la version de Microsoft Access la plus récente sur votre base de données. Pour des renseignements supplémentaires sur les fichiers téléchargés fournis par CIOC, y compris le calendrier de création de fichiers téléchargés actuel, consultez la page sur les téléchargements du site de la communauté CIOC sur : http://community.cioc.ca/resources/how-to/get-your-data-or-statistics-in-a-download (en anglais). Il existe une page de téléchargement par module du logiciel (Information communautaire ou Bénévolat) et seuls les super-utilisateurs du module peuvent accéder à la page.

Créer ou supprimer des ressources supplémentaires

En plus des liens générés automatiquement, vous pouvez ajouter votre propre liste de liens à la page de téléchargement. Les liens peuvent être ajoutés dans une ou plusieurs des langues disponibles dans votre base de données. Pour supprimer un lien personnalisé, cliquez sur le bouton de suppression à côté du lien.
