Cette page vous permet de modifier les détails d'un profil Excel, y compris les éléments de données à inclure et dans quelles vues on peut y accéder. Pour de l'information générale sur les profils Excel, consultez le fichier d'aide sur la page Gérer les profils Excel.

Nom du profil

Un nom unique pour le profil. Utilisé pour sélectionner le profil au cours de la procédure d'exportation.

Vues

Pour rendre ce profil Excel accessible aux fins d'utilisation, vous devez l'activer dans une ou plusieurs vues. Pour des raisons de sécurité, il est recommandé de n'activer chaque profil que dans les vues où il est nécessaire.

Champs

Les champs peuvent être ajoutés à votre profil de sortie Excel ou utilisés pour ordonner les colonnes dans le fichier. Il n'est pas nécessaire qu'un champ possède une valeur à la fois dans les colonnes « Ordre des colonnes » et « Ordre de tri des dossiers ». Le placement par ordre des colonnes ou par ordre de tri des dossiers se fait en ordonnant d'abord par le numéro indiqué, puis par le nom du champ. Vous pouvez utiliser le même numéro plusieurs fois pour plusieurs champs si vous souhaitez les ordonner par ordre alphabétique (par exemple, vous pouvez indiquer 0 pour chaque champ sélectionné dans « Ordre des colonnes » et cela placera automatiquement les colonnes du tableau par ordre alphabétique ; vous pouvez ajouter tous les champs de Niveau 1 d'org. à Niveau 5 d'org. avec un valeur de 0, et cela les ordonnera en commençant par le niveau 1, puis le niveau 2, puis le niveau 3, et ainsi de suite. La valeur maximale pouvant être indiquée est 255 et il n'est pas obligatoire que les numéros soient séquentiels ; en fait, si vous ajoutez des éléments par incrément de 10, cela vous laisse de la place entre chaque champ pour en ajouter dans le futur.
