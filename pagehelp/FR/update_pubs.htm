La page Mettre à jour les publications vous permet d'ajouter un nouveau code de publication à un dossier de la base de données ou de modifier / supprimer un code de publication existant attaché à un dossier. Si vous souhaitez ajouter, modifier ou supprimer un code de publication utilisé dans la base de données, cela doit être effectué par l'élément du menu Publications accessible depuis le menu principal sur la page de recherche simple. Les actions effectuées sur cette page n'affectent que les dossiers sélectionnés.

Ce qui apparaît sur cette page dépend des codes de publication déjà attachés au dossier et si les publications attachées ont des descriptions ou des en-têtes généraux associés.

À noter que les paramètres de votre vue actuelle limite les publications avec lesquelles vous pouvez travailler - vous ne pouvez voir, ajouter ou supprimer que les publications qui sont accessibles dans votre vue actuelle.

Le tableau sur cette page contient six colonnes :

  • Code : tout code de publication attaché à ce dossier apparaît dans cette colonne.
  • Nom : le nom « convivial » de cette publication, le cas échéant.
  • A une description : si une publication a une Description de publication associée pour ce dossier, trois astérisques (***) apparaissent dans cette colonne.
  • A des en-têtes : si une publication a un en-tête général associé pour ce dossier, trois astérisques (***) apparaissent dans cette colonne.
  • A une rétroaction : si un dossier contient une rétroaction sur les publications, trois astérisques (***) apparaissent dans cette colonne.
  • Action : s'il n'y a pas de code de publication encore attaché au dossier, il n'y aura que le bouton Ajouter disponible. À côté de chaque code de publication déjà attaché au dossier se trouvent les boutons Mettre à jour et Supprimer. Cliquez sur le bouton Mettre à jour pour être dirigé vers une nouvelle page où vous pouvez mettre à jour ou ajouter une description de publication ou modifier les en-têtes généraux associés à ce dossier. Le bouton Supprimer vous permet de supprimer un code de publication (et toute description de publication ou en-tête général) d'un dossier. Le fait de cliquer sur Supprimer vous dirigera vers une nouvelle fenêtre pour confirmer votre décision de retirer le code de publication du dossier. Pour ajouter un nouveau de code de publication au dossier, sélectionnez la valeur désirée dans la liste déroulante, puis cliquez sur le bouton Ajouter.
