Les modèles de conception contrôlent le style / l'apparence de votre base de données, y compris les couleurs, logos, feuilles de style, en-têtes et pieds de page, etc.

Les modèles qui sont contrôlés exclusivement par une agence particulière ont le code de cette agence à la fin du nom entre crochets (c'.-à-d. [ABC]). Vous pourrez sélectionner dans la liste des modèles que vous ne possédez pas, afin de les consulter, mais ne serez pas capable de les modifier.

Vous disposerez d'un modèle intégré au système ou plus dans votre base de données. Vous pouvez consulter ou copier ces modèles intégrés et les utiliser dans vos vues si désiré, mais ils ne peuvent être modifiés en aucune manière. Les modèles du système sont indiqués par un *.

Modifier un modèle existant

Sélectionnez dans le menu déroulant le nom du modèle que vous souhaitez modifier, puis cliquez sur le bouton « Voir / Modifier le modèle ».

Ajouter un nouveau modèle

Pour créer un nouveau modèle, cliquez sur le bouton « Ajouter un Modèle » pour être dirigé vers le formulaire d'ajout d'un nouveau modèle. S'il existe déjà des modèles, vous pouvez copier un modèle en le sélectionnant dans la liste « Copier un modèle existant ». Si vous ne sélectionnez pas de modèle à copier, votre nouveau modèle contiendra les paramètres par défaut.
