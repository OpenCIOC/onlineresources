Cette page fournit une liste des 50 dossiers de la base de données les plus consultés à ce jour, y compris leur utilisation par le public (utilisateurs non connectés) et l'utilisation totale (utilisateurs connectés et public). Vous pourriez avoir plus de 50 dossiers dans les résultats si le décompte des consultations de plusieurs dossiers sont « ex æquo » ; certaines lignes peuvent également être dupliquées ou sautées en raison des ex æquo. Si le dossier est accessible dans votre vue actuelle, vous pouvez le consultez en cliquant sur le lien du numéro de dossier (remarque : ce lien n'est pas disponible quand vous consultez le rapport en mode impression).

À noter que ces valeurs sont basées sur les statistiques sur les dossiers consultés internes au logiciel ; les statistiques sur les dossiers consultés ne sont pas conservées lorsque vous passez en mode impression ou changez de vue temporairement, pour ne pas gonfler les statistiques si quelqu'un consulte plusieurs versions d'un même dossier pour les mêmes raisons.
