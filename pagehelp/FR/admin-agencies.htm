Remarque : les contenus de cette page peuvent varier si vous travaillez dans une base de données avec d'autres membres CIOC (une base de données « multimembre »).

La page Gestion des agences vous permet d'ajouter une nouvelle agence (c'est-à-dire un propriétaire de dossiers) à la base de données ou de modifier une agence existante. Il est très important d'avoir une entrée pour chaque code de propriétaire de dossiers utilisé dans des dossiers de votre base de données, comprenant une adresse courriel et un dossier d'organisme associé (centre d'information communautaire) afin que les courriel de rétroaction soient traités correctement.

Ajouter une nouvelle agence

Cliquez sur le lien « Ajout d'une nouvelle agence » pour être dirigé vers un formulaire de saisie des renseignements sur la nouvelle agence. Les super-utilisateurs globaux (des super-utilisateurs qui contrôlent les valeurs partagées par tous les membres d'une base de données) ont la capacité de créer une agence pour n'importe quel membre de la base de données.

Modifier une agence existante

Sous le lien « Ajout d'une nouvelle agence » se trouve un tableau listant tous les codes d'agence associés à votre membriété CIOC. Ce tableau inclut un décompte de tous les utilisateurs, dossiers d'information communautaire et dossiers de bénévolat appartenant à chaque agence. Cliquez sur le lien « Mettre à jour » à côté de l'agence/propriétaire de dossiers que vous souhaitez modifier.

Agences pour les autres membres

Si vous travaillez sur une base de données multimembre, vous pourrez voir une liste des codes d'agence appartenant à d'autres membres CIOC dans votre base de données. Vous pourrez sélectionner l'un ou plusieurs de ces codes aux fins d'affichage sur vos propres formulaires de mise à jour et de recherche. Les super-utilisateurs globaux (des super-utilisateurs qui contrôlent les valeurs partagées par tous les membres de cette base de données) ont la capacité de gérer ces agences appartenant à d'autres membres.
