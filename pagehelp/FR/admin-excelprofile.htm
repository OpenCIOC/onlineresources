Les profils Excel vous permettent de configurer des profils afin de personnaliser des formats d'affichage, optimisés pour l'utilisation sous Microsoft Excel et des programmes similaires. Les profils Excel offrent des choix très variés pour l'affichage des données, combinant des éléments de données habituellement disponibles uniquement dans vos fichiers de téléchargement Microsoft Access, et aussi tous les types de champs pouvant être utilisés dans les pages de détails de dossiers. Tandis que les profils Excel offrent plus de choix sur ce qu'il est possible d'exporter, ils ne procurent aucune structure à l'information et s'avèrent donc moins utiles que les profils d'exportation comme moyen d'utiliser vos données dans d'autres systèmes (et ils ne peuvent pas être utilisés pour échanger des données entre bases de données CIOC).

Bien qu'ils soient distincts des profils d'exportation, les fichiers de sortie Excel sont créés au moyen des pages d'exportation. Les fichiers créés sont générés soit au format HTML, soit au format CSV (sélectionné au moment de l'exportation), qui sont importables dans la majorité des programmes de tableur.

Modifier un profil Excel existant

Pour consulter ou modifier les paramètres d'un profil Excel existant, sélectionnez le profil dans la liste, puis cliquez sur « Voir / Modifier un profil ».

Ajouter un nouveau profil Excel

Pour créer un nouveau profil Excel, vous devez donner un nom unique au profil. S'il existe déjà des profils, vous pouvez copier un profil en le sélectionnant dans la liste « Copier le profil ». Si vous ne sélectionnez pas de profil à copier, votre nouveau profil contiendra les paramètres par défaut, sans de champ ajouté. Cliquez sur le bouton « Ajouter un profil » pour créer le nouveau profil ; vous serez dirigé vers le formulaire de modification de ce nouveau profil.
