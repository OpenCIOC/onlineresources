La page Compte utilisateur vous permet de modifier certains paramètres relatifs à vos informations de compte de base de données. Les paramètres relatifs à vos autorisations dans la base de données (votre type d'utilisateur) ne peuvent être modifiés que par un autre utilisateur possédant un niveau de sécurité équivalent ou supérieur au vôtre.

Les paramètres que vous pouvez modifier vous-même peuvent comprendre votre prénom et votre nom, vos initiales, votre adresse courriel et votre mot de passe à la base de données. Si vous n'avez pas l'autorisation de modifier certains ou tous ces paramètres, vous aurez accès à un formulaire pour demander à votre administrateur local de faire ces changements pour vous.

Modifier votre mot de passe

Veuillez lire attentivement les informations concernant les mots de passe. Pour modifier votre mot de passe, vous devez d'abord saisir votre mot de passe existant, puis saisir le nouveau mot de passe désiré dans les deux champs disponibles. Le fait de cliquer sur le lien « Soumettre ».validera vos changements. Vous n'êtes pas obligé de saisir vos informations de mot de passe, sauf si vous voulez modifier votre mot de passe.

Utilisez un mot de passe sécurisé, mais pas un mot de passe que vous utilisez pour d'autres choses. Vos informations de compte et de mot de passe ne sont pas cryptées, donc n'utilisez jamais un mot de passe que vous utilisez déjà pour des comptes importants. Quelques recommandations pour maintenir un mot de passe sécurisé :

  • Utilisez au moins 8 caractères
  • Utilisez à la fois les lettres MAJUSCULES et minuscules
  • Utilisez au moins un chiffre ou symbole
  • Mettez à jour votre mot de passe régulièrement (tous les six mois, par exemple)
  • Ne donnez jamais votre mot de passe à d'autres personnes

Mémorisez votre mot de passe.

Si vous oubliez votre mot de passe, vous devrez le faire réinitialiser par un super-utilisateur ou un autre utilisateur de la base de données ayant des privilèges de gestion, en fonction de votre niveau de sécurité. Vérifiez que vous savez qui contacter si vous oubliez vos informations de mot de passe. À noter que les représentants de CIOC ne peuvent pas récupérer votre mot de passe pour vous et ne modifieront aucun compte, sauf en cas d'urgence à la demande directe du contact officiel de la membriété.

Remarque : ne modifiez pas le mot de passe de votre compte si vous savez que d'autres utilisent les mêmes informations de connexion pour accéder à la base de données, sauf si vous leur avez communiqué le nouveau mot de passe auparavant. Chaque personne devrait posséder son propre compte dans la base de données, autant que possible.
