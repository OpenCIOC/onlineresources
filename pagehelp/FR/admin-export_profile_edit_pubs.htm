La page Gérer les codes de publications pour l'exportation vous permet de sélectionner quels codes de distribution à inclure dans le champs Publication des dossiers à exporter. Pour ajouter une nouvelle publication à la liste, sélectionnez-là dans la liste déroulante en bas du formulaire, puis cliquez sur le bouton « Ajouter ». Il est possible qu'aucun code de publication ne soit disponible sur cette page, si aucun n'existe dans votre application. Après avoir ajouté le code de publication, vous pouvez le retirer en cliquant sur le bouton « Supprimer » à côté du code de publication.

Pour chaque publication sélectionnée, vous avez le choix d'y inclure toute description ou en-tête général associé à la publication. Cochez ou décochez les cases indiquant si vous souhaitez inclure les descriptions en en-têtes généraux d'une publication, puis cliquez sur le bouton « Mettre à jour » pour sauvegarder vos changements. Vous devez sauvegarder les changements pour chaque publication à la fois.

Remarque importante : le fait de sélectionner des codes de publication particuliers pour vos profils n'affecte pas le choix des dossiers à exporter, seulement les contenus du champ Publication que vous envoyez. Les dossiers à envoyer sont sélectionnés au moment de l'exportation.

Les changements soumis sont mis en œuvre immédiatement. Il n'est pas nécessaire de soumettre le formulaire principal de modification du profil d'exportation pour sauvegarder vos modifications.
