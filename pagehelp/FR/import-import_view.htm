La page Visualiser les données d'importation vous permet de voir le contenu de chaque champ d'un dossier donné, tel qu'il apparaît dans le fichier d'exportation XML qui vient d'être chargé. En haut de la page est indiqué les langues du dossier et le propriétaire du dossier référencé dans le fichier d'exportation.

Certains champs, comme ceux comportant des éléments de listes de contrôle, seront divisés par composantes individuelles. Si le fichier contient plus d'une langue, les données dans chaque langue seront affichées ensemble. Faites particulièrement attention aux champs vides, car cela signifie que toute donnée d'une version existante présente dans ce champ sera supprimée. Si vous importez un dossier multilingue, faites attention si des données manquent dans une langue (l'étiquette de données mentionne la langue entre parenthèses ou l'indiquera de manière explicite, comme « Langue : F » pour français). S'il existe des données dans une langue, mais pas l'autre, le champ de la langue sans données sera supprimé.

À noter que les données des champs à liste de contrôle apparaîtront comme 0 pour décoché ou 1 pour coché. Si un champ de liste de contrôle peut aussi être vide, une valeur vide indique inconnu.

Les dates et heures sont formatées au format de date ISO plutôt qu'au format habituellement utilisé par CIOC. Une fois les données importées, elles s'afficheront au format CIOC habituel.
