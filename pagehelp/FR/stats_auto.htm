Les « rapports automatiques » sont des rapports générés automatiquement en fonction de critères prédéterminés. Ces rapports affichent les consultations ou modifications de dossiers au cours des trois derniers mois : ils comprennent toutes les statistiques disponibles du mois en cours, ainsi que les statistiques des deux mois précédents.

Tous les rapports automatiques sur les consultations de dossiers comprennent toutes les valeurs, y compris celles générés par les robots.

Rapport sur la gestion des données

Ce rapport liste combien de dossiers ont une date de modification quelconque comprise dans le mois en cours ou les deux mois précédents. Ce rapport n'est pas basé sur le système complet d'historique des changements, seulement la valeur actuelle de chaque champ de date donné, si bien que son exactitude diminue avec le temps - surtout le renseignement « dernière modification ».

Dans une base de données multimembre, le rapport détaille encore plus les statistiques sur les dossiers, par dossiers appartenant à votre membriété (locaux) contre ceux partagés avec vous par d'autres membres.

Dossiers consultés par propriétaire de dossiers

Ce rapport détaille les consultations de dossiers par mois, par agence propriétaire de dossiers et par utilisateurs du public ou connectés.

Dossiers consultés par vue

Ce rapport détaille les consultations de dossiers par mois, par vue et par utilisateurs du public ou connectés.

Nombre total d'IP uniques par propriétaire de dossiers

Ce rapport détaille les consultations de dossiers par mois, par agence propriétaire de dossiers et par adresse IP unique.

Nombre total d'IP uniques par vue

Ce rapport détaille les consultations de dossiers par mois, par vue et par adresse IP unique.

Nombre total d'IP uniques

Ce rapport indique le nombre total d'adresses IP uniques sur un mois donné, sans regrouper les informations en fonction d'autres critères comme la vue ou le propriétaire de dossiers. C'est un rapport à utiliser pour obtenir une idée générale du nombre d'utilisateurs individuels sur des périodes de temps données.

À propos des statistiques d'IP

Lorsque vous consultez des statistiques concernant les IP « uniques », veuillez prendre en compte le fait que les accès par une même adresse IP peuvent parfois représenter plusieurs utilisateurs (si tout le trafic d'un organisme paraît provenir de la même adresse IP) et que certains utilisateurs peuvent accéder à la base de données depuis plusieurs adresses IP (s'ils ont plusieurs emplacements, ordinateurs ou une adresse IP allouée dynamiquement). Cependant, cela reste une indication raisonnable du nombre d'utilisateurs individuels.
