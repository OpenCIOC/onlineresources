Les profils d'exportation vous permettent de configurer des profils en personnalisant les formats d'exportation, aux fins d'utilisation des données dans d'autres systèmes. Cela comprend le Format de partage CIOC qui vous permet de partager des dossiers avec d'autres bases de données CIOC. Les profils d'exportation vous permettent de définir quels champs, codes de distribution et codes de publication vous souhaitez partager, et de préciser dans quelles vues le profil d'exportation sera accessible.

À noter que les profils Excel sont gérés séparément. Les profils Excel offrent plus de choix sur ce qu'il est possible d'exporter, mais ils ne procurent aucune structure à l'information et s'avèrent donc moins utiles comme moyen d'utiliser vos données dans d'autres systèmes (et ils ne peuvent pas être utilisés pour échanger des données entre bases de données CIOC).

Modifier un profil d'exportation existant

Pour consulter ou modifier les paramètres d'un profil d'exportation existant, sélectionnez le profil dans la liste, puis cliquez sur « Voir / Modifier un profil ».

Ajouter un nouveau profil de partage

Pour créer un nouveau profil d'exportation, vous devez donner un nom unique au profil. S'il existe déjà des profils, vous pouvez copier un profil en le sélectionnant dans la liste « Copier le profil ». Si vous ne sélectionnez pas de profil à copier, votre nouveau profil contiendra les paramètres par défaut, sans de champ ajouté. Cliquez sur le bouton « Ajouter un profil » pour créer le nouveau profil ; vous serez dirigé vers le formulaire de modification de ce nouveau profil.
