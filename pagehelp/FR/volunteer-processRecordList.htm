Cette page est une page intermédiaire ou passerelle qui contient plusieurs options de sélection et de manipulation des dossiers, en fonction des choix effectués dans la liste déroulante sur la page précédente. Les choix initiaux qui peuvent apparaître sur cette page sont soulignés plus bas et nommés en fonction des sélections disponibles dans la liste déroulante sur la page des résultats de recherche. Ce qui apparaît sur votre page dépend de l'action que vous aurez sélectionnée dans le menu déroulant.

Nouveau Rappel

Le système de rappels vise à aider les utilisateurs à créer des listes de tâches pour eux-mêmes ou d'autres utilisateurs, ainsi qu'à offrir un moyen de communication avec d'autres utilisateurs de CIOC. Les rappels peuvent porter sur des tâches générales ou ils peuvent être associés à un ou plusieurs dossiers. Vous pouvez travailler avec les rappels en cliquant sur le lien Rappels dans la barre supérieure ou (dans le cas de rappels associés à des dossiers précis) par la page de détails du dossier. Vous pouvez également créer un rappel pour un groupe de dossiers par vos résultats de recherche. Cliquez sur le bouton « Prochain » pour débuter la création d'un nouveau rappel sur les dossiers sélectionnés.

Imprimer la liste

Si au moins un profil d'impression est accessible dans votre vue actuelle, vous pourrez choisir un profil spécifique dans la liste déroulante de cette page et commencer la préparation d'un rapport à imprimer personnalisé. On peut ajouter ou modifier des profils d'impression par Configuration > Profils d'impression.

Générer un rapport statistique

Le formulaire pour générer des rapports statistiques sur les dossiers sélectionnés est presque identique à l'écran Recherche sur les statistiques. Consultez les fichiers d'aide disponibles sur les pages de recherche sur les statistiques pour plus de renseignements.

Demander la mise à jour par courriel

Cette fonction vous permet de préparer et d'envoyer des demandes de mise à jour par courriel en nombre à un groupe de dossiers sélectionnés. Cliquez sur le bouton « Prochain » pour débuter la préparation d'un courriel de mise à jour à envoyer aux dossiers sélectionnés.

Rendre les dossiers publics / internes

Cette fonction vous permet de marquer un groupe de dossiers comme Public ou Interne. Sélectionnez l'option désirée dans la liste déroulante, puis cliquez sur « Prochain ».

Supprimer / Restaurer les dossiers

Cette fonction vous permet d'établir ou de supprimer la date de suppression d'un groupe de dossiers. Les options disponibles sont « Établir la date de suppression » (à choisir pour supprimer un groupe de dossiers ou modifier la Date de suppression de dossiers déjà supprimés) ou « Restaurer les dossiers » (à choisir pour retirer le statut de suppression d'un groupe de dossiers).

[ Haut ]

Changer le propriétaire du dossier

Cette fonction vous permet de modifier l'agence désignée comme propriétaire du dossier des dossiers sélectionnés. Sélectionnez l'agence désirée dans la liste déroulante, puis cliquez sur « Prochain ». La liste des agences disponibles ici est établie en fonction des agences marquées comme détenant des dossiers dans le module actuel (Information communautaire ou Bénévolat), dans la section Agences de la configuration.

Ajouter / Supprimer un domaine d'intérêt

Cela vous permettra d'ajouter ou de supprimer un domaine d'intérêt d'un ensemble de dossiers. Si vous regroupez vos domaines d'intérêt en catégories plus larges, vous devez d'abord sélectionner un domaine d'intérêt général dans le menu déroulant et cliquer sur « Soumettre ». Sélectionnez soit « Ajouter », soit « Supprimer » dans le menu déroulant, cochez les cases à côté des domaines d'intérêt spécifiques que vous souhaitez ajouter ou supprimer, puis cliquez sur « Soumettre ». Si vous choisissez l'option « Supprimer toutes les valeurs », vous devrez cocher une case pour confirmer que vous souhaitez bien retirer chaque domaine d'intérêt de tous les dossiers sélectionnés.

Ajouter au profil de partage

Lorsqu'on travaille sur une base de données multimembre, cet outil est utilisé pour ajouter de nouveaux dossiers à un profil de partage existant pour rendre les dossiers accessibles aux autres membres de votre base de données.
