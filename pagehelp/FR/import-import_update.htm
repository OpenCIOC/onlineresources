La page Ajouter / Modifier des dossiers est utilisée pour sélectionner les options d'importation. Vous avez l'occasion de consulter les données des dossiers à importer avant de lancer l'importation. Une fois que les paramètres désirés ont été sélectionnés, cliquez sur le bouton Importer les données du dossier pour débuter le processus d'importation.

L'option Sélectionner les propriétaires du dossier, disponible lorsque le fichier d'importation contient plusieurs propriétaires de dossiers, vous permet de sélectionner seulement les propriétaires de dossiers spécifiques du fichier à importer. Si vous n'en sélectionnez aucun, les données de tous les propriétaires de dossiers dans le fichier seront importées. Remarque importante : le Propriétaire de dossiers peut être différent que ce qui impliqué par le Numéro de dossier.

Le paramètre En cas de désaccord sur la propriété des dossiers vous permet de gérer une situation où le propriétaire d'un dossier existant dans la base de données ne correspond pas au propriétaire indiqué dans le fichier d'exportation. Vous pouvez soit conserver vos renseignements existants sur le propriétaire, soit mettre à jour l'appartenance pour correspondre au propriétaire dans le fichier, ou soit ne pas importer le dossier en question.

Si les dossiers importés contiennent des renseignements confidentiels (sont associés à un profil de confidentialité) la Correspondance avec le profil de confidentialité vous permet d'associer les profils de confidentialité d'un ensemble de données entrant avec des profils de confidentialités existants dans votre base de données. Le système tentera d'établir ces correspondances automatiquement. Si vous ne pouvez pas établir de correspondance, vous devez préciser si les données confidentielles seront ignorées ou importées quoi qu'il en soit - en cas de doute, il vaut mieux ne pas importer de renseignements confidentiels, à moins de pouvoir assurer qu'ils le resteront. Si le dossier existant a un profil de confidentialité différent du dossier entrant, vous devez préciser comment résoudre le conflit (conserver l'ancien profil, accepter le nouveau profil ou ne pas importer le dossier).

Le paramètres Nombre de dossiers à importer vous permet d'importer des dossiers en lots plus petits, ce qui est recommandé si vous avez un grand nombre de dossiers à importer, afin d'empêcher de longs délais d'attente aux autres personnes utilisant le serveur.

Importer les renseignements sur la base de données source, si sélectionné, renseigne le champ SOURCE_DB dans la base de données avec la valeur fournie dans le fichier d'exportation. Cela peut vous aider à créditer correctement vos partenaires qui partagent leurs données avec vous, ainsi que vous rappeler l'origine du dossier. Vous pouvez sélectionner où afficher le champ SOURCE_DB (base de données source) dans votre vue, en l'ajouter ou le retirant des champs de détails sur les dossiers choisis dans les paramètres de votre vue.
