Les pages Ajouter/ Modifier un modèle vous permettent de modifier un modèle créé auparavant ou de créer un nouveau modèle de conception. Les modèles de conception rassemblent plusieurs éléments de conception (mise en page, couleurs, styles) pour vous aider à modifier l'apparence de n'importe quelle vue dans votre base de données.

Si vous modifiez ou copiez un modèle existant, le contenu du modèle existant sera inséré dans les champs du formulaire ; autrement, vous aurez un formulaire vide avec lequel créer un nouveau modèle.

Les éléments primaires à inclure dans la conception d'un modèle depuis cette page :

  • les liens externes pour introduire des logos, icônes, feuilles de style (CSS) et fichiers JavaScript
  • avis de droit d'auteur
  • sélection de polices, couleurs, arrières-plan, etc.
  • sélection d'une mise en page avec en-tête et pied de page et éléments de menu facultatifs pour chacun d'eux
  • sélection d'une mise en page pour la recherche (page principale) et éléments de menu facultatifs

La seconde moitié de la configuration du modèle - tout ce qui suit la sélection de mise en page - traite des différents éléments de l'interface utilisateur jQuery UI. jQuery est une bibliothèque JavaScript largement utilisée dans le logiciel CIOC, et jQuery UI est une bibliothèque de styles complémentaire. Pour des renseignements supplémentaires sur jQuery UI, consultez http://jqueryui.com/ (en anglais).

On peut trouver plus d'informations sur la création et la modification de modèles dans les références (y compris les vidéos de formation) du site de la communauté CIOC, sur http://community.cioc.ca/ (en anglais). Toute question ou difficulté rencontrée lors de la création d'un modèle peut être postée sur le Babillard de la communauté CIOC (en anglais), où d'autres utilisateurs ou le personnel d'assistance de CIOC vous aideront à répondre à des questions ou difficultés spécifiques. Il n'est pas possible de traiter des feuilles de style, en-têtes et pieds de page HTML personnalisés, JavaScript, etc. dans ce document d'aide ; les questions doivent être postées sur le babillard cité plus haut.
