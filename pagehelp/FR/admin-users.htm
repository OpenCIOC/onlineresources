La page Gestion des utilisateurs est accessible aux utilisateurs ayant des niveaux de sécurité leur permettant de gérer des utilisateurs.

REMARQUES IMPORTANTES

  • Seuls les utilisateurs désignés comme super-utilisateurs peuvent modifier les paramètres de compte pour un autre super-utilisateur.
  • Vous ne pouvez modifier que les informations d'un type d'utilisateur pour les modules dans lesquels vous détenez des privilèges de gestion des utilisateurs.
  • Les utilisateurs qui détiennent des privilèges de gestion des utilisateurs, mais qui ne sont pas super-utilisateurs, peuvent modifier les comptes des utilisateurs équivalents ou avec des niveaux de sécurité inférieurs. Ils ne peuvent pas modifier les comptes de super-utilisateurs.
  • Aucun utilisateur, y compris un super-utilisateur, ne peut modifier les autorisations et niveaux de sécurité de son propre compte. Cela réduit la possibilité que le super-utilisateur unique d'une base de données ne se retire ses privilèges d'administration de la base de données par inadvertance. Si seul un super-utilisateur unique existe dans une base de données, il faut d'abord créer un nouveau compte de super-utilisateur avant de pouvoir modifier le premier compte de super-utilisateur.
  • À noter qu'il n'est pas possible de supprimer des utilisateurs existants, seulement de les désactiver.

En utilisant la page Gestion des utilisateurs, vous pouvez :

Créer un nouvel utilisateur

Pour créer un nouvel utilisateur, cliquez sur le lien « Créer un nouvel utilisateur ».

Modifier un compte d'utilisateur existant

Localisez les comptes d'utilisateurs existants en utilisant le formulaire de recherche fourni. Pour retrouver une liste de tous les utilisateurs, sélectionnez si vous souhaitez les utilisateurs actifs, inactifs ou les deux, et soumettez le formulaire de recherche sans autre critère. Si vous avez l'autorisation de mettre à jour l'utilisateur, vous pourrez voir un lien « Mettre à jour » juste à côté du compte dans les résultats de recherche.

Gérer votre propre compte

Tel que mentionné dans la section « Remarques importantes » au-dessus, vous ne pouvez pas modifier le niveau de sécurité, ni les autorisations sur la base de données, de votre propre compte. Seul un autre utilisateur avec un niveau de sécurité équivalent ou supérieur peut modifier votre compte. Cependant, en utilisant le lien « Mon compte » sur cette page, vous pouvez modifier votre nom, vos initiales, votre adresse courriel et votre mot de passe.

Consulter l'historique des modifications du compte

L'outil d'historique des modifications du compte vous permet de consulter les changements passés apportés aux comptes d'utilisateurs.
