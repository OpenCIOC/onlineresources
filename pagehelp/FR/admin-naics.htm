Remarque importante : le Système de Classification des Industries de l'Amérique du Nord (SCIAN) est un système de classification normalisé administré par Industrie Canada ; de manière générale, il n'est pas recommandé d'ajouter ni de modifier les valeurs du système. La version actuellement fournie par CIOC est le SCIAN 2002.

La page Gérer les codes SCIAN vous permet de saisir un code SCIAN à consulter ou à modifier. Pour ajouter un code au système, ne saisissez aucun code, puis cliquez sur « Visualiser / Modifier le code ». Pour modifier un code existant, vous devez connaître le code numérique de 2 à 6 chiffres que vous souhaitez modifier ; si vous ne le connaissez le numéro du code, cliquez sur « Localisateur de codes SCIAN » afin d'ouvrir une fenêtre contextuelle vous permettant de rechercher ou de parcourir le SCIAN.
