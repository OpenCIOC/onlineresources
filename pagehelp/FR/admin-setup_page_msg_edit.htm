Un Message de page vous permet d'insérer certaines informations à afficher sous l'en-tête d'une page, mais avant le reste du contenu. Un message de page apparaît uniquement sur les pages spécifiques et les vues que vous sélectionnez, et uniquement dans la langue spécifiée. Vous pouvez également choisir si le message sera visible lors de l'affichage d'une page en Mode d'impression.

Le HTML est autorisé et doit être utilisé si vous souhaitez ajouter du formatage, comme des en-têtes, des paragraphes, etc. Les messages de page doivent rester brefs et sont appropriés pour des instructions ou des annonces spéciales, lorsque cette information doit être présentée sur certaines pages mais pas d'autres. Si un message doit s'afficher sur toutes les pages, il sera plus approprié de l'ajouter par la configuration de modèle ou de vue.
