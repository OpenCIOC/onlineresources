En fonction de votre niveau d'accès (c'.-à-d. si vous êtes super-utilisateur dans un module ou plusieurs), certaines options de cette page peuvent vous être inaccessibles.

Options générales de configuration

Zone pour modifier les options de configuration qui s'appliquent à toute le base de données, y compris : configurer les modèles de la section "Gestion", permettre ou personnaliser certaines fonctionnalités du logiciel comme les outils hors-ligne, configurer les vue (publiques) par défaut, etc.

Agences

Vous permet de configurer les renseignements relatifs au propriétaire des dossiers pour les agences gérant des dossiers ou avec des utilisateurs de la base de données.

Gestion des utilisateurs

La section Gestion des utilisateurs vous permet de rechercher, de modifier et de créer de nouveaux utilisateurs.

Modèles de conception

Les modèles de conception contrôlent le style / l'apparence de votre base de données, y compris les couleurs, logos, feuilles de style, en-têtes et pieds de page, etc.

Mises en page de modèles

Les mises en page de modèles sont utilisées au sein d'un modèle de conception pour fournir un formatage personnalisé des en-têtes, des pieds de page et des pages du menu principal et de recherche.

Politiques d'inclusion des dossiers

Vous permet de modifier les politiques d'inclusion qui sont fournies lorsque quelqu'un suggère un nouveau dossier pour la base de données.

Types de médias sociaux

La configuration des Types de média sociaux vous permet de modifier la liste des types de médias sociaux, y compris de décider quels types apparaîtront sur les formulaires de mise à jour et de rétroaction. Remarque : comme les icônes de médias sociaux sont requises et doivent être installées dans les répertoires appropriés du logiciel, vous devez communiquer avec les développeurs du logiciel sur l'ajout de nouvelles icônes si vous souhaitez ajouter un type de média social supplémentaire.

Mise en correspondance du nom de domaine

Vous permet de réviser la liste des noms de domaine associés à votre base de données et de les mettre en correspondance avec des vues spécifiques. À noter que les mises en correspondance des noms de domaine s'appliquent au public et aux utilisateurs non connectés. Les utilisateurs connectés sont libres d'accéder à la base de données en utilisant n'importe quel nom de domaine, puisque les vues auxquelles ils accèdent sont établies en fonction des paramètres de sécurité de leur compte, et non du nom de domaine qu'ils utilisent.

Systèmes cartographiques

La configuration des Systèmes cartographiques vous permet d'intégrer plusieurs services de cartographie (p. ex., Google Maps, MapQuest, etc.) au logiciel.

Catégories de cartographie

Les Catégories de cartographie vous permettent d'assigner des étiquettes personnalisées à chaque couleur de bouton de carte lorsque vous utilisez les outils Google Maps intégrés (géocartographie).

Vues

Les Vues contrôlent ce qu'un utilisateur peut voir, en fonction de son compte (ou, s'il n'a pas de compte, le « public »). Les vues contrôlent quels dossiers sont accessibles, quels champs peuvent être affichés ou modifiés, si la colonne d'alerte est visible, quels messages de bas de page ou de menu latéral apparaissent, etc.

Types d'utilisateurs

Les Types d'utilisateurs établissent quelles actions une personne peut ou ne peut pas effectuer dans la base de données, comme ajouter des dossiers, mettre à jour des dossiers, si des alertes sont visibles pour les rétroactions ou les commentaires, si l'utilisateur est un super-utilisateur et d'autres options se rapportant aux fonctionnalités.

Affichage des champs

Vous permet de modifier l'étiquette des champs et l'ordre dans lequel ces champs apparaissent dans les dossiers de la base de données.

Champs masqués

Vous permet de masquer les champs que vous n'utilisez pas. Les champs masqués ne sont pas offerts comme choix lors de la sélection de nouveaux champs dans les formulaires de configuration, comme les vues, les profils d'impression, les profils d'exportation, etc. Cependant, le fait de masquer un champ ne le supprimera pas d'une vue à laquelle il a déjà été ajouté.

Valeurs Oui / Non d'affichage des champs

Pour les champs avec boutons radio, précise les valeurs à afficher dans le dossier lorsque True (Oui) ou False (Non) sont sélectionnés.

Configuration du champ Extra

La Configuration des champs supplémentaires vous permet de créer de nouveaux champs supplémentaires de types variés et, le cas échéant, de définir le nombre de caractères pouvant être saisis dans les champs de texte (longueur maximale). Remarque : dans le logiciel Bénévolat, seule la longueur de deux champs supplémentaires prédéfinis peut être modifiée.

Profils de sortie Excel

Les Profils de sortie Excel vous permettent de créer un affichage personnalisé pouvant être lu sous Microsoft Excel ; de sélectionner les champs à afficher, l'ordre dans lequel les dossiers sont affichés et quelles vues peuvent accéder au profil.

Profils d'exportation

Les Profils d'exportation vous permettent de configurer plusieurs profils pour exporter des dossiers aux fins d'utilisation par des systèmes externes, y compris d'autres bases de données CIOC utilisant le format de partage CIOC (module Information communautaire uniquement). Les profils d'exportation vous permettent d'établir quels champs, codes de publication et de distribution vous souhaitez partager.

Profils d'impression

Les profils d'impression vous permettent de créer des modèles personnalisés pour l'impression de listes de dossiers.

Profils de confidentialité

Les Profils de confidentialité vous permettent de configurer plusieurs profils pour masquer les données privées contenues dans les dossiers individuels au public ou aux utilisateurs non-autorisés, même s'ils pourraient voir ce champ dans leur vue autrement. Il est extrêmement important d'utiliser les profils de confidentialité avec soin et parcimonie, particulièrement pour les dossiers que vous partagez avec des partenaires extérieurs. N'utilisez pas de profils de confidentialité à moins de comprendre pleinement les défis impliqués.

Texte de mise à jour par courriel

Vous permet de créer des valeurs par défaut pour les sections personnalisables d'une demande de mise à jour par courriel (modules Information communautaire ou Bénévolat) ou d'une demande de mise à jour des occasions (module Bénévolat uniquement).

Groupes d'âge pour la recherche

Créé des classes de groupe d'âge aux fins d'utilisation sur les formulaires de recherche simple et de garderie (module Information communautaire uniquement).

Statuts des activités

Personnalise les valeurs des statuts prédéfinis aux fins d'utilisation dans le champ ACTIVITY_INFO.

Profils de partage

Dans une base de données multimembre, les Profils de partage vous permettent de créer un contrat ou plus pour partager des données avec d'autres membres CIOC dans votre base de données.

Outils hors-ligne

Si les Outils hors-ligne sont activés dans la base de données, cette section permet de gérer quels types d'utilisateur ont accès aux ordinateurs / serveurs autorisés ayant installé les outils hors-ligne CIOC.

Configuration des places disponibles

Personnalise les valeurs des statuts prédéfinis aux fins d'utilisation dans le champ VACANCY_INFO.

Gestion des listes de contrôle

Il est important de lire attentivement les renseignements contenus dans les différentes pages « Modifier les valeurs de la liste de contrôle / déroulante ». À deux exception près (Type de rue et Type de case postale), les changements apportés aux valeurs des listes de contrôle et menus déroulants sur ces pages affectent non seulement les valeurs disponibles dans les menus déroulants et les listes de contrôle / boutons radio, mais entraîneront des modifications sur tous les dossiers de la base de données qui utilisent ces valeurs.

Cliquez sur le type de liste de contrôle / menu déroulant de votre choix sur cette page pour passer vers une page de « modification », où vous pouvez modifier, supprimer et ajouter des nouvelles valeurs aux différents menus déroulants et listes de contrôle qui figurent dans la base de données.

Gestion des systèmes de classification

Les systèmes de classification sont utilisés pour catégoriser et chercher des dossiers. De nombreux types de systèmes de classification sont utilisés à travers plusieurs bases de données et peuvent provenir de sources (officielles) externes. À noter que certains types de systèmes de classification sont également accessibles depuis le menu principal (Publications, Taxonomie) car tous ou la plupart des outils de gestion sont accessibles aux utilisateurs qui ne sont pas super-utilisateurs. Ces systèmes sont également liés ici aux fins de commodité et de cohérence.

Titres de pages

Vous permet de remplacer les titres par défaut dans certaines pages du logiciel. Le remplacement est actuellement disponible pour seulement un petit nombre de pages.

Messages de page

Un Messages de page vous permet d'insérer certaines informations dans une page, comme des instructions ou des annonces spéciales. Cette méthode s'avère utile lorsque cette information doit être présentée sur certaines pages mais pas d'autres ; l'information devant apparaître sur toutes les pages doit être insérée dans le modèle de conception. Vous pouvez également sélectionner les vues dans lesquelles vous souhaitez afficher les messages.

Aide sur les champs

Vous permet de modifier l'aide sur les champs qui apparaît dans les formulaires de mise à jour ou de rétroaction de dossiers. Il s'agit des liens apparaissant à côté des champs de saisie de données et fournissant des renseignements sur les règles ou directives de saisie des données pour ce champ.

Conseils de recherche

Vous permet de modifier le texte de la page de Conseils de recherche accessible depuis le lien sur la page de menu principal / recherche simple.
