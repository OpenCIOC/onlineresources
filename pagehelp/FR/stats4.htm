Remarque : la page d'aide sur le formulaire de recherche principal sur les statistiques peut s'avérer utile pour obtenir un aperçu général des différents types de statistiques conservées dans votre base de données CIOC et savoir comment les utiliser de manière adéquate.

Cette page affiche une tableau montrant l'utilisation totale des dossiers de la base de données, triée par propriétaire de dossiers, et le nombre de dossiers par propriétaire de dossiers. L'utilisation par le public et l'utilisation totale sont toutes deux affichées (indiquant à la fois les utilisateurs connectés et non connectés). Dans les bases de données ayant plus d'un propriétaire de dossiers, cela permet de comparer rapidement la manière dont les dossiers sont utilisés à travers la base de données, en fonction des propriétaires de dossiers assignés.

À noter que ces valeurs sont basées sur les statistiques sur les dossiers consultés internes au logiciel ; les statistiques sur les dossiers consultés ne sont pas conservées lorsque vous passez en mode impression ou changez de vue temporairement, pour ne pas gonfler les statistiques si quelqu'un consulte plusieurs versions d'un même dossier pour les mêmes raisons.

En haut de la page figurent des liens vers plusieurs autres outils statistiques : Page d'accueil des statistiques, Utilisation totale des dossiers, 50 premiers dossiers, Rapports automatiques et Supprimer les statistiques anciennes.
