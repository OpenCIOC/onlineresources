Your Search Results

Depending on which version of the Referral list page you are working with, this page contains one of:

  • Referrals that were created or modified in the Past X number of days, listed in reverse order of the date they were Last Modified
  • OR Referrals for the given Opportunity, listed in reverse order of their original Referral Date
  • OR Referrals that are Flagged for Follow-Up, listed in reverse order of the date they were Last Modified
  • OR Referrals that match the criteria provided on the Custom Referral Search form on the Volunteer Referrals page, listed in reverse order of the date they were Last Modified

If you are a Super User, you will see all Referrals; all other users see only Referrals from Opportunities owned by their own Agency. If you want to make modifications to the Referral or see the full details, click the Update link on the right.

Changing the Sort Order

If you have JavaScript enabled in your browser, you will be able to change the sort order of your search results. The current current sort column and direction are indicated by arrows next to the column name. To change the sort column, click the header of the column you would like to sort on. To change the direction of the sort, click the column again. You can sort on any column except the Action column; even though there isn't a column name on the follow-up or placement columns, you can sort on those columns.

Additional Actions

If you have JavaScript enable in your browser, you will be able to change your display options to include extra tools. The list of tools available to you is next to the word Show above your results.

Follow-up Flag Editing

Enabling Follow-Up Flag Editing will display check boxes in the Follow Up Flag column (far left) and a set of buttons below the table: Check All, Un-Check All, Add Follow-Up Flag and Remove Follow-Up Flag. The Add Follow-Up Flag and Remove Follow-Up Flag buttons cause the currently checked records to have their follow-up flag set as indicated. Note: after adding or removing a follow-up flag from a set of records, your search is refreshed. If you are viewing a search which had follow-up flag as a criteria, you may have fewer records in your results after submitting your changes.

Email Follow-Up

You can send a follow-up email request about the selected referral requests to either the posting organization/agency or the potential volunteer using this tool. Select the referral request(s) you wish to send an email about using the provided checkboxes, or select them all using the "Check All" button at the bottom of the page. After submitting, you will have a chance to review the list of selected referrals and compose the message being sent. For more information, view the page help for the Customize Referral Follow-Up Email page.

Note that the ability to send a follow-up email is dependent on the volunteer or posting organization/agency providing a contact email address.
