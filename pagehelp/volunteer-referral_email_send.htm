For more information on Referral follow-up emails, view the help for the Customize Referral Follow-Up Email page.

On this page, the software will attempt to send the Referral follow-up email messages(s) you have created. The software will report any known issues with sending the message(s) and print a report of the Email addresses referral summaries that were sent. For example:

Email sent to: myaddy1@mysite.com (A. Volunteer)
Referral Date: 21 Dec 2006
Position: Number One Position
Organization: The Organization for helping people.

Email sent to: myaddy2@anothersite.ca (B. Volunteer)
Referral Date: 22 Dec 2006
Position: Number Two Position
Organization: The Organization for helping people.

It is recommended that you print this page, as it provides you with the only record of emails successfully sent. Note: it is still possible for an Email delivery to fail even though the system was able to create the Email message; you should receive an Email "bounce-back" message in this situation.
