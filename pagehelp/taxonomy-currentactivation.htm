The Taxonomy Activation and Usage Report displays a list of currently active Terms and the number of times each is used in the database. Note that the usage count may include multiple uses by the same record, e.g. within multiple linked Terms. In the Action column, you can click on the edit icon to go to the edit page for that Term, or the search icon to bring up a list of records using that Term in your current View.
