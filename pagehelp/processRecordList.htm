New Reminder

The Reminders system is intended to assist users in creating task lists for themselves and other users, as well as provide a means of communicating information to other CIOC users. Reminders can be for general tasks or communications, or they may be tied to one or more records. You can work with reminders by clicking the Reminders link in your site bar, or (in the case of Reminders tied to specific records), through the Record Details page. You can also create a reminder for a group of records through your search results. Click the "Next" button to begin creating a new reminder for the selected records.

Add/Remove Distribution Code

This allows you to either add or remove a Distribution code from the selected set of records. You must select either "Add" or "Delete" from the first drop-down menu and then select from the available Distribution codes in the second drop-down menu. A Super User can add new codes to the list by going to Checklists > Distribution from the Main Menu. If you choose the "Delete All" option, you will be asked to check a box to confirm that you wish to remove every Distribution code from the selected records.

Add/Remove Heading

This function will allow you to remove General Headings that are attached/associated with a Publication code that is attached to records. This is a two step process. First you must select the Publication code associated with the General Heading you wish to add or remove and click "Submit" (Note: only Publications with associated General Headings will be in the list). Next select either "Add" or "Delete" from the first drop-down and the General Heading you want to work with from the second drop-down menu and click "Submit". If you choose the "Delete All" option, you will be asked to check a box to confirm that you wish to remove every General Heading in the current publication from the selected records.

In addition to the general tool to add and remove headings, you may also synchronize your indexing for the current publication to the indexing that was done on another publication. This will only apply to cases where the same headings exist in both publications. It is important to choose one Publication as the "master", in which you will maintain the indexing for the record, and then copy it to other publications. You will select the master publication from the "Synchronize Headings to the classifications found in" drop down.

Add/Remove NAICS Code

This function allows you to add or remove a valid North American Industry Classification System (NAICS) code from a group of records. Select either "Add" or "Delete" from the drop-down menu, and then enter a valid 2-6 digit NAICS code to add/remove. If you do not know the code you want to work with, use the "Find a Code" link to open a pop-up window with a form to search or browse for a NAICS code. If you choose the "Delete All" option, you will be asked to check a box to confirm that you wish to remove every NAICS code from the selected records.

Add/Remove Publication Code

This allows you to either add or remove a Publication Code from the selected set of records. You must select either "Add" or "Delete" from the first drop-down menu and then select from the available Publication codes in the second drop-down menu. Note the alert text on the page as deleting a Publication code from records has other potential implications beyond simply removing the code from the record(s). A user with Publication management privileges can add new codes to the list by going to Publication from the Main Menu. If you choose the "Delete All" option, you will be asked to check a box to confirm that you wish to remove every Publication code from the selected records.

Add/Remove Taxonomy Term

This allows you to either add or remove a single Taxonomy Term (Service Category) from the selected set of records. You must select either "Add" or "Delete" from the first drop-down menu and then enter a valid Taxonomy Code for an Active Term in the box provided. If you do not know the Code of the Term you want to Add/Remove, use the link to the Service Category Finder tool provided.

If you are Removing an existing Term from the records, some additional options apply. Because some Taxonomy Terms may occur within a list of linked Terms, you must choose what to do when the Term is found within a set of linked Terms. By default, the Term will only be removed if it is not linked together with any other Terms.

If you choose the "Delete All" option, you will be asked to check a box to confirm that you wish to remove all Taxonomy indexing from the selected records.

Update Geocoding

There is no need to use this tool on a regular basis for records that have already been geocoded. If you have included the GEOCODE field on your update form, your mapping will be automatically updated when you edit the record.

This function allows you to update the map location of a list of records. By default, this tool will use the existing mapping setting and confirm that the map location is correct according to current address or intersection information (as applicable). You may optionally change or set the mapping type (to site location, intersection, or do not map) and/or set the mapping category of records that are updated. To see the category name (if available), hold your mouse over the map pin (hold mouse over the pin to display "Mapping Category" ).
