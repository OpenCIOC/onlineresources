This page permits the user to create a new code for a Publication (a grouping of records) or edit an existing Publication code. This page is not used to edit information about the records that are included in a Publication - only information about the Publication itself.

A Publication is useful for:

  • marking sets of records that you frequently need to print as a group
  • creating a set of related records for use in the "Quick List" dropdown menu on the basic search page.
  • creating a View of the database that contains only a subset of your records (known as a "Publication View").
  • creating a custom classification that applies to only a subset of records (done through the General Headings system that is part of Publications)
  • creating a custom text field that applies to only some records (done through the use of a Publication Description field)

For very simple groupings of records (data export, leases, internal use, etc.) Distribution Codes may be a more appropriate choice, as they are simpler to manage.

List of Publications

Publications on this page are broken down into one or three sections:

  • Main listing of Publications owned/managed by you (your membership). In a single-member database, this is the only list available.
  • Shared Publications. In a multi-member database, this represents Publications that can be used by any member in the database. Shared Publications can be created and managed by Global Super Users only, but any member may add these Publications to their records. If you do not wish to use a shared Publication, you can "Hide" it by checking the box next to the appropriate Publication and clicking the "Update" button at the bottom of the form.
  • Publications owned by other members. In a multi-member database, a link is provided to see a list of codes in use exclusively by other CIOC Members in the same database. Global Super Users can convert these into shared publications, which will make them available to other members.

The columns in the table listing of Publications include some or all of:

  • Code - The unique alpha-numeric code identifying this Publication
  • Name - This is the "friendly name(s)" associated with this Publication. If nothing appears beside a Publication in this column, a friendly name has not been created for this code.
  • Non-Public - If the Publication has been marked as a non-public Publication, an asterisk will appear in this column.
  • Headings - If General Headings are part of this Publication, a link is provided to view a list of the Headings. In print mode, the number of General Headings associated with the Publication is printed instead.
  • (Local) Records - The number of your records assigned to this Publication.
  • Other Records - In a multi-member database, the number of records owned by other CIOC Members in your database that are assigned to this Publication.
  • (Local) Views - The number of your Views using this Publication to limit the available records (a "Publication View").
  • Other Views - In a multi-member database, the number of Views owned by other CIOC Members in your database. Visible only to Global Super Users.
  • (Local) Quick List - The number of Views in which this Publication is present in the Quick List on the main search page. Visible only when viewing the Print Version of the list of Publications.
  • Other Quick List - In a multi-member database, the number of Views owned by other CIOC Members in your database in which this Publication is present in the Quick List on the main search page. Visible only to Global Super Users, and only when viewing the Print Version of the list of Publications.

Edit an Existing Publication

To view or edit the settings of an existing Publication, click the "Edit" link next to the Publication.

In a multi-member database, you may not have access to edit all Publications. The following security restrictions apply:

  • Only Global Super Users can modify shared Publications that are available for use by any member in the database.
  • Only Global Super Users can convert an exclusively owned Publication into a shared Publication.
  • Global Super Users cannot edit Publications that are exclusively owned by another member without first converting them into a shared Publication.

Create New Publication

Click the "Add New Publication" link at the top of the page to create a new Publication code.

All new Publications start out exclusively owned by the member that created them. In a multi-member database, a Global Super User can later convert these into shared publications that can be used by other members.

Print a report of your Publications

Clicking the link "Print Version (New Window)" will open up a print-friendly list of all the publication codes in the database.

The "Publications List" page provides a printable list of all current Publications in the database available to your CIOC Membership. In a multi-member database, this report will not include Publications owned exclusively by another member. Includes some information not available from the main table, as mentioned in the previous section describing the parts of the Publication list table.

This list appears in "Print Mode" - if you have a print template configured for your current View, that template will be used to display the records.
