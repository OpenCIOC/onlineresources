Each Taxonomy Term can be associated with a Source which is meant to represent where the term originated from (i.e. "INFO LINE " or "Local"). This is a general explanation of where the term originated from, unlike the Modified By information that indicates a particular person who last made modifications to the Term. This page allows you to add new source values, as well as edit or delete existing values. Use the "Return to Manage Taxonomy" link at top of page to return to thesaurus management main page.

Note: Values on this page can only be added, updated and deleted one row at a time. Remember to use the "Action" buttons beside each row after each change is made before making a change to another row or adding another value.

Add a New Source

Type the name you wish to use as the Source for Taxonomy Terms into the bottom most text entry box on the page and click the "Add" Button.

Edit an Existing Source

Make the required changes to the Source you wish to edit on the page and click the "Update" button beside that value.

Delete an Existing Source

Click the "Delete" button beside the value you wish to delete. You will not be able to delete values that are currently in use by Taxonomy Term(s).
