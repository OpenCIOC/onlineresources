On the Send Email Update Request page, the software will attempt to send the Email Update Request(s) you have created. The software will report any known issues with sending the message(s) and print a report of the email addresses and record numbers that were successfully sent. For example:

Email sent to: myaddress1@mysite.com (ACT0001)
Email sent to: myaddress2@anothersite.ca (ACT0002)
Email sent to: myaddress3@thatplace.ca (ACT0003)

It is recommended that you print this page, as it provides you with the only record of emails successfully sent.

Note: it is still possible for an email delivery to fail even though the system was able to create the email message; you should receive a "bounce-back" email message in this situation, but you will generally not be notified if the message was caught in a spam filter.
