Export Profiles allow you to configure profiles customizing export formats for using data in other systems. This includes the CIOC Sharing Format that allows you to share records to other CIOC databases. Export Profiles allow you to define which fields, Distribution and Publication Codes that you want to share and specify which View(s) the export profile will be available in.

Note that Excel Profiles are managed separately. Excel Profiles offer more options in terms of what can be exported, but do not provide any structure to the information and so tend to be less useful as a means for using your data in other systems (and cannot be used to exchange data between CIOC databases).

Edit an Existing Export Profile

To view or edit the settings of an existing Export Profile, select the profile from the list and click "View / Edit Profile"

Add New Export Profile

To create a new Export Profile, you must specify a unique name for the Profile. If you have existing Profiles, you can copy a profile by selecting it from the copy Profile list. If you do not select a Profile to copy, your new Profile will have default settings and no fields added. Click the "Add Profile" button to create the new Profile; you will be taken to the form to edit your new profile.
