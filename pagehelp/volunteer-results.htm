These instructions are shared by several different types of searches, including the Main Page (Basic) Search, Advanced Search, Step-by-Step Search, New Results Set (using checkboxes to create a new list of records), Previous Search Results, and various "Browse" searches. Not all features exist for all searches; see notes in each section if applicable.

The Search Results page displays records that match the criteria you entered from either a Basic or Advanced search. If the message that appears on this page is: "There are no records that match your criteria", then there are no records in the database that match the criteria you chose. In this case, use your back button or return to the Main Menu / Start Page or Advanced Search page and begin a new search that is less restrictive.

At the top of the page, a legend is printed mapping specific community group names to coloured balls. The coloured balls are printed next to the opportunities, indicating in which general areas the position is offered. The setup for which communities are grouped together, the names of the groupings, and the coloured ball associated with the group can be changed by a Super User from the Volunteer Communities setup.

Each record in your search results will have a link to view the full Record Details. The link will be found on either the Position Title or Opportunity ID. Whether the name or ID is linked is determined by your Display Options, discussed below.

About Your Search

Most searches include a description of your search at the top of the page. From Advanced Search, the default is that the details of your search are not included, but you can check a box at the bottom of the search form to have the details of your search printed on the results page. When the details of your search are available, the information can be used to describe your search when you Refine Your Search, when you view your list of Recent Searches, and when you Save Your Search.

Refine Your Search

Clicking the  Refine Search link at the top of your search results will send the current search criteria to the Advanced Search form so that you can add additional criteria to search within the current results.

Record List Display Options

All users with a login into the database have the option of changing the way their search results are displayed. The exact options you have available to you for your display will be determined by your permissions in the database. You can change your display options temporarily for your session, or save your changes as your default to use every time you log into the database. To change your settings, click the Change Display Options link; a pop-up window will appear that allows you to modify your current settings. Note that your changes will not be reflected until you perform a new search, or refresh your current search results page.

Very Important Note: the Use Table Format option (discussed in the Show Options section below) is required for many of the options to be available. The regular view (with Table Format turned off) is only intended for basic searching needs.

Show Fields

Show Fields allows you to decide what information from your records will be shown in your results. You may select from a set of predefined fields, as well as choose an unlimited number of "Custom" fields. The list of Custom fields available to you is determined by what information you are allowed to see from the Record Details page.

The Alert Column is a special box that appears on the far left of your search results. Your permissions dictate whether you will be able to see the Alert Column, and which alerts you are allowed to see. If you are not using the Table Format for your display, the Alert Column is not available, and some of these alerts will instead be printed beside the link to the Opportunity Details. The following are a list of the available alerts:

  • Alert Coloured Box / N - When the alert column is coloured, the record is Non-public (the letter N is used if there is no alert column)
  • C - Record has Comments or an Internal Memo
  • E - Record is Epired (Display Until Date has passed)
  • F - Record has Feedback
  • P - Record is Pending deletion (a deletion date set in the future)
  • S - Record has been Shared with you by another CIOC Member in your database
  • X - Record has been deleted

Show Options

Show Options will provide you direct links to perform various types of operations on your list of records. The exact options available to you will depend on your permissions.

The default display (not Table Format) prints the Position Title, coloured balls indicating the communities where the positions are available, the Duties field, and the name of the Organization offering the position. The Use Table Format option allows the user to change the display into a table with column headings, with each record printed as one row of the table. This is similar to the results display for the Community Information module of the software, and makes many of the other display options possible, including the Show Fields options described in the next section.

The Select Checkbox option is used to perform operations on many records at once. A checkbox will appear beside each record in your results, allowing you to select the records you want and perform operations on them. Some options (depending on your permissions) include:

  • Create a new results set from selected records
  • Create a new reminder for the selected records
  • Print a list of records
  • Create usage statistics reports
  • Bulk Email Update Request
  • Add or remove "Areas of Interest"
  • Set the Public status for the selected records
  • Delete / Restore Records
  • Change the owner of the selected records
  • Add the selected records to a Sharing Profile to make it available to another CIOC Member in your database

Sort by

This section allows you to change the order in which records are listed. You may select from one of the pre-defined sort options, or a custom choice from the drop-down list.

Print Version

If you want to print a report of your search results, clicking the Print Version link (next to Change Display Options) will allow you to see a report of your results formatted for printing, without the menu and extra links. The Super User of the database can customize this print version and optimize it for printing by creating a special Design Template just for printing, and adding it to the current View.

Save This Search

To save your search criteria to your list of saved searches, click the "Save This Search" link. Helpful Hint: If the details of your search are displayed at the top of the page, they will be carried over to the notes field of your search.

This feature is not available when performing a "Browse" search, or when working with a "New Results Set" or viewing "Previous Search Results".
