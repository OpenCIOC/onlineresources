Note: the contents of this page vary depending on whether you are working in a database with other CIOC Members (a "Multi-Member" database).

The Manage Agencies page allows you to add a new Agency (AKA Record Owner) to the database or edit an existing Agency. It is very important to have an entry for every record owner code used for records in your database, including an email address and associated organization (Community Information) record so that feedback emails can be processed correctly.

Add a New Agency

Click the "Add New Agency" link to be taken to a form for entering information for a new Agency. Global Super Users (Super Users that control values shared by all Members in this database) have the ability to create an Agency for any Member in the database.

Edit an Existing Agency

Under the "Add New Agency" link, there is a table listing all the Agency Codes associated with your CIOC Membership. This table includes a count of all users, Community Information records, and Volunteer records that are owned by each Agency. Click the "Update" link next to the Agency / Record Owner you wish to edit.

Agencies for Other Members

If you are working in a Multi-Member database, you will see a listing of Agency Codes belonging to other CIOC Members in your database. You will have the option to select one or more of these codes for display on your own update and search forms. Global Super Users (Super Users that control values shared by all Members in this database) will have the ability to manage these Agencies belonging to other Members.
