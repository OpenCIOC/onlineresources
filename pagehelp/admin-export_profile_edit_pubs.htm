The Manage Export Publication Codes page allows you to select which Publication codes will be included in the Publication field of the records you export. To add a new Publication to the list, select it from the drop-down list at the bottom of the form and select the "Add" button. It is possible that there will be no Publication codes available to select from this page if none exist within your application. After adding the Publication Code, you can remove it by clicking the "Delete" button next to the Publication code.

For each Publication you select, you have the option of including any Descriptions or General Headings associated with the Publication. Check or uncheck the boxes indicating whether you want to include Descriptions or General Headings for a Publication and click the "Update" button to save your changes. You must process changes to each Publication one at a time.

Important Note: Selecting particular Publication codes for your profiles does not impact which records you will be exporting, only the contents of the Publication field you are sending. Records to send are selected at the time of export.

Submitted changes take effect immediately. It is not necessary to submit the main Edit Export Profile form to save your changes.
